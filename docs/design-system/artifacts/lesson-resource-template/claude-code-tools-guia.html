Mecanismos de Extensão

Não são "tools", mas ampliam exponencialmente o poder do agente

Skills

✓

Pastas com SKILL.md + scripts/templates que o modelo carrega automaticamente quando relevante (descoberta semântica).

EXEMPLOS:
Excel Analyst PR Drafter Database Migrator API Documentation Generator

Commands

✓

Ações explícitas do usuário no REPL com prefixo /, como /config, /permissions, /skills.

EXEMPLOS:
/config /permissions /skills /help

Hooks

✓

Gatilhos shell executados antes/depois de certas fases (por tool, por evento), garantindo ações determinísticas.

EXEMPLOS:
Formatar código após edições Rodar testes automaticamente Bloquear caminhos sensíveis

Sub-agents

✓

Perfis especializados com contexto isolado e conjunto específico de tools permitido, invocados via Task.

EXEMPLOS:
code-reviewer test-writer documentation-specialist security-auditor

Cobertura Completa

As 16 tools nativas cobrem todos os aspectos do desenvolvimento: desde manipulação de arquivos até execução de comandos, passando por busca web e gerenciamento de tarefas.

Extensibilidade

Os 4 mecanismos de extensão (Skills, Commands, Hooks, Sub-agents) permitem customizar e ampliar as capacidades do Claude Code para suas necessidades específicas.

Poder Real

A combinação de tools nativas com Progressive Disclosure e Sub-agents cria um sistema de automação sem precedentes.

Casos de Uso Práticos

Refatoração Completa de Projeto

Use Glob para encontrar arquivos, MultiEdit para refatorar, Bash para rodar testes e TodoWrite para acompanhar o progresso.

Análise de Dados em Notebooks

Combine NotebookRead para analisar dados existentes, WebSearch para buscar melhores práticas e NotebookEdit para implementar melhorias.

Documentação Automática

Use uma Skill de documentação que combina Read, Grep e Write para gerar docs automaticamente baseadas no código.

Code Review Automatizado

Invoque um Sub-agent especializado via Task que usa Grep e Read para revisar código e sugerir melhorias.

Conclusão

O Claude Code oferece um conjunto completo de ferramentas que cobrem todas as necessidades do desenvolvimento moderno. Das operações básicas de arquivo às capacidades avançadas de sub-agentes, cada tool foi projetada para maximizar produtividade e automação.

A combinação de 16 tools nativas com 4 mecanismos de extensão cria um ecossistema único que permite desde tarefas simples até automações complexas de nível empresarial.

Domine essas ferramentas e você terá o poder de automatizar praticamente qualquer aspecto do seu workflow de desenvolvimento.
