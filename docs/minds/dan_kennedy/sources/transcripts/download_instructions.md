## Ferramentas para Download:

### Para YouTube:
```bash
# Baixar áudio apenas
yt-dlp -x --audio-format mp3 [URL]

# Baixar com legendas/transcrição
yt-dlp --write-auto-sub --sub-lang en [URL]
```

### Para Podcasts:
- Podcast Addict (Android)
- Overcast (iOS)
- Podbean

## Processo de Transcrição:

### Opção 1: Whisper (OpenAI)
```bash
whisper audio.mp3 --language en --model medium
```

### Opção 2: YouTube Auto-Captions
- Já incluídas em muitos vídeos
- Usar yt-dlp com flag --write-auto-sub

### Opção 3: Services Online
- Rev.com
- Otter.ai
- Temi.com
