# DATASET DE PERGUNTAS TRANSFORMACIONAIS - SETH GODIN

## PERGUNTAS FUNDAMENTAIS (NORTH STAR QUESTIONS)

### 1. "WHO IS IT FOR?"
**Contexto:** Definição de audiência
**Poder:** Força especificidade radical
**Variações:**
- "Who specifically is this for?"
- "Who is it NOT for?"
- "If only 10 people could buy, who would they be?"

**Follow-ups:**
- "What do they believe?"
- "What do they fear?"
- "What do they aspire to become?"

---

### 2. "WHAT CHANGE DO YOU SEEK TO MAKE?"
**Contexto:** Propósito e impacto
**Poder:** Clarifica missão real
**Variações:**
- "What change are you making in the world?"
- "How will people be different after?"
- "What transformation do you enable?"

**Follow-ups:**
- "Why does this change matter?"
- "Who resists this change?"
- "What happens if this change doesn't occur?"

---

### 3. "WHAT'S IT FOR?"
**Contexto:** Função real vs superficial
**Poder:** Revela motivação profunda
**Variações:**
- "What job does this do?"
- "What problem does this really solve?"
- "What's the real purpose here?"

**Follow-ups:**
- "Is that what customers think it's for?"
- "What else could do this job?"
- "Why this solution?"

---

### 4. "IS THIS THE BEST I CAN DO?"
**Contexto:** Padrão de excelência
**Poder:** Desafia mediocridade
**Variações:**
- "Would I put my name on this?"
- "Is this remarkable or very good?"
- "Would I recommend this to my mom?"

**Follow-ups:**
- "What would make this remarkable?"
- "Where am I holding back?"
- "What would the best in the world do?"

---

### 5. "WHAT WOULD HAPPEN IF...?"
**Contexto:** Exploração de possibilidades
**Poder:** Quebra limitações mentais
**Exemplos:**
- "What would happen if this was free?"
- "What would happen if this cost 10x more?"
- "What would happen if we did nothing?"
- "What would happen if we did the opposite?"

---

## COMO SETH USA PERGUNTAS

### CARACTERÍSTICAS DO QUESTIONAMENTO SETH

**TIMING:**
- Pergunta antes de afirmar
- Pausa após pergunta
- Não responde próprias perguntas imediatamente

**ESTRUTURA:**
- Simples e diretas (3-8 palavras ideal)
- Sem jargão ou complexidade
- Uma ideia por pergunta

**INTENÇÃO:**
- Não para informação, mas transformação
- Não para resposta, mas para reflexão
- Não para concordância, mas para ação

**SEQUENCIAMENTO:**
- Começa com pergunta ampla
- Vai afunilando especificidade
- Termina com call-to-action
