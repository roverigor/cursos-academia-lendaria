# MANUAL DE USO - CLONE COGNITIVO SETH GODIN PARA POSICIONAMENTO

## COMO ATIVAR O CLONE

### INSTRUÇÕES DE IMPLEMENTAÇÃO

1. **Copie o System Prompt completo** do arquivo `SYSTEM_PROMPT_SETH_GODIN_POSICIONAMENTO.md`
2. **Cole em seu LLM preferido** (Claude, GPT-4, etc.)
3. **Configure temperatura:** 0.7-0.8 (balanceando criatividade e consistência)
4. **Inicie com uma pergunta de teste** dos cenários validados

### PRIMEIRA INTERAÇÃO RECOMENDADA

```
"Seth, sou [sua situação específica]. Para quem devo criar meu posicionamento?"
```

Isso ativa imediatamente o modo característico de Seth de forçar especificidade.

## MELHORES PRÁTICAS DE USO

### 1. SEJA ESPECÍFICO
❌ **Evite:** "Como melhorar meu marketing?"
✅ **Prefira:** "Sou designer freelancer há 2 anos, trabalho com startups, mas perco clientes para agências maiores. Como me posicionar?"

### 2. PERMITA PROVOCAÇÃO
- Seth vai questionar suas premissas
- Não defenda - explore a tensão
- A resistência indica onde está o ouro

### 3. IMPLEMENTE IMEDIATAMENTE
- Seth sempre dá forward motion
- Execute a ação sugerida em 24h
- Ship antes de estar pronto

## CASOS DE USO IDEAIS

### DESCOBERTA DE PURPLE COW PESSOAL
**Prompt Inicial:**
"Seth, todo mundo no meu mercado faz [X]. Eu também faço [X]. Onde está minha Purple Cow?"

**O que esperar:**
- Questionamento do "todo mundo faz X"
- Busca pela weird edge única
- Especificação do "para quem"
- Ação remarkable para testar

### DEFINIÇÃO DE SMALLEST VIABLE AUDIENCE
**Prompt Inicial:**
"Quero alcançar o máximo de pessoas com meu [produto/serviço]"

**O que esperar:**
- Inversão completa da premissa
- Foco em 10 pessoas específicas
- Critérios de validação profunda
- Como expandir através de intensidade

### SUPERAÇÃO DE PARALISIA
**Prompt Inicial:**
"Estou há [tempo] preparando meu [projeto] mas não consigo lançar"

**O que esperar:**
- Ship it mentality imediata
- Emotional labor vs inspiração
- Redefinição de "pronto"
- Ação específica para hoje

### TRANSFORMAÇÃO DE COMMODITIZAÇÃO
**Prompt Inicial:**
"Meu serviço virou commodity, todos cobram o mesmo preço"

**O que esperar:**
- Questionamento de "todos"
- Para quem NÃO é o serviço
- Elementos de premium positioning
- História que muda percepção

## NAVEGANDO PELOS MODOS

### ATIVANDO MODO PROVOCADOR
Use quando precisar de breakthrough thinking:
- "O que todo mundo acredita sobre [X] que está errado?"
- "Por que [sabedoria convencional] não funciona?"

### ATIVANDO MODO COACH
Use quando precisar de passos práticos:
- "Me ajude a implementar [conceito]"
- "Quais os próximos 3 passos para [objetivo]?"

### ATIVANDO MODO STORYTELLER
Use quando precisar de inspiração contextualizada:
- "Me conte sobre alguém que [situação similar]"
- "Como [empresa/pessoa] aplicou isso?"

### ATIVANDO DENSITY MASTER
Use quando precisar de clareza cristalina:
- "Resuma [conceito complexo] em uma frase"
- "Qual a essência de [situação]?"

1. **Min 0-5:** "Para quem é meu trabalho?"
2. **Min 5-10:** "O que eles acreditam?"
3. **Min 10-15:** "Que mudança busco fazer?"
4. **Min 15-20:** "Onde está minha Purple Cow?"
5. **Min 20-25:** "Quais meus anti-valores?"

## TROUBLESHOOTING

### SE AS RESPOSTAS PARECEM GENÉRICAS:
- Adicione contexto específico seu
- Use nomes, números, situações reais
- Pergunte "Para mim especificamente?"

### SE FALTA PROVOCAÇÃO:
- Peça explicitamente o contrarian view
- "O que eu não estou vendo?"
- "Onde estou sendo very good?"

### SE MUITO FILOSÓFICO:
- Peça aplicação prática
- "Como isso se traduz em ação?"
- "O que faço amanhã às 9h?"

### SE MUITO TÁTICO:
- Peça o princípio por trás
- "Por que isso funciona?"
- "Qual o meta-padrão aqui?"
