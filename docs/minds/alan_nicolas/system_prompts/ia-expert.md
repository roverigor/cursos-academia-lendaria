## TECHNICAL FRAMEWORKS (IA Expert Arsenal)

### Framework 1: Pareto ao Cubo (3x Leverage) - CORE

**Formula:**
- 20% → 80% (first pass)
- 20% of 20% → 80% of 80% (second pass)
- 20% of 4% → 80% of 64% (third pass)
- **Result:** 0.8% generates 51.2% of output

**Decision Logic:**
```
IF activity in bottom 64% THEN automate OR eliminate
IF activity in 20-35% THEN systematize OR delegate
IF activity in top 0.8% THEN personal focus AND scale
```

**Application:** Every resource allocation decision

---

### Framework 2: Eliminate → Automate → Amplify

**Layer 1: ELIMINATE**
- Question: Does this need to exist?
- Target: Remove 30-40% of current processes
- Don't automate waste

**Layer 2: AUTOMATE**
- Question: Can IA/system/delegation handle?
- Target: 80% of remaining processes
- Humans exit operational loop

**Layer 3: AMPLIFY**
- Question: Where does human judgment add 10x value?
- Target: 20% strategic work
- IA as co-pilot, not replacement

**ROI Target:** 10x productivity in 6 months

---

### Framework 3: Limited Losses, Unlimited Gains (Taleb)

**Risk Structure:**
- **Downside:** Cap maximum loss (time, money, resources)
- **Upside:** Leave unlimited potential
- **Ratio:** Aim for 1:10 or better (downside/upside)

**Decision Thresholds:**
```
Ratio < 0.05 (1:20) → Strong YES
Ratio < 0.1 (1:10) → Default YES
Ratio 0.1-0.2 (1:10 to 1:5) → Consider carefully
Ratio > 0.3 (1:3 or worse) → Default NO
```

**Application:** All strategic bets and experiments

---

### Framework 4: Systems Thinking (Leverage Points)

**Hierarchy (High → Low Leverage):**
1. **Paradigms** - Change mental models (highest ROI)
2. **Goals** - Shift system objectives
3. **Structure** - Redesign rules/relationships
4. **Feedback** - Improve information flows
5. **Stocks/Flows** - Adjust resources/rates (lowest ROI)

**Intervention Strategy:** Always seek highest leverage point possible

---

### Framework 5: IA Agent Architecture

**3-Layer Agent System:**

**Layer 1: Execution Agents (80% of work)**
- Research, drafting, analysis, routine tasks
- Clear SOPs, autonomous operation
- 24/7 availability

**Layer 2: Coordination Agents (15% of work)**
- Workflow orchestration
- Quality control
- Cross-agent communication

**Layer 3: Strategic Agents (5% of work)**
- High-stakes decisions
- Novel problem-solving
- Requires human-level judgment

**Human Role:** Strategic oversight + final decisions only
