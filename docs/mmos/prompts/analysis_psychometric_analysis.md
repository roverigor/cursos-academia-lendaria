# PSYCHOMETRIC ANALYSIS

## METADADOS
- Versão: 3.0 ACS Neural Flow
- Input: analysis/quotes.md, analysis/timeline.md, analysis/cognitive_patterns.md, analysis/behavioral_triggers.md, sources/
- Output: analysis/personality_profile.json
- Dependências: 01_quote_extraction.md, 02_behavioral_patterns.md, 04_cognitive_architecture.md

## OBJETIVO PRINCIPAL
Analisar o clone alvo criando perfil psicométrico completo através de frameworks obrigatórios (DISC, Eneagrama, MBTI, Estrato Cognitivo) e complementares baseados em evidências observáveis.

## INPUT NECESSÁRIO

```
clone_target: "[NOME COMPLETO]"
quotes_file: "analysis/quotes.md"
timeline_file: "analysis/timeline.md"
cognitive_file: "analysis/cognitive_patterns.md"
behavioral_file: "analysis/behavioral_triggers.md"
sources_path: "sources/"
```

## METODOLOGIA

### FASE 1: COLETA DE DADOS

Analisar todos os dados disponíveis:

- Biografias, entrevistas, documentários
- Decisões documentadas e seus contextos
- Padrões comportamentais repetidos
- Testemunhos de pessoas próximas
- Produções escritas/faladas diretas
- Métricas objetivas de performance
- Histórico educacional e profissional
- Crises e como respondeu a elas
- Relacionamentos e sua evolução
- Hobbies e interesses persistentes

### FASE 2: ANÁLISE ESTRUTURADA

Aplicar frameworks psicométricos validados com evidências comportamentais.

### FASE 3: SÍNTESE INTEGRATIVA

Integrar resultados dos diferentes frameworks identificando convergências e tensões.
