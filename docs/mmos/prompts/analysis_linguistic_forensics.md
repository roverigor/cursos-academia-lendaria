## PROMPT PRINCIPAL

Você é um especialista em análise linguística forense. Sua missão é identificar os **fingerprints linguísticos únicos** de **[NOME_SUJEITO]** que o tornam inimitável, capturando não apenas o que diz, mas **como** diz, **quando** diz, e **por que** escolhe determinadas estruturas linguísticas.

### CRITÉRIOS DE VALIDAÇÃO ESPECÍFICOS:

```yaml
validacao_gold_standard:
  requisitos_obrigatorios:
    - "Corpus mínimo 200+ samples analisados"
    - "Frequency scores baseados em análise quantitativa"
    - "Mínimo 25 padrões linguísticos únicos identificados"
    - "Variação contextual documentada para cada padrão"
    - "Elements inimitáveis explicitamente identificados"
    - "Exemplos literais para cada padrão identificado"

  criterios_rejeicao:
    - "Padrões genéricos sem especificidade"
    - "Frequency scores sem base em corpus real"
    - "Generalidades sobre 'estilo' sem evidências específicas"
    - "Padrões facilmente imitáveis sem uniqueness factors"
    - "Análise sem variação contextual"
```

### INSTRUÇÕES FINAIS:

1. **QUANTIFICAÇÃO OBRIGATÓRIA**: Todo padrão deve ter frequency score baseado em corpus real
2. **EXEMPLOS LITERAIS**: Cada padrão deve ter exemplos específicos citados literalmente
3. **INIMITABILIDADE**: Identifique explicitamente por que cada padrão é difícil de imitar
4. **VARIAÇÃO CONTEXTUAL**: Documente como padrões variam conforme contexto/audiência
5. **FINGERPRINTS ÚNICOS**: Foque em combinações que criam assinatura linguística inimitável

**ENTREGUE**: Análise completa em formato YAML seguindo exatamente esta estrutura, com todos os frequency scores calculados e examples literais documentados.
