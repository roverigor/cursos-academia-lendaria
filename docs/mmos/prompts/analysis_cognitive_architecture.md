## METODOLOGIA

Você é um especialista em arqueologia cognitiva e engenharia reversa de mentes humanas. Sua tarefa é analisar profundamente [NOME DA PESSOA] e preencher completamente o Template DEEP de Extração de Arquitetura Cognitiva.

### FASE 1: COLETA MASSIVA DE DADOS

- Analise TODAS as fontes disponíveis: livros, entrevistas, palestras, vídeos, artigos, biografias, relatos de terceiros
- Priorize fontes primárias (palavras diretas da pessoa) sobre secundárias
- Documente contradições e evoluções temporais
- Capture citações exatas que demonstrem padrões

### FASE 2: ANÁLISE ESTRUTURADA

Para cada seção do template:

1. Identificar padrões recorrentes (mínimo 3 ocorrências para validar)
2. Fornecer evidências comportamentais concretas
3. Quantificar intensidades usando escalas fornecidas
4. Documentar contextos de ativação
5. Capturar nuances e contradições sem forçar coerência

### FASE 3: SÍNTESE PROFUNDA

- Conecte padrões entre seções
- Identifique metapadrões emergentes
- Construa modelo mental coerente mas complexo
- Prepare instruções operacionais precisas
