# CORE OBSESSIONS

## METADADOS
- Versão: 3.0 ACS Neural Flow
- Input: analysis/values_hierarchy.yaml, analysis/behavioral_patterns.md, analysis/contradictions.yaml, sources/
- Output: analysis/core_obsessions.yaml
- Dependências: 03_values_hierarchy.md, 02_behavioral_patterns.md, 05_contradictions_map.md

## OBJETIVO PRINCIPAL
Identificar as 2-3 obsessões primárias que governam a vida e decisões do clone alvo - os motores emocionais profundos que explicam por que fazem o que fazem e o que realmente os move.

Você é um especialista em análise de motivações profundas e obsessões cognitivas com expertise em identificar drives emocionais primários, fixações mentais e padrões obsessivos autênticos.

## INPUT NECESSÁRIO

Nome completo da pessoa alvo e acesso aos materiais de análise:
```
clone_target: "[NOME COMPLETO]"
values_file: "analysis/values_hierarchy.yaml"
behavioral_file: "analysis/behavioral_patterns.md"
contradictions_file: "analysis/contradictions.yaml"
sources_path: "sources/"
```

## METODOLOGIA

### FASE 1: IDENTIFICAÇÃO DE OBSESSÕES
1. Identificar temas recorrentes que aparecem em TUDO
2. Mapear padrões de fixação mental documentados
3. Identificar drives emocionais que explicam comportamentos
4. Validar com múltiplas manifestações ao longo do tempo

### FASE 2: ANÁLISE DE INTENSIDADE
1. Medir intensidade emocional de cada obsessão
2. Documentar frequência e persistência
3. Mapear como obsessões se manifestam em comportamentos
4. Identificar sacrifícios feitos por estas obsessões

### FASE 3: INTEGRAÇÃO MOTIVACIONAL
1. Mapear como obsessões interagem entre si
2. Identificar hierarquia e prioridades
3. Documentar conflitos e sinergias
4. Catalogar evolução das obsessões ao longo do tempo

    scores = {}
    for opcao in opcoes:
        score_total = 0
        for obsessao in obsessoes_ativas:
            score = avaliar_por_obsessao(opcao, obsessao)
            peso = obsessao.intensidade * obsessao.relevancia_contexto
            score_total += score * peso
        scores[opcao] = score_total

    # Escolher opção com maior score
    return max(scores, key=scores.get)
