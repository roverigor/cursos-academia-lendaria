## INSTRUÇÕES DE USO

### Técnicas de Mineração:

#### EXTRAÇÃO SISTEMÁTICA
- Leia/ouça TODO o material
- Pause para documentar quotes exatas
- Note contexto SEMPRE
- Marque timestamp/página

#### CATEGORIZAÇÃO IMEDIATA
- Classifique enquanto extrai
- Use tags temporárias se incerto
- Agrupe quotes similares
- Identifique variações

#### ANÁLISE DE PADRÕES
- Palavras que se repetem
- Estruturas recorrentes
- Metáforas favoritas
- Temas obsessivos

### O que extrair:
- Quotes longas que revelam processo de pensamento
- Quotes curtas que são mantras/princípios
- Contradições mesmo que embaraçosas
- Admissões especialmente as raras
- Metáforas que usa repetidamente
- Silêncios - o que nunca diz

### Como documentar contexto:
1. Quando: Data exata se possível
2. Onde: Local, evento, plataforma
3. Para quem: Audiência específica
4. Por quê: O que provocou a fala
5. Estado: Emocional/mental aparente

## TÉCNICAS AVANÇADAS

### Identificar Subtexto:
- O que a pessoa está REALMENTE dizendo?
- Que emoção está por trás?
- Que defesa está ativada?
- Que imagem quer projetar?

### Detectar Evolução:
- Compare quotes sobre mesmo tema em diferentes épocas
- Note mudanças de vocabulário
- Identifique quando narrativas mudam
- Marque pontos de inflexão ideológica

### Encontrar Patterns Ocultos:
- Palavras que aparecem juntas
- Estruturas gramaticais repetidas
- Ritmo e cadência característicos
- Triggers que geram respostas similares

## ALERTAS CRÍTICOS
- Não edite quotes - Preserve exatamente como ditas
- Não ignore contradições - São essenciais para autenticidade
- Não descontextualize - Contexto muda significado
- Não seja seletivo - Extraia tanto positivo quanto negativo
- Preserve idioma original quando relevante
- Arquivo quotes.md deve estar em analysis/ conforme OUTPUTS_GUIDE.md
- As citações são a voz direta da pessoa - será consultado constantemente
