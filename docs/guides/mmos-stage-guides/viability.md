# Etapa 1: Viability Assessment

## Objetivo
Avaliar se vale a pena criar o clone usando SCORECARD APEX.

## Prompts desta Etapa
1. `01_scorecard_apex.md` - Avaliação inicial
2. `02_prd_generator.md` - Gera PRD (paralelo)
3. `02_dependencies_mapper.md` - Mapeia influências (paralelo)
4. `03_todo_initializer.md` - Cria TODO

## Outputs Esperados
- PRD.md em docs/
- TODO.md em docs/
- dependencies.yaml em metadata/
- viability_assessment.yaml em logs/

## Critérios de Sucesso
- Score ≥ 5 legal
- Score ≥ 6 impacto
- Arquétipo identificado

## Human Checkpoint
Aprovar PRD, TODO e investimento estimado.
