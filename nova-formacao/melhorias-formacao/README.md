## Contexto

Durante análise das trilhas (2025-01-01), identificamos que:

1. **Problema:** Mistura de conteúdo estratégico com técnico intimida o empresário
2. **Solução:** Separar em duas camadas (Negócio + Técnica)
3. **Benefício:** Maior adoção, resultado mais rápido, upsell natural
