## [PROTOCOLO DE TESTE] - 2 minutos

**[PROFESSOR DIZ:]**

> Alerta só vale se funciona quando precisa.
>
> E você não quer descobrir que estava quebrado justo quando precisava.
>
> Por isso, protocolo de teste.

**[MOSTRAR NA TELA:]**

```
PROTOCOLO DE TESTE DE ALERTAS

1. SIMULAR CENÁRIO
   → Coloca um dado "ruim" na planilha
   → Executa o workflow manualmente
   → Verifica se o alerta dispara

2. VERIFICAR MENSAGEM
   → A mensagem chegou?
   → Os dados estão corretos?
   → É clara o suficiente?

3. RESTAURAR DADO
   → Volta o dado original
   → Confirma que não dispara sem motivo

4. REPETIR PRA CADA ALERTA
```

**[PROFESSOR DIZ:]**

> Faz isso agora, antes de seguir.
>
> Coloca um faturamento falso de R$0 na planilha.
>
> Roda o workflow de Crise.
>
> Vê se a mensagem chega.
>
> Depois restaura o dado real.
>
> Repete pro de Tendência e pro de Meta.
