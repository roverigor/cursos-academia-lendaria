## [O QUE É N8N] - 2 minutos

**[PROFESSOR DIZ:]**

> n8n é uma ferramenta de automação.
>
> Funciona assim: você conecta coisas.
>
> Planilha com WhatsApp. CRM com E-mail. Sistema com Slack.
>
> Tipo Zapier ou Make, mas com uma diferença: é open source.
>
> Você pode hospedar de graça em servidores como Render ou Railway.
>
> Custo: R$0.

**[MOSTRAR NA TELA:]**

```
O QUE É N8N

✅ Automação visual (arrasta e solta)
✅ Open source (código aberto)
✅ Self-hosted (você hospeda, R$ 0)
✅ 400+ integrações prontas
✅ Comunidade ativa
```

**[PROFESSOR DIZ:]**

> A lógica é simples.
>
> Você cria um "workflow".
>
> O workflow olha sua planilha de tempos em tempos.
>
> Se encontrar algo fora do normal, dispara uma ação.
>
> No nosso caso: manda WhatsApp.

## [POR QUE WHATSAPP] - 2 minutos

**[PROFESSOR DIZ:]**

> Por que WhatsApp e não e-mail?
>
> Porque você lê WhatsApp.
>
> E-mail vai pra caixa de entrada. Você vê quando der.
>
> WhatsApp vai pro seu bolso. Você vê na hora.
>
> Pra alerta de crise, WhatsApp é obrigatório.
>
> (pausa de 2 segundos)
>
> Pra conectar n8n com WhatsApp, eu uso a Evolution API.
>
> É uma API gratuita que transforma seu WhatsApp em canal de automação.
>
> Você conecta, e o n8n consegue mandar mensagem.

**[MOSTRAR NA TELA:]**

```
STACK DE ALERTAS

┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  Google     │ →→→ │    n8n      │ →→→ │  WhatsApp   │
│  Sheets     │     │ (automação) │     │ (Evolution) │
└─────────────┘     └─────────────┘     └─────────────┘

Custo total: R$ 0
```

**[MOSTRAR NA TELA:]**

```
ALTERNATIVAS

AUTOMAÇÃO:

- Evolution API (grátis) ← Recomendado
```
