>

>
> E a melhor parte? Não é um sistema caro. Não é um ERP novo.
>
> É uma planilha. Simples. Mas poderosa.

---

## [A ANALOGIA] - 2 minutos

**[PROFESSOR DIZ:]**

> Antes de te mostrar a ferramenta, deixa eu te fazer uma pergunta.
>
> Você já usou GPS?
>
> Waze, Google Maps, qualquer um.
>
> O GPS não cria estradas novas. Ele não muda a cidade.
>
> Ele só te mostra ONDE as coisas estão. E COMO chegar lá.
>
> (pausa de 2 segundos)
>
> O Mapa de Dados é a mesma coisa, só que pro seu negócio.
>
> Ele não cria dados novos. Não inventa informação.
>
> Ele só organiza e mostra ONDE cada dado está. QUEM é responsável. E o que tá FALTANDO.
>
> É seu GPS de informação.

**[MOSTRAR NA TELA:]**

```
GPS DA CIDADE              MAPA DE DADOS
     |                          |
     v                          v
Mostra onde são            Mostra onde estão
as ruas e lugares          seus dados

Calcula a melhor           Mostra o caminho mais
rota                       rápido pra achar

Avisa de problemas         Revela os gaps e
no caminho                 problemas escondidos
```

---

## [O FRAMEWORK] - 4 minutos

**[PROFESSOR DIZ:]**

> O Mapa de Dados divide seus dados em 4 categorias.
>
> Anota essas 4.

**[MOSTRAR NA TELA:]**

```
AS 4 CATEGORIAS DO MAPA DE DADOS

┌─────────────────┬─────────────────┐
│   FINANCEIRO    │    COMERCIAL    │
│                 │                 │
│ - Faturamento   │ - Leads         │
│ - Custos        │ - Conversão     │
│ - Lucro         │ - Ticket médio  │
│ - Fluxo caixa   │ - CAC           │
└─────────────────┴─────────────────┘
┌─────────────────┬─────────────────┐
│   OPERACIONAL   │     CLIENTE     │
│                 │                 │
│ - Entregas      │ - NPS           │
│ - Produtividade │ - Churn         │
│ - Retrabalho    │ - Reclamações   │
│ - Estoque       │ - Satisfação    │
└─────────────────┴─────────────────┘
```

**[PROFESSOR DIZ:]**

> Financeiro: tudo que é dinheiro. Quanto entra, quanto sai, quanto sobra.
>
> Comercial: tudo que é venda. Leads, conversão, quanto cada cliente gasta.
>
> Operacional: tudo que é execução. Entregas, produtividade, problemas.
>
> Cliente: tudo que é satisfação. Se o cliente tá feliz, se tá saindo, se tá reclamando.
>
> Todo negócio tem dados nessas 4 áreas. A questão é: você sabe onde eles estão?

---

## [A ESTRUTURA] - 2 minutos

**[PROFESSOR DIZ:]**

> Pra cada dado, você vai responder 6 perguntas.

**[MOSTRAR NA TELA:]**

```
ESTRUTURA DO MAPA DE DADOS

1. QUAL DADO? (ex: Faturamento mensal)
2. ONDE ESTÁ? (ex: Planilha do financeiro)
3. QUEM CUIDA? (ex: Maria)
4. QUAL FREQUÊNCIA? (ex: Atualiza diário)
5. QUAL A NOTA? (1 a 5 de confiabilidade)
6. OBSERVAÇÕES? (ex: Às vezes atrasa)
```

**[PROFESSOR DIZ:]**

> É simples. Mas poderoso.
>
> Porque quando você preenche isso... você descobre coisas.
>
> "Opa, meu lucro líquido não está em lugar nenhum."
>
> "Opa, ninguém é responsável pelo NPS."
>
> "Opa, esse dado aqui tem nota 2 de confiabilidade. Não posso confiar nele."
>
> O Mapa revela o que tá escondido.
