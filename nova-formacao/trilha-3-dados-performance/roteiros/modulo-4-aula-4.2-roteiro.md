## [A PIRÂMIDE DE MATURIDADE] - 3 minutos

**[PROFESSOR DIZ:]**

> Existe uma pirâmide de maturidade de dados.
>
> São 4 níveis.

**[MOSTRAR NA TELA:]**

```
PIRÂMIDE DE MATURIDADE DE DADOS

         /\
        /  \
       /    \
      / 4.   \    PREVISÃO
     / PREVER \   "O que VAI acontecer?"
    /──────────\
   / 3. ANALISAR\  ANÁLISE
  / "Por que      \ "Por que aconteceu?"
 /   aconteceu?"    \
/────────────────────\
| 2. VISUALIZAR      | VISUALIZAÇÃO
| "O que aconteceu?" | ← Você está aqui
|────────────────────|
| 1. COLETAR         | DADOS
| "Onde estão os dados?" |
└────────────────────┘
```

**[PROFESSOR DIZ:]**

> Nível 1: Coletar. Você sabe onde estão os dados. Fez isso no Módulo 1.
>
> Nível 2: Visualizar. Você vê o que aconteceu. Fez isso no Módulo 2.
>
> Nível 3: Analisar. Você entende POR QUE aconteceu. Aqui a maioria trava.
>
> Nível 4: Prever. Você sabe o que VAI acontecer. Pouquíssimos chegam.
>
> (pausa de 2 segundos)
>
> Se você está no Nível 2, você está melhor que 80% das empresas.
>
> Mas o salto pro Nível 3 é onde a mágica acontece.
>
> É onde dado vira decisão.

---

## [POR QUE AS EMPRESAS TRAVAM] - 2 minutos

**[PROFESSOR DIZ:]**

> Por que a maioria trava no Nível 2?
>
> Porque o Nível 3 tradicionalmente exige analista de dados.
>
> Alguém que olha os números, faz cruzamentos, identifica padrões.
>
> Isso custa R$5-15 mil por mês.
>
> Pequena e média empresa não tem esse budget.
>
> Então fica no "vejo o número mas não sei o que fazer".

**[MOSTRAR NA TELA:]**

```
CUSTO TRADICIONAL DO NÍVEL 3

Analista de dados júnior:    R$ 5.000/mês
Analista de dados pleno:     R$ 10.000/mês
Consultor de BI:             R$ 15.000/mês

Pequena empresa: "Não tenho esse orçamento"
              ↓
Fica travado no Nível 2
```

## [A MUDANÇA DE JOGO] - 2 minutos

**[PROFESSOR DIZ:]**

> Aqui entra a IA.
>
> Claude, ChatGPT, Gemini... essas ferramentas fazem análise de dados.
>
> Você cola uma tabela, faz uma pergunta, e recebe insight.
>
> Custo? R$100/mês no máximo. Ou grátis nas versões básicas.
>
> (pausa de 2 segundos)
>
> Você não precisa mais de analista pra chegar no Nível 3.
>
> Precisa de PROMPTS certos.
>
> E é isso que eu vou te ensinar nesse módulo.

**[MOSTRAR NA TELA:]**

```
NOVA REALIDADE

Analista humano:     R$ 5.000 - 15.000/mês
IA como analista:    R$ 0 - 100/mês

Mesma função:
- Olhar dados
- Identificar padrões
- Explicar o que significa
- Sugerir ações

ROI: 50x - 150x de economia
```
