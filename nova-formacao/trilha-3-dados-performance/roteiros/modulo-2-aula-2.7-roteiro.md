## [3 NÍVEIS DE AUTOMAÇÃO] - 2 minutos

**[PROFESSOR DIZ:]**

> Existem 3 níveis de automação pra seu dashboard.

**[MOSTRAR NA TELA:]**

```
3 NÍVEIS DE AUTOMAÇÃO

NÍVEL 1: SEMI-AUTOMÁTICO
→ Você atualiza a planilha manualmente
→ Dashboard atualiza sozinho
→ Trabalho: 5-10 min/dia

NÍVEL 2: AUTOMÁTICO
→ Planilha puxa dados de outras fontes
→ Fórmulas IMPORTRANGE, QUERY, Google Forms
→ Trabalho: 0 min/dia (só revisar 1x/semana)

NÍVEL 3: INTEGRADO
→ n8n, Zapier, Make conectam seus sistemas
→ Dados fluem automaticamente
→ Trabalho: 0 (só setup inicial)
```

**[PROFESSOR DIZ:]**

> Onde você está agora?
>
> Se você criou o dashboard conectado na planilha, você está no Nível 1.
>
> Isso já é ótimo. Você economiza o trabalho de criar visualizações.
>
> Mas ainda precisa atualizar a planilha.
>
> (pausa)
>
> O Nível 2 é usar fórmulas pra puxar dados automaticamente.
>
> Por exemplo: IMPORTRANGE puxa dados de outra planilha.
>
> Se seu CRM exporta pra Sheets, você pode puxar direto.
>
> (pausa)
>
> O Nível 3 é usar automação tipo n8n.
>
> Vamos ver isso no próximo módulo.
