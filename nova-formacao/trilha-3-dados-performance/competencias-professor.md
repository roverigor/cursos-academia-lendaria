# Perfil de Competencias - Professor Trilha 3

## Dados e Performance

---

**Documento para:** Time de Recrutamento / RH
**Cargo:** Professor/Instrutor de Curso Online
**Trilha:** Dados e Performance - De dados dispersos a decisoes automaticas
**Carga Horaria do Curso:** ~6h15min (35 aulas em 5 modulos)
**Modalidade:** EAD Assincrono (gravacao de video-aulas)

---

## 1. RESUMO DO CURSO

Este curso ensina **empresarios de PMEs** a:
- Mapear todas as fontes de dados do negocio
- Criar dashboard automatizado com metricas essenciais
- Configurar alertas inteligentes via WhatsApp
- Usar IA como analista de dados pessoal
- Implementar rotina diaria de 15 minutos para decidir com dados

**Publico-alvo:** Donos de PMEs, gestores, empreendedores (30-55 anos)

---

## 2. COMPETENCIAS TECNICAS OBRIGATORIAS

### 2.1 Analise de Dados e Metricas

| Competencia | Nivel | Descricao |
|-------------|-------|-----------|
| Metricas de Negocio | Avancado | Dominio de KPIs essenciais: faturamento, conversao, churn, ticket medio, CAC, LTV |
| Google Sheets | Avancado | Formulas, formatacao condicional, conexao com outras ferramentas |
| Looker Studio | Intermediario | Criacao de dashboards, conexao de fontes, visualizacoes |
| Interpretacao de Dados | Avancado | Capacidade de traduzir numeros em insights acionaveis |
| Automacao de Dados | Intermediario | Fluxos de atualizacao automatica, APIs basicas |

### 2.2 Automacao e Alertas

| Competencia | Nivel | Descricao |
|-------------|-------|-----------|
| n8n ou Similar | Intermediario | Criar workflows de automacao, webhooks, triggers |
| Evolution API/WhatsApp | Basico | Configurar alertas via WhatsApp Business |
| Logica de Alertas | Avancado | Definir thresholds, regras de disparo, priorizacao |

### 2.3 Inteligencia Artificial Aplicada

| Competencia | Nivel | Descricao |
|-------------|-------|-----------|
| ChatGPT/Claude | Avancado | Usar LLMs para analise de dados e geracao de insights |
| Prompt Engineering | Avancado | Criar prompts eficazes para analise de negocios |
| Contextualizacao | Avancado | Configurar IA com contexto de negocio para respostas precisas |

## 5. COMPETENCIAS COMPORTAMENTAIS

### 5.1 Perfil Ideal

| Competencia | Por Que Importa |
|-------------|-----------------|
| Didatica para Leigos | Empresarios nao sao analistas de dados |
| Orientacao para Resultado | Foco em decisoes, nao em metricas bonitas |

## 9. RESUMO EXECUTIVO PARA RH

**Estamos buscando um profissional que:**

- Domine analise de dados e dashboards (Looker Studio, Google Sheets)
- Saiba explicar metricas de forma simples para empresarios
- Tenha experiencia pratica com automacoes (n8n, WhatsApp)
- Conheca e use IA para analise de dados
- Tenha boa presenca de camera e didatica clara

**Perfil ideal:** Analista de dados ou consultor de BI com experiencia em PMEs, que ja ensinou ou palestrou para publico nao-tecnico, com facilidade para gravar video-aulas.

**Formacao minima:** Superior em Administracao, Engenharia, Sistemas de Informacao, Estatistica ou areas correlatas.

**Certificacoes desejaveis:** Google Analytics, Looker Studio, Power BI, certificacoes de dados.
