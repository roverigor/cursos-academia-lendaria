## Stack Tecnológica

### Stack Principal (Ensinada)

| Função | Ferramenta | Custo | Por que essa? |
|--------|------------|-------|---------------|
| Organização de dados | Google Sheets | Grátis | Já conhece, colaborativo |
| Dashboard | Looker Studio | Grátis | Conecta com Sheets, visual |
| Automação/Alertas | n8n | Grátis* | Open source, WhatsApp |
| WhatsApp | Evolution API | Grátis* | Self-hosted, sem mensalidade |
| IA para análise | Claude | ~R$ 100/mês | Melhor para análise técnica |

*Requer hospedagem (Render/Railway = R$ 0-30/mês)

### Custo Total para Aluno

| Perfil | Custo Mensal |
|--------|--------------|
| Mínimo (tudo grátis) | R$ 0 |
| Recomendado (com Claude) | R$ 100/mês |
| Completo (hospedagem paga) | R$ 130/mês |

### Alternativas por Contexto

| Se o aluno... | Usar em vez de... |
|---------------|-------------------|
| Já usa Microsoft | Power BI, Excel, Power Automate |
| Não quer hospedar | Make (R$ 50/mês) em vez de n8n |
| Não quer pagar IA | ChatGPT Free ou Gemini |
| Quer máximo controle | Metabase self-hosted |

## Diferenciais Competitivos

### Por que NÃO é "mais um curso de Power BI"

| Cursos de BI | Esta Trilha |
|--------------|-------------|
| Ensina ferramenta | Ensina decisão |
| 40-80 horas | 6-8 horas |
| Termina no dashboard | Vai até rotina de decisão |
| Power BI (R$ 60/mês) | Stack grátis |
| Para analistas | Para donos de negócio |
| Certificado de que assistiu | Entregáveis funcionando |

### Diferenciais Únicos

| Diferencial | Descrição |
|-------------|-----------|
| **Módulo de Alertas** | Único curso que ensina alertas proativos |
| **IA como Analista** | Não é curso de ChatGPT genérico, é IA para dados |
| **Rotina de 15 min** | Ensina o hábito, não só a ferramenta |
| **Stack 100% grátis** | Opção de custo zero real |
| **Problema-Primeiro** | Começa pela dor, não pela ferramenta |
