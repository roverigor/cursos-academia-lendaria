## Prompt IA: Análise de Gaps

```
Sou dono de uma empresa de [TIPO DO NEGÓCIO] com faturamento de [VALOR]/mês.
Temos [NÚMERO] funcionários e nosso principal desafio é [DESAFIO].

Aqui está meu Mapa de Dados atual:

DADOS FINANCEIROS:
[Colar dados financeiros preenchidos]

DADOS COMERCIAIS:
[Colar dados comerciais preenchidos]

DADOS DE CLIENTE:
[Colar dados de cliente preenchidos]

DADOS DE MARKETING:
[Colar dados de marketing preenchidos]

DADOS OPERACIONAIS:
[Colar dados operacionais preenchidos]

Analise meu mapa e me diga:

1. GAPS CRÍTICOS: Quais dados estão faltando que TODO negócio do meu tipo deveria ter?

2. DUPLICAÇÕES: Existem dados sendo coletados em mais de um lugar? Como consolidar?

3. QUALIDADE: Para os dados marcados como "Parcial" ou "Inexistente", qual a ordem de prioridade para resolver?

4. DECISÕES CEGAS: Baseado no meu mapa, quais decisões importantes estou tomando SEM dados adequados?

5. QUICK WIN: Qual é o PRIMEIRO dado que devo resolver para ter maior impacto imediato?

Seja específico e prático. Não quero teoria, quero ação.
```

## Checklist de Validação

- [ ] Preenchi pelo menos 10 linhas de dados
- [ ] Identifiquei todas as fontes de dados (sistemas, planilhas, plataformas)
- [ ] Classifiquei a qualidade de cada dado
- [ ] Conectei cada dado a uma decisão de negócio
- [ ] Identifiquei pelo menos 3 gaps
- [ ] Rodei o prompt de IA para validar
- [ ] Defini responsável para resolver os gaps prioritários
