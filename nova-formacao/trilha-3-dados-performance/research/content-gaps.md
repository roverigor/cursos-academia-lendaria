# Gaps de Conteúdo: Trilha 3 - Dados e Performance

**Data:** 2025-12-29
**Baseado em:** Análise de 40+ cursos concorrentes

---

## 1. Gaps Identificados no Mercado

### Gap 1: IA como Analista de Dados Interno

| O que o mercado oferece | O que falta |
|------------------------|-------------|
| Cursos de ChatGPT genéricos | IA analisando **dados do próprio negócio** |
| Prompts para pesquisa de mercado | Prompts para **interpretar métricas internas** |
| IA para criação de conteúdo | IA para **diagnóstico operacional** |

**Oportunidade:** Ensinar empresário a usar IA para:
- Analisar dashboard e gerar insights automáticos
- Identificar anomalias e explicar causas
- Fazer previsões baseadas em histórico

**Relevância Domo (2024):** "72% das organizações globais já integraram IA em pelo menos uma função de negócio." Porém, PMEs brasileiras ainda não sabem COMO fazer isso com seus dados.

---

### Gap 2: Conexão Dados → Decisão

| O que o mercado oferece | O que falta |
|------------------------|-------------|
| Como criar dashboard | **Qual decisão** o dashboard informa |
| Como configurar KPI | **Quando agir** baseado no KPI |
| Como coletar dados | **O que fazer** com os dados |

**Oportunidade:** A Trilha 3 já endereça isso com:
- Módulo 1: Mapa de Dados conectando dado → decisão
- Módulo 5: Rotina de 15 min/dia para decidir

**Validação:** Nenhum curso concorrente ensina **rotina de decisão** sistemática.

---

### Gap 3: Alertas Proativos (Não só Dashboard)

| O que o mercado oferece | O que falta |
|------------------------|-------------|
| Dashboards (retrovisqr) | Alertas (farol) |
| Visualização do passado | Prevenção de problemas |
| Relatórios semanais | Notificações em tempo real |

**Oportunidade:** Módulo 3 (Alertas Inteligentes) é **único no mercado**:
- Limite crítico antes de virar crise
- Tendência negativa antes de colapsar
- Oportunidade de lead quente parado

**Validação:** Cursos de BI ignoram alertas. Cursos de automação não conectam com decisão.

---

### Gap 4: Ferramentas Acessíveis (Não Power BI)

| O que o mercado oferece | O que falta |
|------------------------|-------------|
| Power BI (licença cara) | Alternativas gratuitas |
| Tableau (muito técnico) | Ferramentas simples |
| Soluções enterprise | Soluções para PME |

**Oportunidade:** Trilha 3 usa:
- **Metabase** (open source, gratuito)
- **Looker Studio** (Google, gratuito)
- **Google Sheets** (básico, funcional)
- **Supabase** (banco de dados gratuito)

**Validação:** Pouquíssimos cursos ensinam Metabase. Nenhum ensina stack 100% gratuita para PME.

---

### Gap 5: Automação de Coleta para Não-Técnicos

| O que o mercado oferece | O que falta |
|------------------------|-------------|
| Cursos de n8n técnicos | n8n para **empresário** |
| Automação para desenvolvedores | Automação para **gestores** |
| 85h de conteúdo | 2h focada em **coleta de dados** |

**Oportunidade:** Módulo 2 ensina apenas o necessário:
- Coletar dados de CRM, Google, Meta
- Consolidar em Supabase
- Atualizar dashboard automaticamente

**Validação:** Cursos como Autotic (85h) são overkill para quem só quer automatizar coleta.

---

### Gap 6: Mapeamento de Dados Existentes

| O que o mercado oferece | O que falta |
|------------------------|-------------|
| Como criar novos dados | Como **usar dados que já tem** |
| Implementar data warehouse | **Inventário** do que existe |
| Big Data | Small Data da PME |

**Oportunidade:** Módulo 1 (Mapa de Dados) é diferencial:
- O que você JÁ TEM (e não usa)
- Onde cada dado vive
- Quais gaps precisam ser resolvidos

**Validação:** Nenhum curso começa por "mapear o que existe". Todos assumem que precisa criar do zero.

---

### Gap 7: Integração Completa (End-to-End)

| O que o mercado oferece | O que falta |
|------------------------|-------------|
| Curso só de dashboard | Sistema completo |
| Curso só de automação | Dados → Dashboard → Alerta → IA → Decisão |
| Curso só de IA | Jornada integrada |

**Oportunidade:** Trilha 3 é a **única** que integra:
1. Mapeamento → 2. Coleta → 3. Dashboard → 4. Alertas → 5. IA → 6. Rotina

**Validação:** Empresário teria que fazer 5+ cursos para ter o mesmo resultado.
