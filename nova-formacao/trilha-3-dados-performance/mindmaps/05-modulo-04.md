Modulo 4: Analista de Dados com IA
	Objetivo
		Usar IA como analista de dados pessoal
		Interpretar numeros e gerar insights
	Problema
		Gap entre TER dados e ENTENDER dados
		A pergunta: E dai? O que fazer?
		Analista caro (R$5-15K/mes)
	Solucao
		IA como analista por R$100/mes
		Contexto de negocio configurado
		4 tipos de analise prontos
	Piramide de Maturidade
		Nivel 1: Dados
			Tem os numeros
		Nivel 2: Visualizacao
			Ve os numeros
		Nivel 3: Analise
			Entende os numeros
		Nivel 4: Previsao
			Antecipa os numeros
