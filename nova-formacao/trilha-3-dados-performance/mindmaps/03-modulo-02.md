Modulo 2: Dashboard Automatizado
	Objetivo
		Criar dashboard que se atualiza sozinho
		Visualizar 5-7 metricas essenciais
	Problema
		Dashboards que ninguem usa
		Atualizacao manual = desatualizado
		Muitas metricas = paralisia
	Solucao
		Dashboard focado no essencial
		Automacao com Looker Studio
		Cores condicionais
	Framework
		5-7 Metricas que Importam
			Criterios de Selecao
				Impacto no DRE
				Frequencia de mudanca
				Acionabilidade
			Tipos
				Saude: Faturamento, Margem
				Crescimento: Leads, Conversao
				Retencao: Churn, NPS
				Operacao: SLA, Eficiencia
	Ferramenta
		Looker Studio
			Gratuito
			Conecta com Google Sheets
			Atualizacao automatica
			Cores condicionais
		Alternativas
			Power BI
			Metabase
			Notion
	3 Niveis de Automacao
		Nivel 1: Manual
			Atualiza planilha manualmente
			Dashboard puxa dados
		Nivel 2: Semi-automatico
			Exporta CSV periodicamente
			Importa para planilha
		Nivel 3: Totalmente automatico
			API conecta fonte direta
			Zero intervencao humana
	Aulas
		2.1 Quick Win: O Dashboard que ninguem usa (5 min)
		2.2 Conceito: Dados sem visualizacao (10 min)
		2.3 Conceito: 5-7 metricas que importam (10 min)
		2.4 Ferramenta: Looker Studio setup (10 min)
		2.5 Demo: Criando dashboard ao vivo (15 min)
		2.6 Exercicio: Monte seu dashboard (20 min)
		2.7 Validacao: Automacao + Hook M3 (5 min)
	Entregavel
		Dashboard com 5-7 metricas + cores condicionais
	Duracao: 75 minutos
