Modulo 1: Mapa de Dados
	Objetivo
		Mapear todas as fontes de dados do negocio
		Identificar gaps criticos
	Problema
		Dados dispersos em 5-15 lugares
		Tempo perdido procurando informacoes
		Cemiterio de planilhas e sistemas
	Solucao
		Mapa de Dados centralizado
		Inventario completo de fontes
		Classificacao por categoria
	Framework
		4 Categorias de Dados
			Financeiro
				Faturamento
				Custos
				Fluxo de caixa
			Comercial
				Leads
				Conversao
				Ticket medio
			Cliente
				Churn
				NPS
				Tempo de vida
			Operacional
				Entregas
				Tempo de resposta
				Eficiencia
	Template
		6 Colunas
			Dado
			Fonte
			Frequencia
			Responsavel
			Formato
			Gap/Problema
	Aulas
		1.1 Quick Win: Calculo do tempo perdido (5 min)
		1.2 Conceito: O Cemiterio dos Dados (10 min)
		1.3 Conceito: O Mapa - GPS de Informacao (10 min)
		1.4 Ferramenta: Template de 6 colunas (10 min)
		1.5 Demo: Mapeamento ao vivo (15 min)
		1.6 Exercicio: Construa seu mapa (20 min)
		1.7 Validacao: Checklist + Hook M2 (5 min)
	Entregavel
		Mapa de Dados com 10+ itens mapeados
	Duracao: 75 minutos
