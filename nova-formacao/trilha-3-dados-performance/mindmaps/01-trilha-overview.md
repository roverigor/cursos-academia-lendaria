Trilha 3: Dados e Performance - Overview
	Promessa
		De dados dispersos a decisoes automaticas em ~6 horas
	Problema que Resolve
		Dados fragmentados em 5-15 lugares
		4-8h/semana juntando informacoes
		Decisoes no feeling por falta de visibilidade
		Problemas descobertos tarde demais
	Solucao
		Sistema integrado de inteligencia de negocios
		Dashboard automatico + Alertas + IA + Rotina
	5 Modulos
		M1: Mapa de Dados
			Inventario de todas as fontes
			Identificar gaps criticos
		M2: Dashboard Automatizado
			5-7 metricas que importam
			Atualizacao automatica
		M3: Alertas Inteligentes
			Notificacao antes da crise
			WhatsApp via n8n
		M4: Analista IA
			Interpretacao automatica
			4 tipos de analise
		M5: Rotina de Decisao
			15 minutos diarios
			Framework OIAD
	Entregaveis Finais
		Mapa de Dados preenchido
		Dashboard funcional
		3 alertas configurados
		4 prompts calibrados
		Rotina OIAD + Tracker 21 dias
	Investimento
		Tempo: ~6h15min de conteudo
		Custo ferramentas: R$0-130/mes
		ROI: Economia de R$5-15K/mes vs analista
	Metodologia
		GPS Framework: Destino > Origem > Rota
		Didatica Lendaria: Analogias + Diagramas
		Microlearning: 5-20 min por aula
		70% pratica / 30% teoria
