Modulo 3: Alertas Inteligentes
	Objetivo
		Ser notificado antes do problema virar crise
		Configurar 3 tipos de alerta
	Problema
		Dashboard e retrovisor (mostra passado)
		Descobrir problema tarde demais
		Custo de reagir vs prevenir
	Solucao
		Alertas sao farol (antecipam)
		Notificacao proativa no WhatsApp
		3 tipos de alerta complementares
	Framework
		3 Alertas Essenciais
			Alerta de Crise
				Acao imediata
				Threshold critico ultrapassado
				Ex: Churn > 5%, Estoque zerado
			Alerta de Tendencia
				Atencao necessaria
				Padrao preocupante
				Ex: Conversao caindo 3 dias seguidos
			Alerta de Meta
				Acompanhamento
				Progresso vs objetivo
				Ex: Atingiu 80% da meta semanal
	Stack Tecnica
		n8n
			Automacao open-source
			Self-hosted gratuito
			Visual, sem codigo
		Evolution API
			WhatsApp gratuito
			Alternativa a Z-API
			Integracao com n8n
		Alternativas
			Zapier (pago)
			Make (pago)
			Z-API (pago)

		3.6 Exercicio: Configure 3 alertas (20 min)
		3.7 Validacao: Teste real + Hook M4 (5 min)
	Entregavel
		3 alertas funcionando (Crise, Tendencia, Meta)
	Duracao: 75 minutos
