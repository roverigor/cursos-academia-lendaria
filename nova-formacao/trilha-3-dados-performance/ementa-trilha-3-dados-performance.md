## Objetivo Geral

Capacitar o empresario a tomar decisoes baseadas em dados, usando sistema automatizado de coleta, visualizacao, alertas e analise por IA — saindo de dados dispersos para decisoes confiantes em ~6 horas.

---

## Objetivos Especificos

Ao final do curso, o aluno sera capaz de:

1. Mapear todas as fontes de dados do negocio e identificar gaps criticos
2. Criar dashboard automatizado com 5-7 metricas que se atualizam sozinhas
3. Configurar alertas inteligentes que notificam antes do problema virar crise
4. Usar IA como analista de dados pessoal para interpretar numeros e gerar insights
5. Implementar rotina diaria de 15 minutos para decidir com dados
6. Reduzir tempo gasto juntando dados de 4h/semana para 15min/dia

## Metodologia

- **GPS Framework**: Goal (30-60s) → Position (60-90s) → Steps
- **Didatica Lendaria**: 5 Porques, Semiotica (analogias, diagramas), Regra de Ouro (max 2-3 conceitos)
- **Microlearning**: 5-20 min por aula, objetivo unico por aula
- **Hooks Novela**: Cada aula termina com gancho pra proxima
- **Acao Rapida**: Toda aula tem micro-acao de 2 min
