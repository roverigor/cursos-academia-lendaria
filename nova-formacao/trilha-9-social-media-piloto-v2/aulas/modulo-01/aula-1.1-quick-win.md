## PARTE 2: 1 POST COM IA (5 min)

### Prompt Pronto

```
Crie 1 post para Instagram.

NICHO: [seu nicho]
PUBLICO: [seu ICP - ex: empresarios PME]
OBJETIVO: [engajamento/leads]
TOM: [profissional/casual]

ESTRUTURA:
- Gancho (1 frase que para o scroll)
- Corpo (3-5 pontos ou historia)
- CTA (acao especifica)
- 5 hashtags

FORMATO: Texto para carrossel de 5 slides
```

### Revisao Rapida

- [ ] Gancho chama atencao?
- [ ] Corpo entrega valor?
- [ ] CTA esta claro?
