# Aula 1.3: Exercicio - 10 Posts com IA

## Modulo 1 | 25 minutos | Pratica

---

**Crie seu banco de conteudo para as proximas 2 semanas.**

---

## PROMPT PRINCIPAL

Cole no ChatGPT/Claude:

```
Gere 10 posts para Instagram seguindo a Piramide de Conteudo:

CONTEXTO:
- Nicho: [seu nicho]
- Publico: [seu ICP]
- Tom: [profissional/casual/inspirador]
- Minha oferta principal: [produto/servico]

DISTRIBUICAO:
- 5 posts TOPO (atrair desconhecidos)
- 3 posts MEIO (gerar confianca)
- 2 posts FUNDO (converter)

PARA CADA POST:
1. Etapa (T/M/F)
2. Gancho (primeira frase impactante)
3. Corpo (3-5 pontos ou mini-historia)
4. CTA
5. Formato sugerido (carrossel/reels/imagem)
6. 5 hashtags

Organize em tabela.
```

---

## PASSO A PASSO (25 min)

### Bloco 1: Gerar (10 min)

1. Cole o prompt na IA
2. Preencha os campos
3. Copie o resultado

### Bloco 2: Revisar (10 min)

Para cada post, verifique:
- [ ] Gancho para o scroll?
- [ ] Corpo entrega valor?
- [ ] CTA claro?

Ajuste o que precisar.

### Bloco 3: Organizar (5 min)

Organize no seu banco:

| # | Etapa | Gancho | Formato | Status |
|---|-------|--------|---------|--------|
| 1 | Topo | | Carrossel | Pronto |
| 2 | Topo | | Reels | Pronto |
| 3 | Topo | | Imagem | Pronto |
| 4 | Topo | | Carrossel | Pronto |
| 5 | Topo | | Carrossel | Pronto |
| 6 | Meio | | Carrossel | Pronto |
| 7 | Meio | | Reels | Pronto |
| 8 | Meio | | Carrossel | Pronto |
| 9 | Fundo | | Carrossel | Pronto |
| 10 | Fundo | | Imagem | Pronto |

---

## PROMPT BONUS: Melhorar Gancho

Se algum gancho ficou fraco:

```
Melhore este gancho para Instagram:
"[gancho atual]"

Crie 3 versoes:
1. Com curiosidade
2. Com controversia
3. Com numero/dado

Escolha o mais forte.
```
