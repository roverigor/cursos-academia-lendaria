## Objetivo Geral

Capacitar o empresario a montar sua infraestrutura de IA e automacao do zero, usando ferramentas de nivel enterprise a custo de PME, sem depender de desenvolvedor ou agencia.

---

## Objetivos Especificos

Ao final da trilha, o aluno sera capaz de:

1. Configurar ambiente n8n funcional para automacoes
2. Implementar banco de dados Supabase com API
3. Gerenciar VPS propria para hospedagem
4. Integrar LLMs (ChatGPT, Claude, Gemini) ao stack
5. Criar automacoes end-to-end funcionais
6. Desenvolver stack completo integrado (n8n + Supabase + LLMs)

# CONTEUDO PROGRAMATICO

## Modulo 1: Stack Lendario (1 aula)

- Por que n8n + Supabase + VPS + LLMs
- Comparativo de custos: tradicional vs Stack Lendario
- Criacao de contas nas plataformas
- Selecao de processo para automatizar

## Modulo 2: VPS e Infraestrutura (1 aula)

- VPS vs Cloud tradicional
- Provedores recomendados (Hostinger, Contabo, Hetzner)
- Instalacao de n8n com Docker
- Configuracao de dominio e HTTPS

## Modulo 3: Supabase Database (1 aula)

- PostgreSQL vs planilhas/Airtable
- Criacao de projeto e tabelas
- API REST e autenticacao
- Modelagem de dados para negocios

## Modulo 4: n8n na Pratica (1 aula)

- Interface e conceitos basicos
- Automacao 1: Lead WhatsApp → Supabase
- Automacao 2: Form → Email + CRM
- Automacao 3: Relatorio Diario automatico

## Modulo 5: LLMs Integrados (1 aula)

- Comparativo: GPT-4o vs Claude vs Gemini
- Configuracao de APIs no n8n
- Classificacao inteligente de leads
- Respostas automaticas personalizadas
- Gestao de custos de LLM

## Projeto Final

Stack Lendario completo: n8n + Supabase + LLMs integrados em fluxo real.
