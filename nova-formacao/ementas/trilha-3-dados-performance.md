**Categoria:** Reducao de Custo
**Nivel:** Intermediario
**Versao:** 1.0

## Objetivo Geral

Capacitar o empresario a tomar decisoes baseadas em dados, criar dashboards automaticos, identificar gargalos de performance e otimizar operacoes com IA analitica.

---

## Objetivos Especificos

Ao final da trilha, o aluno sera capaz de:

1. Mapear fontes de dados e definir KPIs do negocio
2. Criar dashboards automatizados com metricas-chave
3. Implementar coleta automatica de dados
4. Analisar performance com IA
5. Identificar e corrigir gargalos operacionais
6. Desenvolver sistema de inteligencia de negocios

---

- 5 Aulas (90 min cada)

# CONTEUDO PROGRAMATICO

## Modulo 1: Mapeamento de Dados e KPIs (1 aula)

- Quais dados voce ja tem (e nao usa)
- Definicao de KPIs por area (vendas, marketing, ops, financeiro)
- Fontes de dados: CRM, planilhas, plataformas
- Mapa de Dados do Negocio

## Modulo 2: Coleta Automatizada (1 aula)

- Automacoes de coleta com n8n
- Integracao de APIs (Meta, Google, CRMs)
- Consolidacao em Supabase
- Scheduling e rotinas

## Modulo 3: Dashboards e Visualizacao (1 aula)

- Principios de visualizacao de dados
- Ferramentas: Metabase, Looker Studio, Sheets
- Dashboard operacional vs estrategico
- Alertas automaticos

## Modulo 4: Analise com IA (1 aula)

- Prompts para analise de dados
- Identificacao de padroes e anomalias
- Geracao automatica de insights
- Recomendacoes acionaveis

## Modulo 5: Otimizacao de Performance (1 aula)

- Identificacao de gargalos
- Priorizacao por impacto no DRE
- Experimentos e testes
- Loop de melhoria continua

## Projeto Final

Sistema de Business Intelligence com coleta automatica, dashboard e analise por IA.

# PLANO DE AULAS

## Modulo 1, Aula 1: Mapeamento de Dados

### Objetivos

- Identificar todas as fontes de dados
- Definir KPIs prioritarios
- Criar Mapa de Dados do Negocio

### Conteudo Principal

- Tipos de dados: transacionais, comportamentais, financeiros
- KPIs por area do negocio
- Qualidade vs quantidade de dados

### Atividades Praticas

- Inventario de fontes de dados
- Definicao de 5 KPIs principais
- Mapa de Dados preenchido

### Recursos

- Template Mapa de Dados.xlsx
- Prompt de definicao de KPIs

---

## Modulo 2, Aula 1: Coleta Automatizada

### Objetivos

- Configurar automacoes de coleta
- Integrar APIs de plataformas
- Consolidar dados em banco central

### Conteudo Principal

- APIs de Meta, Google, CRMs
- Webhooks e schedulers no n8n
- Estrutura de dados no Supabase

### Atividades Praticas

- Criar automacao de coleta diaria
- Integrar 2+ fontes de dados
- Validar dados no Supabase

### Recursos

- Workflows de coleta.json
- Prompt de mapeamento de APIs

---

## Modulo 3, Aula 1: Dashboards

### Objetivos

- Criar dashboard operacional
- Configurar alertas automaticos
- Visualizar KPIs em tempo real

### Conteudo Principal

- Principios de data viz
- Metabase vs Looker Studio
- Graficos certos para cada metrica
- Alertas por threshold

### Atividades Praticas

- Criar dashboard com 5+ metricas
- Configurar 2 alertas automaticos

### Recursos

- Template Dashboard.json
- Guia de visualizacao

---

## Modulo 4, Aula 1: Analise com IA

### Objetivos

- Usar IA para analisar dados
- Identificar padroes automaticamente
- Gerar insights acionaveis

### Conteudo Principal

- Prompts de analise de dados
- Deteccao de anomalias
- Correlacoes e tendencias
- Formato de insights

### Atividades Praticas

- Analisar dados da semana com IA
- Gerar relatorio de insights
- Definir 3 acoes prioritarias

### Recursos

- Prompts de Analise.md
- Template de Relatorio de Insights

---

## Modulo 5, Aula 1: Otimizacao

### Objetivos

- Identificar gargalos de performance
- Priorizar por impacto financeiro
- Criar plano de otimizacao

### Conteudo Principal

- Analise de funil e conversao
- Calculo de impacto no DRE
- Framework de experimentos
- Rotina de melhoria continua

### Atividades Praticas

- Identificar top 3 gargalos
- Criar plano de experimentos
- Definir rotina semanal

### Recursos

- Template Plano de Experimentos.xlsx
- Prompt de identificacao de gargalos
