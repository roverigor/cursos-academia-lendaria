### Os 6 Tipos de Criativos

| # | Tipo | Quando Usar | Duracao Ideal |
|---|------|-------------|---------------|
| 1 | **Talking Head** | Autoridade, confianca | 30-60s |
| 2 | **Problema-Solucao** | Awareness, topo funil | 15-30s |
| 3 | **Testemunho** | Prova social | 30-45s |
| 4 | **Demonstracao** | Produto visual | 15-30s |
| 5 | **Lista** | Beneficios rapidos | 15-20s |
| 6 | **Antes/Depois** | Transformacao | 15-30s |

---

### Tipo 1: Talking Head

**Estrutura:**
```
[HOOK] "A maioria [do publico] comete esse erro..."

[BODY] "[Especialista] explica o problema e a solucao"

[CTA] "Link na bio pra [beneficio]"
```

**Quando funciona:** Quando voce (ou cliente) tem autoridade

---

### Tipo 2: Problema-Solucao

**Estrutura:**
```
[HOOK] Visual: pessoa frustrada
        Texto: "Cansado de [problema]?"

[BODY] "Existe uma forma melhor..."
        Mostra a solucao

[CTA] "Clique e descubra como"
```

**Quando funciona:** Topo de funil, publico frio

---

### Tipo 3: Testemunho

**Estrutura:**
```
[HOOK] "[Resultado impressionante]"
        Pessoa real falando

[BODY] "Antes eu [situacao ruim]...
        Depois de [produto] eu [situacao boa]"

[CTA] "Quer o mesmo resultado?"
```

**Quando funciona:** Quebrar ceticismo

---

### Tipo 4: Demonstracao

**Estrutura:**
```
[HOOK] Produto em acao, resultado visual

[BODY] Passo a passo de como funciona

[CTA] "Teste gratis" ou "Veja como"
```

**Quando funciona:** Produto com beneficio visual claro

---

### Tipo 5: Lista de Beneficios

**Estrutura:**
```
[HOOK] "5 razoes para [resultado]"

[BODY] Beneficio 1... âœ…
        Beneficio 2... âœ…
        Beneficio 3... âœ…
        ...

[CTA] "Qual razao mais importa pra voce?"
```

**Quando funciona:** Resumir proposta de valor

---

### Tipo 6: Antes/Depois

**Estrutura:**
```
[HOOK] Split screen: ANTES | DEPOIS

[BODY] Transicao visual da transformacao

[CTA] "Sua vez de transformar"
```

**Quando funciona:** Transformacao visual clara

---

### Qual Tipo Usar?

| Objetivo | Tipo Recomendado |
|----------|------------------|
| Gerar awareness | Problema-Solucao, Lista |
| Construir confianca | Talking Head, Testemunho |
| Mostrar produto | Demonstracao, Antes/Depois |
| Remarcar leads | Testemunho, Lista |
