### Prompt 1: Diagnostico de Funil

```
Aja como analista de growth marketing.

DADOS DO FUNIL (ultimo mes):
[Cole seus numeros]

ANALISE:
1. Taxa de conversao por etapa
2. Compare com benchmarks do setor
3. Identifique o maior gargalo
4. Sugira 3 hipoteses para o problema
5. Recomende 1 acao prioritaria

Formato: tabela + bullet points
```

---

### Prompt 2: Analise de Criativos

```
Aja como especialista em copy e conversao.

CONTEXTO:
- Produto: [seu produto]
- ICP: [seu ICP]
- Resultado atual: CTR [X%]

CRIATIVOS ATUAIS:
[Cole textos/descricoes]

ANALISE:
1. Qual angulo esta sendo usado?
2. Esta alinhado com a dor principal do ICP?
3. O que esta faltando?
4. Sugira 3 variacoes para teste

Formato: analise + sugestoes concretas
```

---

### Prompt 3: Analise de LP

```
Aja como especialista em landing pages.

URL ou DESCRICAO DA LP:
[Cole ou descreva]

DADOS:
- Visitantes: [X]
- Conversoes: [Y]
- Taxa: [Z%]

ANALISE:
1. Headline esta clara?
2. Proposta de valor em 5 segundos?
3. Prova social presente?
4. CTA claro e urgente?
5. 3 melhorias prioritarias

Formato: checklist + recomendacoes
```

---

### Prompt 4: Analise de Email

```
Aja como especialista em email marketing.

SEQUENCIA:
[Cole emails ou descreva]

DADOS:
- Open rate: [X%]
- Click rate: [Y%]
- Conversao: [Z%]

ANALISE:
1. Subjects estao curiosos?
2. Preview text otimizado?
3. CTA claro em cada email?
4. Sequencia tem progressao logica?
5. 3 melhorias prioritarias

Formato: analise por email + sugestoes
```

---

### Prompt 5: Plano de Experimentos

```
Aja como growth hacker.

CONTEXTO:
- Meta: [sua meta]
- Metrica principal: [KPI]
- Orcamento: [R$ ou horas disponiveis]

GARGALO ATUAL:
[Descreva o problema]

GERE:
1. 5 hipoteses de melhoria
2. Avalie cada uma com ICE (1-10)
3. Defina experimento para top 2
4. Sugira cronograma de 2 semanas

Formato: tabela ICE + plano de acao
```
