---

### BLOCO 1: Diagnostico (5 min)

1. Abra ChatGPT
2. Use o Prompt de Diagnostico
3. Cole seus dados reais
4. Anote:
   - Gargalo identificado
   - 3 hipoteses sugeridas

**[PAUSA: 5 minutos]**

---

### BLOCO 2: Gerar Hipoteses (5 min)

1. Peca a IA para expandir hipoteses:
   ```
   Para o gargalo [X], gere mais 5 hipoteses
   de melhoria alem das 3 ja listadas.

   Para cada hipotese, sugira:
   - Acao concreta
   - Metrica de sucesso
   - Tempo de implementacao
   ```

2. Voce deve ter 5-8 hipoteses no total

**[PAUSA: 5 minutos]**

---

### BLOCO 3: Avaliar ICE (5 min)

1. Monte tabela no papel ou planilha:

| Hipotese | I | C | E | ICE |
|----------|---|---|---|-----|
| 1. | | | | |
| 2. | | | | |
| 3. | | | | |
| 4. | | | | |
| 5. | | | | |

2. De nota 1-10 para cada criterio
3. Calcule ICE = (I+C+E)/3
4. Ordene por score

**[PAUSA: 5 minutos]**

---

### BLOCO 4: Montar Rotina (3 min)

**Sua Rotina Semanal de Otimizacao:**

| Dia | Atividade | Tempo |
|-----|-----------|-------|
| Segunda | Review metricas + diagnostico IA | 15 min |
| Quarta | Checar experimentos ativos | 10 min |
| Sexta | Documentar resultados + planejar proximo | 20 min |

**Template de Review Semanal:**
```
SEMANA [X]:
- Experimento ativo: [nome]
- Resultado parcial: [numero]
- Status: Continuar / Pivotar / Escalar
- Proximo experimento: [nome]
```
