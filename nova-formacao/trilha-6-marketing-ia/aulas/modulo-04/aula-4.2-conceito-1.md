### Modelo de Scoring

| Acao/Caracteristica | Pontos |
|---------------------|--------|
| **COMPORTAMENTO** | |
| Baixou material | +10 |
| Abriu email | +5 |
| Clicou no email | +10 |
| Visitou LP | +15 |
| Visitou pricing | +25 |
| Respondeu WhatsApp | +20 |
| Perguntou preco | +30 |
| **PERFIL** | |
| Cargo decisor (dono, diretor) | +25 |
| Empresa tamanho ideal | +15 |
| Setor alvo | +10 |
| **NEGATIVO** | |
| Email invalido | -50 |
| Nao abriu 3+ emails | -10 |
| Disse "nao tenho interesse" | -100 |

---

### Classificacao por Score

| Score | Classificacao | Acao |
|-------|---------------|------|
| 0-30 | FRIO | Nutricao automatica |
| 31-60 | MORNO | SDR contacta |
| 61-80 | QUENTE | Prioridade alta |
| 81+ | MUITO QUENTE | Contato imediato |

---

### Por Que Funciona

> "Ligar pra lead frio = desperdicio de tempo"
>
> "Demorar pra ligar pra lead quente = perda de venda"
>
> Scoring resolve os dois problemas.
