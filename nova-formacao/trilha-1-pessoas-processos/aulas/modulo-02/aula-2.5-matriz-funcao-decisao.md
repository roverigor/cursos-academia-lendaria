## TEMPLATE DA MATRIZ

| Funcao | Decisao | Quem | Freq | Impacto | Tipo | Regra? | IA? |
|--------|---------|------|------|---------|------|--------|-----|
| | | | D/S/M | B/M/A | E/T/O | S/N/P | S/N/P |
| | | | D/S/M | B/M/A | E/T/O | S/N/P | S/N/P |
| | | | D/S/M | B/M/A | E/T/O | S/N/P | S/N/P |

**Legenda:**
- D = Diaria, S = Semanal, M = Mensal
- B = Baixo, M = Medio, A = Alto
- E = Estrategica, T = Tatica, O = Operacional
- S = Sim, N = Nao, P = Parcial

---

## FORMULA DE PRIORIZACAO

```
PRIORIDADE 1 (Automatizar primeiro):
- Frequencia = D (diaria)
- Tipo = O (operacional)
- Tem Regra = S (sim)

PRIORIDADE 2:
- Frequencia = D ou S
- Tipo = O ou T
- Tem Regra = S ou P

NAO AUTOMATIZAR:
- Tipo = E (estrategica)
- Tem Regra = N
```
