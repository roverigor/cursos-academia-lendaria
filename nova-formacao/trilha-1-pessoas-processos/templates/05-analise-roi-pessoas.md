## Instrucoes de Uso

1. Liste todas as funcoes/cargos da empresa
2. Calcule o custo REAL de cada funcao (salario + encargos + ferramentas + tempo)
3. Estime o valor gerado (receita influenciada, custo evitado, tempo liberado)
4. Calcule o ROI de cada funcao
5. Decida: Manter / Automatizar / Reestruturar

## Formulas Essenciais

### Custo Total de uma Funcao

```
CUSTO TOTAL = Salario Bruto
            + Encargos (INSS, FGTS, ferias, 13o) ~70-100%
            + Beneficios (VR, VT, plano saude)
            + Ferramentas (software, equipamento)
            + Espaco (% do aluguel)
            + Gestao (tempo do gestor)
```

### Valor Gerado por uma Funcao

```
VALOR GERADO = Receita Direta (vendas, contratos)
             + Receita Influenciada (leads, conversoes)
             + Custo Evitado (erros prevenidos, multas evitadas)
             + Tempo Liberado (horas x valor/hora do beneficiado)
```

### ROI da Funcao

```
ROI = (Valor Gerado - Custo Total) / Custo Total x 100

ROI > 100% = Funcao gera mais do que custa
ROI = 0-100% = Funcao cobre o custo, margem baixa
ROI < 0% = Funcao custa mais do que gera
```
