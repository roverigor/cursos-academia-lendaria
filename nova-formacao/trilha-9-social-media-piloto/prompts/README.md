# Prompts de IA - Trilha 9: Social Media em Piloto

Colecao completa de prompts para usar com ChatGPT, Claude ou outras IAs.

## Como Usar

1. Copie o prompt desejado
2. Substitua os campos entre colchetes [assim] pelos seus dados
3. Cole no ChatGPT/Claude
4. Ajuste a resposta conforme necessario

## Dicas de Uso

1. **Seja especifico** - Quanto mais contexto, melhor a resposta
2. **Itere** - Peca ajustes se a primeira resposta nao for ideal
3. **Salve os bons** - Guarde prompts que funcionaram para reutilizar
4. **Personalize** - Adapte para seu tom de voz e nicho

## Modelos Recomendados

| Modelo | Melhor Para |
|--------|-------------|
| GPT-4 | Analises complexas, estrategia |
| GPT-3.5 | Tarefas rapidas, volume alto |
| Claude | Textos longos, analise detalhada |
| Gemini | Alternativa gratuita |
