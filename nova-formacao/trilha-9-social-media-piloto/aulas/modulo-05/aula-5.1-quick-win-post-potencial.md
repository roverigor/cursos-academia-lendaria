### Passo 2: Calcule o Score Viral (2 min)

Use esta formula:

```
Score Viral = (Salvamentos x 3) + (Compartilhamentos x 5) + (Comentarios x 2)
```

**Por que esses pesos?**
- Salvamentos (x3) = Conteudo util, voltam para ver
- Compartilhamentos (x5) = Alcance organico, viralidade
- Comentarios (x2) = Engajamento real, algoritmo favorece

## PROMPT DE IA: ANALISE APROFUNDADA

Use este prompt para entender POR QUE o post funcionou:

```
Analise este post que teve alto engajamento.

POST:
- Texto: [cole o texto]
- Formato: [carrossel/reels/imagem]
- Metricas: [alcance, curtidas, salvamentos, compartilhamentos]

IDENTIFIQUE:
1. Qual o gancho que chamou atencao?
2. Qual emocao ele provoca?
3. Por que pessoas salvaram/compartilharam?
4. Qual elemento e replicavel?
5. Como criar uma variacao deste post?

SUGIRA:
- 3 variacoes do mesmo tema
- Melhor horario para repostar
- Se vale impulsionar (sim/nao + por que)
```
