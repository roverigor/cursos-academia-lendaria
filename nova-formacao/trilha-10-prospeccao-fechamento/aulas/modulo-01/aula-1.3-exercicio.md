# Aula 1.3 - Monte Sua Campanha Outbound

## Metadados

| Campo | Valor |
|-------|-------|
| **Tipo** | Exercício |
| **Duração** | 20 minutos |
| **Ferramentas** | ChatGPT/Claude, Planilha, Apollo/LinkedIn |
| **Entregável** | Campanha outbound completa |

---

## Objetivo

Criar uma campanha de prospecção outbound completa: ICP definido, lista de 50 prospects, 5 templates de email e cadência configurada.

---

## Passo 1: Defina Seu ICP para Outbound (3 min)

Preencha:

```
MEU ICP DE OUTBOUND

Cargo: _________________________
Setor: _________________________
Tamanho da empresa: _____________
Dor principal: __________________
Gatilho (momento de compra): ____
Onde encontrar: _________________
```

**Exemplo:**
```
Cargo: Diretor de Marketing / CMO
Setor: E-commerce de moda
Tamanho: 50-200 funcionários
Dor: CAC alto, baixa conversão de tráfego pago
Gatilho: Acabou de levantar investimento / Está contratando
Onde encontrar: LinkedIn, eventos do setor
```

---

## Passo 2: Gere Lista de 50 Prospects (7 min)

### Opção A: LinkedIn + Apollo

1. Busque no LinkedIn pelo cargo + setor
2. Use Apollo.io para encontrar emails
3. Exporte para planilha

### Opção B: Use IA para Listar Empresas

```
Liste 50 empresas brasileiras que se encaixam neste perfil:

- Setor: [seu setor]
- Tamanho: [faixa de funcionários]
- Características: [outras características]

Para cada empresa, inclua:
- Nome da empresa
- Cidade
- Tamanho estimado
- Site (se souber)
```

### Template de Planilha

|---------|---------|-------|-------|----------|--------|
| | | | | | |

## Passo 3: Crie 5 Variações de Cold Email (5 min)

Use o prompt:

```
Crie 5 variações de cold email para o seguinte ICP:

ICP: [cole seu ICP]
Minha solução: [o que você oferece]

Cada variação deve ter:
- Assunto diferente
- Gancho diferente
- Mesma estrutura (gancho, dor, solução, CTA)

Tom: Profissional e direto.
```

### Organize os Templates

| # | Assunto | Uso |
|---|---------|-----|
| 1 | [assunto] | Email inicial |
| 2 | [assunto] | Follow-up 1 |
| 3 | [assunto] | Case/Prova social |
| 4 | [assunto] | Provocação |
| 5 | [assunto] | Break-up |

---

## Passo 4: Monte a Cadência (3 min)

Preencha sua cadência:

| Dia | Canal | Template | Ação |
|-----|-------|----------|------|
| 1 | Email | #1 | Enviar apresentação |
| 3 | Call | Script | Tentar ligar |
| 5 | Email | #2 | Follow-up |
| 8 | Email | #3 | Case de sucesso |
| 10 | Call | Script | Segunda tentativa |
| 12 | Email | #4 | Provocação/Insight |
| 14 | Email | #5 | Break-up |

---

## Passo 5: Crie Script de Cold Call (2 min)

```
SCRIPT DE COLD CALL

[Toca telefone]

"Oi [Nome], aqui é [Seu Nome] da [Empresa].

Mandei um email alguns dias atrás sobre [tema].
Não sei se você viu, mas queria entender rapidinho:

Vocês estão tendo desafios com [dor principal]?

[Se sim] → Agenda call
[Se não] → Agradece e encerra

Posso te roubar 15 minutos essa semana pra mostrar
como estamos ajudando empresas como a [empresa] a [resultado]?"
```
