# Aula 3.3 - Crie Template + Preço

## Metadados

| Campo | Valor |
|-------|-------|
| **Tipo** | Exercício |
| **Duração** | 20 minutos |
| **Entregável** | Template de proposta + 3 pacotes + calculadora |

---

## Parte 1: Defina Seus 3 Pacotes (7 min)

### Passo 1: Liste o que você entrega

```
TUDO QUE VOCÊ PODE ENTREGAR:
1. _________________________
2. _________________________
3. _________________________
4. _________________________
5. _________________________
6. _________________________
7. _________________________
```

### Passo 2: Distribua nos pacotes

| Pacote | O que inclui |
|--------|--------------|
| **Essencial** | Itens 1, 2 (o básico) |
| **Recomendado** | Itens 1, 2, 3, 4, 5 (completo) |
| **Premium** | Todos + exclusivos |

### Passo 3: Defina preços

Use a calculadora:

```
CALCULADORA DE PREÇO

1. Qual o custo do problema para o cliente? (anual)
   R$ ___________

2. Quanto % você resolve/economiza?
   _____ % (típico: 20-50%)

3. Valor que você gera = (1) × (2)
   R$ ___________

4. Seu preço = 10-30% de (3)
   R$ ___________ a R$ ___________

APLICANDO NOS PACOTES:
- Essencial: R$ _______ (piso)
- Recomendado: R$ _______ (1.5x essencial)
- Premium: R$ _______ (2-3x essencial)
```

---

## Parte 2: Crie Template de Proposta (8 min)

### Template Base

```
PROPOSTA COMERCIAL
[Sua Empresa] para [Empresa Cliente]
Data: ___/___/___

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. CONTEXTO

Olá [Nome],

Após nossa conversa, entendi que [resumo do contexto].

[2-3 linhas mostrando que você ouviu]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

2. O DESAFIO

O principal desafio que identificamos:

[Dor específica]

Impacto estimado: R$ [valor] por [período]

[Por que isso é urgente]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

3. NOSSA SOLUÇÃO

Propomos [nome da solução], que inclui:

✓ [Entregável 1]
✓ [Entregável 2]
✓ [Entregável 3]

Diferenciais:
• [Por que somos diferentes]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

4. RESULTADOS ESPERADOS

Com base em projetos similares:

• [Resultado 1] em [prazo]
• [Resultado 2] em [prazo]
• [Resultado 3] em [prazo]

Case: [Cliente similar] alcançou [resultado].

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

5. INVESTIMENTO

□ ESSENCIAL │ R$ _______
  [O que inclui]

□ RECOMENDADO │ R$ _______  ★ Mais escolhido
  [O que inclui]

□ PREMIUM │ R$ _______
  [O que inclui]

Condições: [pagamento, garantia]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

6. PRÓXIMOS PASSOS

1. Aprovação desta proposta
2. Kickoff em até [X] dias
3. Primeiro [marco] em [prazo]

Para avançar: [CTA - email, WhatsApp, link]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## Parte 3: Versão Visual (5 min)

Crie versão em:
- Google Slides
- Canva
- PowerPoint

Dicas:
- 1 seção por slide
- Visual limpo
- Sua marca presente
- Fácil de apresentar

## Checklist de Entrega

- [ ] 3 pacotes definidos com entregáveis claros
- [ ] Preços calculados por valor
- [ ] Template de proposta em texto
- [ ] Template visual (Slides/Canva)
- [ ] Calculadora de preço preenchida
