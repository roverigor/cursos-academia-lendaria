# Aula 5.3 - Monte Seu Plano de Escala

## Metadados

| Campo | Valor |
|-------|-------|
| **Tipo** | Exercício |
| **Duração** | 20 minutos |
| **Entregável** | Plano de escala comercial para 90 dias |

---

## Parte 1: Diagnóstico Atual (5 min)

### Preencha seus números:

```
MÉTRICAS ATUAIS

Clientes ativos: _____
Ticket médio: R$ _____
Faturamento mensal: R$ _____

Taxa de conversão de leads: ____%
Taxa de indicação atual: ____%
Taxa de recompra: ____%

CAC estimado: R$ _____
LTV estimado: R$ _____
Razão LTV:CAC: _____
```

### Diagnóstico:

| Razão LTV:CAC | Situação |
|---------------|----------|
| < 1:1 | Perdendo dinheiro |
| 1-3:1 | Sobrevivendo |
| > 3:1 | Pronto para escalar |

Minha situação: _______________

---

## Parte 2: Crie Plano com IA (10 min)

### Use o prompt:

```
Crie um plano de escala comercial para 90 dias.

Meu contexto:
- Negócio: [descreva]
- Ticket médio: R$ [valor]
- Clientes atuais: [número]
- Meta de faturamento: R$ [valor] em 90 dias
- Equipe: [só eu / X pessoas]

Situação atual:
- Fonte principal de leads: [qual]
- Taxa de conversão: [%]
- Taxa de indicação: [%]
- Maior gargalo: [qual]

Crie um plano com:

1. METAS POR MÊS
   - Mês 1: [foco]
   - Mês 2: [foco]
   - Mês 3: [foco]

2. AÇÕES POR SEMANA
   - Semanas 1-4
   - Semanas 5-8
   - Semanas 9-12

3. INDICADORES PARA ACOMPANHAR
   - 3 métricas principais
   - Frequência de medição
   - Meta para cada uma

4. RISCOS E CONTINGÊNCIAS
   - O que pode dar errado
   - Como mitigar

Seja específico e realista para meu contexto.
```
