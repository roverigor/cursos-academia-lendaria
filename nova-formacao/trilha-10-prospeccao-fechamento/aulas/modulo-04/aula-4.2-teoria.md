# Aula 4.2 - Negociar sem Ceder

## Metadados

| Campo | Valor |
|-------|-------|
| **Tipo** | Teoria Mínima |
| **Duração** | 15 minutos |

---

## Objetivo

Entender os princípios de negociação que protegem margem e aceleram fechamento.

---

## Os 4 Pilares da Negociação

### 1. BATNA (Best Alternative)

```
BATNA = Sua melhor alternativa se não fechar

Quanto melhor seu BATNA, mais poder você tem.

Exemplos de BATNA forte:
- Pipeline cheio de leads
- Outros clientes interessados
- Não precisa deste contrato
- Pode esperar pelo cliente certo
```

**Regra**: Nunca negocie desesperado. Tenha sempre alternativas.

---

### 2. Ancoragem

```
Quem apresenta primeiro, ancora a conversa.

Você: "O investimento é R$ 15.000"
Cliente: Agora ele negocia a partir de 15k

Cliente: "Quanto custa?"
Você: "Depende, qual seu orçamento?"
Cliente: "R$ 5.000"
Você: Agora você negocia a partir de 5k ❌
```

**Regra**: Sempre apresente seu preço primeiro.

---

### 3. Concessões Estratégicas

```
NUNCA DÊ SEM RECEBER

❌ "Posso fazer 10% de desconto"
✅ "Consigo 10% se fecharmos hoje"

❌ "Ok, incluo o bônus"
✅ "Incluo o bônus se você indicar 2 clientes"
```

**Fórmula**: "Consigo [benefício] SE você [condição]"

---

### 4. Fechamento

| Tipo | Frase | Quando usar |
|------|-------|-------------|
| **Assumido** | "Vamos começar na segunda?" | Cliente demonstrou interesse |
| **Alternativa** | "Prefere Essencial ou Recomendado?" | Reduzir opções |
| **Urgência** | "Essa condição vale até sexta" | Acelerar decisão |
| **Resumo** | "Então, resolvemos X, Y, Z. Fechamos?" | Após rebater objeções |

---

## Matriz de Objeções

| Objeção | Tipo | Resposta |
|---------|------|----------|
| "Está caro" | Preço | "Caro comparado a quê?" |
| "Preciso pensar" | Adiamento | "O que especificamente precisa avaliar?" |
| "Concorrente faz por menos" | Comparação | "O que exatamente eles entregam?" |
| "Sem orçamento agora" | Timing | "Quando seria o momento ideal?" |
| "Preciso falar com X" | Decisor | "Podemos agendar uma call com vocês?" |

## O Fluxo Completo

```
┌──────────────┐
│  APRESENTAR  │ ← Você ancora o preço
│    PROPOSTA  │
└──────┬───────┘
       ▼
┌──────────────┐
│   OBJEÇÃO    │ ← Cliente resiste
└──────┬───────┘
       ▼
┌──────────────┐
│  PERGUNTAR   │ ← "O que especificamente?"
└──────┬───────┘
       ▼
┌──────────────┐
│   RESPONDER  │ ← Rebater com fatos/valor
└──────┬───────┘
       ▼
┌──────────────┐
│   NEGOCIAR   │ ← "Consigo X se você Y"
└──────┬───────┘
       ▼
┌──────────────┐
│   FECHAR     │ ← "Vamos começar quando?"
└──────────────┘
```

---

## Resumo

1. **BATNA forte** = mais poder
2. **Ancore primeiro** = controle a conversa
3. **Dê SE receber** = nunca de graça
4. **Feche com confiança** = assuma o sim
