# Aula 4.4 - Simule 3 Cenários

## Metadados

| Campo | Valor |
|-------|-------|
| **Tipo** | Validação |
| **Duração** | 10 minutos |

---

## Objetivo

Testar seu playbook em 3 cenários diferentes com IA.

---

## Cenário 1: O Pechincheiro

Use o prompt:

```
Você é um cliente que:
- Quer desconto a qualquer custo
- Compara com 3 concorrentes
- Diz que não tem orçamento
- Mas claramente precisa do serviço

Meu serviço: [descreva brevemente]
Meu preço: R$ [valor]

Comece a negociação pedindo desconto agressivamente.
Eu vou responder usando meu playbook.
Após 5 trocas, me diga:
1. Se eu mantive minha posição
2. Se ofereci concessões estratégicas
3. Nota de 1-10 para minha negociação
```

---

## Cenário 2: O Procrastinador

Use o prompt:

```
Você é um cliente que:
- Gostou da proposta
- Mas "precisa pensar"
- Sempre adia decisão
- Tem medo de decidir errado

Meu serviço: [descreva brevemente]
Meu preço: R$ [valor]

Comece a negociação dizendo "Gostei, mas preciso pensar".
Eu vou tentar fechar. Você resiste educadamente.
Após 5 trocas, me diga:
1. Se eu descobri o real motivo
2. Se ofereci próximos passos claros
3. Nota de 1-10 para minha técnica
```

---

## Cenário 3: O "Tenho que Falar com..."

Use o prompt:

```
Você é um cliente que:
- Não é o decisor final
- Precisa aprovar com sócio/chefe
- Está interessado pessoalmente
- Mas não pode decidir sozinho

Meu serviço: [descreva brevemente]
Meu preço: R$ [valor]

Comece dizendo "Adorei, mas preciso falar com meu sócio".
Eu vou tentar avançar a negociação.
Após 5 trocas, me diga:
1. Se eu identifiquei o processo de decisão
2. Se propus incluir o decisor
3. Nota de 1-10 para minha abordagem
```

---

## Avaliação Final

Após os 3 cenários, peça:

```
Agora me dê um feedback consolidado:

1. Minha maior força como negociador
2. Minha maior fraqueza
3. Uma técnica específica para melhorar
4. Média das 3 notas

Seja honesto e direto.
```

---

## Checklist de Validação

- [ ] Cenário 1 (Pechincheiro) simulado
- [ ] Cenário 2 (Procrastinador) simulado
- [ ] Cenário 3 (Não-decisor) simulado
- [ ] Feedback consolidado recebido
- [ ] Pontos de melhoria anotados
- [ ] Playbook ajustado com aprendizados
