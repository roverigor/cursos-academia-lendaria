## Parte 2: Crie Respostas com IA (10 min)

### Prompt para cada objeção:

```
Sou um [sua profissão] que vende [seu serviço].

A objeção mais comum que recebo é:
"[cole a objeção aqui]"

Me dê 3 respostas diferentes para essa objeção:

1. Uma resposta que QUESTIONE (descubra mais)
2. Uma resposta que REFRAME (mude a perspectiva)
3. Uma resposta que PROPONHA SOLUÇÃO (ofereça alternativa)

Para cada resposta, explique:
- Por que funciona
- Quando usar
- O que fazer se não funcionar
```

### Template de Resposta:

```
OBJEÇÃO: "[texto da objeção]"

RESPOSTA QUESTIONAR:
"_________________________________"
Uso quando: _____________________

RESPOSTA REFRAME:
"_________________________________"
Uso quando: _____________________

RESPOSTA SOLUÇÃO:
"_________________________________"
Uso quando: _____________________
```

Repita para cada objeção.

---

## Parte 3: Monte Seu Playbook (5 min)

### Organize no formato:

```
PLAYBOOK DE NEGOCIAÇÃO
[Seu Nome] - [Data]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

OBJEÇÃO 1: "Está caro"

→ Questionar: "Caro comparado a quê?"
→ Reframe: "Se resolver [problema], quanto vale?"
→ Solução: "Temos o pacote Essencial por R$..."

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

OBJEÇÃO 2: "Preciso pensar"

→ Questionar: "O que especificamente?"
→ Reframe: "Entendo. O que ajudaria você decidir?"
→ Solução: "Posso agendar call para amanhã?"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[Continue para todas as 5 objeções]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

MINHAS CONCESSÕES PERMITIDAS:

Posso oferecer:
□ Desconto de até ___% se _______
□ Prazo estendido se _______
□ Bônus X se _______
□ Parcelamento se _______

NÃO negocio:
□ _______________________
□ _______________________

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

FRASES DE FECHAMENTO:

1. "Vamos começar na ___?"
2. "Prefere o pacote ___ ou ___?"
3. "Essa condição vale até ___"
4. "Fechamos então?"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
