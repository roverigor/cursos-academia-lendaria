### 1.3 Scripts de Indicacao

**Script 1: Direto**
```
"[NOME], uma pergunta:

Voce conhece alguem que [dor que resolvo]?

Porque posso ajudar como ajudei voce.

Se tiver alguem, me passa o contato
e eu cuido de tudo."
```

**Script 2: Com Recompensa**
```
"[NOME], lancamos um programa de indicacoes.

Funciona assim:
- Voce indica alguem que [dor]
- Se a pessoa virar cliente, voce ganha [recompensa]
- E [bonus para indicado] tambem ganha

Tem alguem em mente?"
```

**Script 3: Especifico**
```
"[NOME], to buscando clientes como voce:
[caracteristica 1], [caracteristica 2].

Na sua rede, quem voce conhece assim?

Me passa o nome que eu abordo
mencionando voce."
```

### 3.2 Calculo Financeiro

| Campo | Valor |
|-------|-------|
| **Salario fixo (estimado)** | R$ |
| **Comissao (estimada)** | R$ |
| **Custo total/mes** | R$ |
| **Meta de vendas** | R$ |
| **Margem dos produtos** | ___% |
| **Ponto de equilibrio** | R$ (vendas necessarias) |

```
Ponto de Equilibrio = Custo Total / Margem%

Exemplo:
Custo: R$ 5.000/mes
Margem: 50%
Ponto de equilibrio: R$ 10.000 em vendas/mes
```
