## 4. FORMULAS DE POST

### Formula 1: Lista Numerada

```
[Numero] [coisas] que [resultado]:

1. [Item] - [explicacao curta]
2. [Item] - [explicacao curta]
3. [Item] - [explicacao curta]
...

Qual voce ja aplica?
```

### Formula 2: Historia + Licao

```
[Situacao/gancho]

[O que aconteceu - 3-5 frases]

[O que aprendi]

[Licao para o leitor]

Ja passou por isso?
```

### Formula 3: Opiniao Controversa

```
[Afirmacao forte/controversa]

[Contexto - por que pensa assim]

[Argumento 1]
[Argumento 2]
[Argumento 3]

Concorda ou discorda?
```

### Formula 4: Case de Sucesso

```
[Cliente] tinha [problema].

Situacao:
- [Dor 1]
- [Dor 2]
- [Dor 3]

O que fizemos:
[Solucao resumida]

Resultados:
✓ [Resultado 1]
✓ [Resultado 2]
✓ [Resultado 3]

Quer resultados assim?
[CTA]
```

### Formula 5: Dica Rapida

```
Dica rapida sobre [tema]:

[Dica em 1-3 frases]

Por que funciona:
[Explicacao breve]

Salva pra usar depois.
```

## 5. ROTINA DE CONEXAO

| Enviar conexoes | 5-10 | 10 min |
| Comentar em posts | 5-10 | 10 min |

### Script de Conexao

```
Oi [NOME], vi que voce [OBSERVACAO].

Trabalho com [AREA] e adorei seu [CONTEUDO/PERFIL].

Queria me conectar para trocar ideias.

Abraco!
```

### Script de DM (apos aceitar)

```
[NOME], obrigado por aceitar!

Vi que voce [CONTEXTO DA EMPRESA/CARGO].

Posso te fazer uma pergunta sobre [TEMA]?
```
