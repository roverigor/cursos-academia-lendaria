## Instrucoes de Uso

1. Personalize para cada cliente
2. Foque no valor, nao no preco
3. Ofereca 3 opcoes (ancoragem)

## 2. CALCULADORA DE PRECO

### Metodo 1: Custo + Margem

| Campo | Valor |
|-------|-------|
| **Horas estimadas** | ___ horas |
| **Custo por hora** | R$ ___ |
| **Custo total** | R$ ___ |
| **Margem desejada** | ___% |
| **Preco final** | R$ ___ |

```
Preco = Custo Total x (1 + Margem%)

Exemplo:
40 horas x R$ 100 = R$ 4.000
R$ 4.000 x 1,50 (50% margem) = R$ 6.000
```

### Metodo 2: Baseado em Valor

| Campo | Valor |
|-------|-------|
| **Problema do cliente** | R$ ___ / mes |
| **Duracao do problema** | ___ meses |
| **Valor total do problema** | R$ ___ |
| **% do valor capturado** | ___% (10-30%) |
| **Preco sugerido** | R$ ___ |

```
Preco = Valor do Problema x % Captura

Exemplo:
Cliente perde R$ 50.000/mes
Problema ha 12 meses = R$ 600.000
Captura de 10% = R$ 60.000 de fee
```

### Metodo 3: Comparacao de Mercado

| | R$ | |

| Pacote | % do Completo | Margem | Preco |
