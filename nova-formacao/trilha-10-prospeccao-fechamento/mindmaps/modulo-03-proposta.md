## Precificação por Valor

```
┌────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│            CUSTO                           VALOR                           │
│        (como NÃO fazer)                (como fazer)                        │
│                                                                             │
│   ┌────────────────────┐         ┌────────────────────┐                   │
│   │ "Quanto custa      │         │ "Quanto vale       │                   │
│   │  pra fazer?"       │         │  resolver isso?"   │                   │
│   │                    │         │                    │                   │
│   │  Horas × Taxa      │         │  Problema × 10     │                   │
│   │  = Preço baixo     │         │  = Preço justo     │                   │
│   └────────────────────┘         └────────────────────┘                   │
│                                                                             │
│   FÓRMULA DE VALOR:                                                        │
│                                                                             │
│   Se o problema custa R$100k/ano para o cliente...                        │
│   E você resolve em 90 dias...                                             │
│   Seu preço pode ser R$10-30k (10-30% do valor gerado)                    │
│                                                                             │
│   REGRA: Preço = 10-30% do valor que você gera para o cliente             │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘
```

## Apresentar vs Enviar

```
┌────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│              ENVIAR                          APRESENTAR                    │
│            (evitar)                          (preferir)                    │
│                                                                             │
│   ┌────────────────────┐         ┌────────────────────┐                   │
│   │ • Cliente lê       │         │ • Você guia        │                   │
│   │   sozinho          │         │   a leitura        │                   │
│   │                    │         │                    │                   │
│   │ • Vai direto       │         │ • Constrói         │                   │
│   │   pro preço        │         │   valor antes      │                   │
│   │                    │         │                    │                   │
│   │ • Sem chance de    │         │ • Responde         │                   │
│   │   rebater objeções │         │   objeções na hora │                   │
│   │                    │         │                    │                   │
│   │ • "Vou pensar"     │         │ • Fecha na call    │                   │
│   └────────────────────┘         └────────────────────┘                   │
│                                                                             │
│   QUANDO ENVIAR: Só se o cliente insistir muito                            │
│   QUANDO APRESENTAR: Sempre que possível (call de 30-45 min)              │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘
```

## Calculadora de Preço

```
┌────────────────────────────────────────────────────────────────────────────┐
│                         COMO CALCULAR SEU PREÇO                            │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   PASSO 1: Qual o custo do problema para o cliente? (anual)                │
│            [________] R$                                                    │
│                                                                             │
│   PASSO 2: Quanto % desse valor você gera/economiza?                       │
│            [________] % (típico: 20-50%)                                   │
│                                                                             │
│   PASSO 3: Valor que você gera = Passo 1 × Passo 2                         │
│            [________] R$                                                    │
│                                                                             │
│   PASSO 4: Seu preço = 10-30% do valor que gera                            │
│            [________] R$                                                    │
│                                                                             │
│   EXEMPLO:                                                                  │
│   Problema custa R$200k/ano                                                │
│   Você resolve 30% = R$60k de economia                                     │
│   Seu preço = R$6k a R$18k (10-30% de R$60k)                               │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘
```
