## Sistema de Indicações

```
┌────────────────────────────────────────────────────────────────────────────┐
│                     PROGRAMA DE INDICAÇÕES                                 │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   QUANDO PEDIR:                                                            │
│   ─────────────                                                            │
│   • Após resultado positivo (cliente feliz)                                │
│   • No fechamento ("quem mais poderia se beneficiar?")                     │
│   • Em momentos de pico de satisfação                                      │
│                                                                             │
│   COMO PEDIR:                                                              │
│   ────────────                                                             │
│   "Você conhece 2-3 pessoas que enfrentam o mesmo desafio que você        │
│    enfrentava? Adoraria ajudá-las também."                                 │
│                                                                             │
│   O QUE OFERECER:                                                          │
│   ───────────────                                                          │
│   • Nada (muitos indicam só por gratidão)                                  │
│   • Desconto na renovação                                                  │
│   • Bônus/serviço extra                                                    │
│   • Comissão em dinheiro (se fizer sentido)                               │
│                                                                             │
│   ESTRUTURA DO PROGRAMA:                                                   │
│   ──────────────────────                                                   │
│   1. Definir benefício (pro indicador e pro indicado)                     │
│   2. Criar material de comunicação                                         │
│   3. Pedir ativamente (não esperar acontecer)                             │
│   4. Agradecer publicamente                                                │
│   5. Medir e otimizar                                                      │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘
```

## Mapa de Upsell/Cross-sell

```
┌────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   UPSELL: Vender versão superior do que já comprou                         │
│   CROSS-SELL: Vender produto/serviço complementar                          │
│                                                                             │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   O QUE OFERECER         QUANDO OFERECER         COMO OFERECER            │
│   ────────────────       ────────────────        ───────────────           │
│                                                                             │
│   Upgrade de plano       Quando atinge           "Você está usando         │
│                          limite do atual         100% do plano. Faz        │
│                                                  sentido evoluir?"         │
│                                                                             │
│   Serviço adicional      Quando surge            "Percebi que você         │
│                          necessidade             precisa de X. Posso       │
│                          relacionada             ajudar nisso também."     │
│                                                                             │
│   Renovação antecipada   3 meses antes           "Quer travar o preço     │
│                          do vencimento           atual por mais 1 ano?"   │
│                                                                             │
│   Produto complementar   Após 60-90 dias         "Clientes como você      │
│                          de uso                  geralmente precisam       │
│                                                  de Y. Conhece?"           │
│                                                                             │
│   REGRA: Upsell quando faz sentido pro cliente, não só pra você           │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘
```

## Quando Contratar Vendedor

```
┌────────────────────────────────────────────────────────────────────────────┐
│                         SINAIS DE QUE É HORA                               │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   ✅ HORA DE CONTRATAR:                                                    │
│                                                                             │
│   • Você está recusando leads por falta de tempo                           │
│   • Seu processo está documentado e funciona                               │
│   • Você tem caixa pra 6 meses de salário                                  │
│   • Você consegue treinar alguém (tem playbook)                            │
│   • Seu tempo vale mais fazendo outra coisa                                │
│                                                                             │
│   ❌ NÃO É HORA AINDA:                                                     │
│                                                                             │
│   • Você não sabe vender seu produto                                       │
│   • Não tem processo documentado                                           │
│   • Está contratando pra "não ter que vender"                              │
│   • Não tem caixa pra bancar o ramp-up                                     │
│                                                                             │
│   REGRA: Primeiro você domina, depois delega                               │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘
```

## Modelo de Comissionamento

```
┌────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   ESTRUTURAS COMUNS:                                                       │
│                                                                             │
│   ┌────────────────────────────────────────────────────────────────────┐   │
│   │ SDR (Pré-vendas)                                                    │   │
│   │ Base: R$ 2-4k + Bônus por SQL gerado (R$ 50-200/SQL)               │   │
│   └────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│   ┌────────────────────────────────────────────────────────────────────┐   │
│   │ CLOSER (Vendedor)                                                   │   │
│   │ Base: R$ 3-6k + Comissão 5-15% sobre vendas                        │   │
│   └────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│   ┌────────────────────────────────────────────────────────────────────┐   │
│   │ FULL-CYCLE (Faz tudo)                                               │   │
│   │ Base: R$ 3-5k + Comissão 8-20% sobre vendas                        │   │
│   └────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│   REGRA: Base deve cobrir o básico. Comissão é o upside.                   │
│   Meta de OTE (On-Target Earnings): 2-3x o salário base                   │
│                                                                             │
│   EXEMPLO:                                                                  │
│   Base: R$ 4.000 + Comissão 10% → Se vender R$ 60k = R$ 10k total         │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘
```
