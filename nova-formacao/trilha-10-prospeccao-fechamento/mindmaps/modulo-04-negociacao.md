## BATNA (Melhor Alternativa)

```
┌────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   BATNA = Best Alternative To Negotiated Agreement                         │
│   (Melhor Alternativa ao Acordo Negociado)                                 │
│                                                                             │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   SUA BATNA                          BATNA DO CLIENTE                      │
│   ────────────                       ──────────────────                    │
│                                                                             │
│   "Se esse cliente                   "Se não fechar                        │
│    não fechar, eu..."                 comigo, ele..."                      │
│                                                                             │
│   • Tenho outros leads?              • Tem outras opções?                  │
│   • Preciso muito dele?              • Precisa muito resolver?             │
│   • Qual meu pipeline?               • Qual a urgência dele?               │
│                                                                             │
│   REGRA: Quem tem BATNA mais forte, tem mais poder na negociação           │
│                                                                             │
│   SE SUA BATNA É FORTE:              SE BATNA DO CLIENTE É FRACA:          │
│   → Você não precisa ceder           → Ele precisa mais de você            │
│   → Pode manter preço                → Ele tem urgência                    │
│   → Pode impor condições             → Ele não tem alternativa boa         │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘
```

## Concessões Estratégicas

```
┌────────────────────────────────────────────────────────────────────────────┐
│                          REGRAS DE CONCESSÃO                               │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   ❌ ERRADO                           ✅ CERTO                              │
│                                                                             │
│   "Posso dar 10%                     "Se fechar hoje,                      │
│    de desconto"                       posso incluir X"                     │
│                                                                             │
│   Ceder sem pedir                    Trocar: "Se você...,                  │
│   nada em troca                      então eu..."                          │
│                                                                             │
│   Ceder rápido                       Ceder devagar,                        │
│   demais                             com resistência                       │
│                                                                             │
│   Ceder no preço                     Ceder em valor:                       │
│   (margem)                           bônus, prazo, suporte                 │
│                                                                             │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   FÓRMULA DE CONCESSÃO:                                                    │
│                                                                             │
│   "Entendo que [pedido do cliente].                                        │
│    Se você [condição], eu posso [concessão de valor, não preço]."         │
│                                                                             │
│   EXEMPLO:                                                                  │
│   "Se você fechar até sexta, incluo 1 mês extra de suporte."              │
│   "Se pagar à vista, incluo o módulo bônus."                              │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘
```

## Objeções de Preço

```
┌────────────────────────────────────────────────────────────────────────────┐
│                    RESPOSTAS PARA OBJEÇÕES DE PREÇO                        │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   "TÁ CARO"                                                                │
│   └── "Caro comparado a quê? Ao custo de não resolver?"                   │
│   └── "O que seria um preço justo pra você? Me ajuda a entender."         │
│   └── "Se eu mostrar que o retorno é 3x o investimento, faz sentido?"     │
│                                                                             │
│   "PRECISO DE DESCONTO"                                                    │
│   └── "Qual desconto você precisa e por quê?"                             │
│   └── "Se eu der X%, você fecha agora?"                                   │
│   └── "Em vez de desconto, posso incluir [valor]. Funciona?"              │
│                                                                             │
│   "O CONCORRENTE É MAIS BARATO"                                            │
│   └── "O que exatamente ele oferece pelo preço? Vamos comparar."          │
│   └── "Mais barato em quê? No preço ou no resultado?"                     │
│   └── "Se o resultado for igual, faz sentido ir pelo mais barato.         │
│        Mas é igual?"                                                       │
│                                                                             │
│   "NÃO TENHO ORÇAMENTO"                                                    │
│   └── "Quanto você tem disponível? Vamos ver o que cabe."                 │
│   └── "Se o orçamento não fosse problema, você fecharia?"                 │
│   └── "Posso parcelar / começar menor / postergar?"                       │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘
```

## Técnicas de Fechamento

```
┌────────────────────────────────────────────────────────────────────────────┐

│   FECHAMENTO ALTERNATIVO                                                   │

│   "Faz mais sentido o pacote A ou B?"                                      │
│                                                                             │
│   FECHAMENTO RESUMO                                                        │
│   ────────────────────                                                     │

│   "Essa condição vale até sexta. Depois volta ao preço normal."           │
│   "Tenho só mais 2 vagas esse mês."                                        │
│                                                                             │
│   FECHAMENTO DIRETO                                                        │
│   ───────────────────                                                      │
│   "Podemos fechar?"                                                        │
│   "O que falta pra gente avançar?"                                         │
│                                                                             │
│   FECHAMENTO SILÊNCIO                                                      │
│   ────────────────────                                                     │
│   Faça a pergunta de fechamento e FIQUE EM SILÊNCIO.                       │
│   Quem fala primeiro, perde.                                               │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘
```
