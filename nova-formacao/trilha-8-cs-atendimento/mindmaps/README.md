## Formato

Os mindmaps usam indentacao por tabs para representar hierarquia:

```
Topico Principal
	Subtopico 1
		Detalhe 1.1
		Detalhe 1.2
	Subtopico 2
		Detalhe 2.1
```

## Uso

- Importar em ferramentas de mindmap (XMind, MindMeister, etc.)
- Usar como referencia rapida do conteudo
- Base para apresentacoes e resumos
