### NPS: O QUE E E COMO FUNCIONA (2.5 minutos)

```
[MOSTRAR CONCEITO NPS]

"NPS = Net Promoter Score.

[MOSTRAR PERGUNTA]

A pergunta classica:
'De 0 a 10, qual a chance de voce recomendar
[empresa/produto] para um amigo ou colega?'

[MOSTRAR CLASSIFICACAO]

As respostas dividem em 3 grupos:

PROMOTORES (9-10)
- Amam voce
- Indicam ativamente
- Leais

NEUTROS (7-8)
- Satisfeitos, mas nao entusiasmados
- Podem sair facilmente
- Nao indicam

DETRATORES (0-6)
- Insatisfeitos
- Falam mal
- Risco de churn

[MOSTRAR CALCULO]

FORMULA NPS:
NPS = % Promotores - % Detratores

Exemplo:
- 100 respostas
- 50 deram 9-10 (50% promotores)
- 30 deram 7-8 (30% neutros - ignora)
- 20 deram 0-6 (20% detratores)

NPS = 50% - 20% = +30

[MOSTRAR FAIXAS]

Interpretacao:
- Acima de +50: Excelente
- +30 a +50: Bom
- 0 a +30: Precisa melhorar
- Negativo: Critico"
```

### CSAT: SATISFACAO POR ATENDIMENTO (2 minutos)

```
[MOSTRAR CONCEITO CSAT]

"CSAT = Customer Satisfaction Score.

[MOSTRAR DIFERENCA]

NPS: mede relacionamento geral
CSAT: mede interacao especifica

[MOSTRAR PERGUNTA]

A pergunta:
'Como voce avalia o atendimento que acabou de receber?'

Opções:
- Muito satisfeito
- Satisfeito
- Neutro
- Insatisfeito
- Muito insatisfeito

Ou escala de estrelas (1 a 5).

[MOSTRAR CALCULO]

FORMULA CSAT:
CSAT = (Satisfeitos + Muito Satisfeitos) / Total x 100

Exemplo:
- 100 respostas
- 70 deram 4 ou 5 estrelas
- CSAT = 70%

[MOSTRAR FAIXAS]

Interpretacao:
- Acima de 90%: Excelente
- 75-90%: Bom
- 60-75%: Regular
- Abaixo de 60%: Critico"
```

### QUANDO USAR CADA UM (1.5 minutos)

```
[MOSTRAR COMPARATIVO]

"NPS vs CSAT: quando usar cada um.

[MOSTRAR TABELA]

| Metrica | Mede | Quando Enviar | Frequencia |
|---------|------|---------------|------------|
| NPS | Relacionamento geral | Trimestral | 3-4x/ano |
| CSAT | Atendimento especifico | Apos ticket | Sempre |

[MOSTRAR COMBINACAO]

Melhor estrategia: usar os DOIS.

CSAT: apos cada atendimento
→ Feedback imediato
→ Identifica problemas pontuais
→ Corrige rapido

NPS: trimestral ou apos marcos
→ Visao geral
→ Tendencia de longo prazo
→ Benchmark do mercado

[MOSTRAR REGRA]

Regra de ouro:
- CSAT baixo + NPS alto = atendimento ruim, produto bom
- CSAT alto + NPS baixo = atendimento bom, produto ruim
- Ambos baixos = problema serio"
```

## MATERIAL DE APOIO

### Comparativo NPS vs CSAT

| Caracteristica | NPS | CSAT |
|----------------|-----|------|
| O que mede | Lealdade geral | Satisfacao pontual |
| Pergunta | Recomendaria? | Satisfeito? |
| Escala | 0-10 | 1-5 ou emojis |
| Frequencia | Trimestral | Apos cada atendimento |
| Uso | Benchmark, tendencia | Feedback imediato |

### Formula NPS

```
NPS = % Promotores - % Detratores

Promotores: notas 9-10
Neutros: notas 7-8 (ignorados)
Detratores: notas 0-6

Faixas:
- > +50: Excelente
- +30 a +50: Bom
- 0 a +30: Regular
- < 0: Critico
```

### Formula CSAT

```
CSAT = (Satisfeitos + Muito Satisfeitos) / Total x 100

Ou:
CSAT = (Notas 4-5) / Total x 100

Faixas:
- > 90%: Excelente
- 75-90%: Bom
- 60-75%: Regular
- < 60%: Critico
```

### Matriz de Diagnostico

| CSAT | NPS | Diagnostico |
|------|-----|-------------|
| Alto | Alto | Tudo funcionando ✓ |
| Alto | Baixo | Atendimento bom, produto/preco ruim |
| Baixo | Alto | Atendimento ruim, produto bom |
| Baixo | Baixo | Problema serio em tudo |
