# Modulo 2: Base de Conhecimento + Resposta Padrao

## Consistencia Que Escala

---

> **Duracao:** 90 minutos
> **Bloom:** Nivel 6 - Criar
> **Entregavel:** KB v1 + Prompt Motor de Resposta

---

## Objetivo do Modulo

Construir a **Base de Conhecimento v1** e um **Motor de Resposta com IA** para reduzir tempo e variacao nas respostas.

## Fluxo de Aprendizagem

```
2.1 Quick Win      →  Resposta com IA (resultado em 2 min)
      ↓

      ↓

      ↓
2.8 Validacao      →  Testar com tickets reais
```

## Pre-requisitos

- [ ] Funil de Suporte Instrumentado (Modulo 1)
- [ ] Top 3 categorias de ticket identificadas
- [ ] Acesso a ChatGPT ou Claude
- [ ] 10-20 tickets reais para teste

## Quick Win: O Que o Aluno Sai Com em 5 Minutos

- Prompt funcionando que gera resposta de atendimento
- Primeira resposta automatica criada
- Entendimento do poder de KB + IA

## Entregavel do Modulo

**KB v1 + Motor de Resposta** com:
- 20-40 FAQs organizadas por categoria
- Politicas documentadas (reembolso, prazos, suporte)
- 10-20 how-tos (passo a passo)
- Regras de tom de voz
- Prompt unico que gera respostas consistentes
- 10 tickets testados com o motor

## Conexao com Proximo Modulo

A KB sera usada no Modulo 3 para alimentar o sistema de triagem automatica e auto-resposta.
