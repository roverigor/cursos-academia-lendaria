### Dicas para Iterar Melhor

**Seja cirúrgico:**
```
❌ "Melhore o visual"
✅ "Mude a cor do botão de azul para verde (#22C55E)"
```

**Uma coisa por vez:**
```
❌ "Mude a cor, adicione campo e arrume o footer"
✅ "Adicione um campo para telefone abaixo do email"
(depois) "Mude a cor do botão para verde"
(depois) "Adicione logo no footer"
```

**Use referências:**
```
✅ "Faça o card parecido com os cards do Stripe"
✅ "Use o mesmo estilo de botão do Notion"
```
