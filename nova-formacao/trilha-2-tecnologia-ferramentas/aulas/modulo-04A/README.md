# Módulo 4A: Vibe Coding — Criar Sem Programar

## Caminho A: Empresário Estratégico

## Visão Geral

| Item | Valor |
|------|-------|
| **Duração** | 55 minutos |
| **Aulas** | 4 |
| **Tipo** | Caminho A (Empresário) |
| **Entregável** | 1 protótipo funcional publicado |

## Objetivo

Criar aplicações funcionais usando ferramentas de "vibe coding" — sem escrever uma linha de código.

## Pré-requisito

Módulos Core (1-3) concluídos.

## Estrutura

| Aula | Título | Tipo | Duração |
|------|--------|------|---------|
| 4A.1 | Seu primeiro app em 15 minutos | Quick Win | 15 min |
| 4A.2 | Dominando Vibe Coding | Teoria | 15 min |
| 4A.3 | Do Prompt ao Produto | Exercício | 15 min |
| 4A.4 | Deploy e Compromisso 48h | Validação | 10 min |

## Ferramentas Cobertas

| Ferramenta | Foco | Melhor Para |
|------------|------|-------------|
| **Bolt** | Full-stack rápido | MVPs completos |
| **Lovable** | Apps bonitos | Landing pages, dashboards |
| **v0** | Componentes UI | Interfaces específicas |
| **Replit Agent** | Protótipos rápidos | Testar ideias |

## Conceitos-Chave

- Prompt para geração de código
- Iteração conversacional
- Deploy sem configuração
- Quando usar vs quando delegar

## Entregável

**1 Protótipo Funcional:**
- App publicado na internet
- Função principal funcionando
- Compartilhável via link

## Compromisso 48h

- Compartilhar link do app com 3 pessoas
- Coletar feedback
- Iterar com base no feedback
