### Perguntas para Feedback

1. **Primeira impressão:** "O que você entendeu que esse app faz?"
2. **Usabilidade:** "Conseguiu fazer [ação principal] sem ajuda?"
3. **Valor:** "Você usaria isso? Por quê?"
4. **Problemas:** "O que te confundiu ou frustrou?"
5. **Sugestões:** "O que está faltando?"

### Template de Mensagem para Enviar

```
Oi [Nome]!

Criei um [tipo de app] e queria sua opinião honesta.

Link: [URL]

Pode testar e me responder:
1. O que você entendeu que faz?
2. Conseguiu usar sem dificuldade?
3. Usaria de verdade?

Feedback sincero me ajuda muito! Obrigado.
```

## Métricas de Sucesso

### Definição de Sucesso

| Métrica | Meta | Como Medir |
|---------|------|------------|
| **Entendimento** | 3/3 entendem | Pergunta 1 |
| **Usabilidade** | 2/3 conseguem usar | Pergunta 2 |
| **Valor** | 2/3 usariam | Pergunta 3 |
| **Problemas** | <3 críticos | Pergunta 4 |

### Resultado Esperado

```
Se 2 de 3 pessoas:
- Entenderam o propósito
- Conseguiram usar
- Usariam de verdade

→ O projeto é um sucesso validado
```
