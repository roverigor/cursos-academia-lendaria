## O Que Sao LLMs

**LLM = Large Language Model**

- Modelos treinados em bilhoes de textos
- Preveem a "proxima palavra" mais provavel
- NAO pensam, NAO entendem, NAO tem consciencia
- Sao muito bons em padroes de linguagem

### Principais LLMs

| Modelo | Empresa | Forca |
|--------|---------|-------|
| GPT-4 | OpenAI | Versatil, codigo |
| Claude | Anthropic | Texto longo, analise |
| Gemini | Google | Multimodal, pesquisa |
| Llama | Meta | Open source |

## O Que IA Faz BEM

| Tarefa | Exemplo |
|--------|---------|
| **Gerar texto** | Emails, posts, descricoes |
| **Resumir** | Documentos, reunioes, feedback |
| **Traduzir** | Idiomas, linguagem tecnica |
| **Analisar** | Sentimento, padroes, tendencias |
| **Estruturar** | Dados bagunçados → organizados |
| **Criar variacoes** | 10 versoes de um titulo |
| **Responder perguntas** | Sobre textos fornecidos |
| **Codigo** | Scripts, automacoes, debug |

---

## O Que IA Faz MAL

| Limitacao | Por Que |
|-----------|---------|
| **Fatos atuais** | Dados de treinamento tem corte |
| **Calculos complexos** | Nao e calculadora |
| **Decisoes criticas** | Sem responsabilidade legal |
| **Informacoes confidenciais** | Dados vao para servidores |
| **Criatividade "original"** | Recombina, nao inventa |
| **Consistencia longa** | Pode contradizer |
| **Alucinacoes** | Inventa fatos com confianca |

## Quando Usar Cada Modelo

| Situacao | Modelo Recomendado |
|----------|--------------------|
| Texto e analise | Claude |
| Codigo e tecnico | GPT-4 |
| Pesquisa com internet | Gemini, Perplexity |
| Custo importa | GPT-3.5, Claude Haiku |
| Imagens | DALL-E, Midjourney |
| Privacidade total | Modelos locais (Llama) |

---

## API vs Chat

| Aspecto | Chat (site) | API |
|---------|-------------|-----|
| Uso | Manual, um por um | Automatizado, em massa |
| Custo | Assinatura fixa | Por token usado |
| Integracao | Nenhuma | Total |
| Quando usar | Tarefas pontuais | Automacoes, apps |
