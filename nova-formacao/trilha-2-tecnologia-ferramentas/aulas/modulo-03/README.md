# Módulo 03: Engenharia de Prompt

## Visão Geral

| Item | Valor |
|------|-------|
| **Duração** | 55 minutos |
| **Aulas** | 4 |
| **Tipo** | Core (obrigatório) |
| **Entregável** | Biblioteca de 10 prompts otimizados |

## Objetivo

Dominar a arte de comunicar com IAs para obter resultados precisos, consistentes e úteis para o negócio.

## Estrutura

| Aula | Título | Tipo | Duração |
|------|--------|------|---------|
| 3.1 | Prompt que funciona em 10 min | Quick Win | 10 min |
| 3.2 | Anatomia do Prompt Perfeito | Teoria | 15 min |
| 3.3 | Construindo sua Biblioteca | Exercício | 20 min |
| 3.4 | Validação e Compromisso 48h | Validação | 10 min |

## Conceitos-Chave

- Estrutura de prompt (contexto, instrução, formato)
- Zero-shot vs Few-shot
- Chain-of-Thought (CoT)
- Personas e papéis
- Iteração e refinamento
- Templates reutilizáveis

## Entregável

**Biblioteca de 10 Prompts Otimizados:**
1. 3 prompts para análise/resumo
2. 3 prompts para criação de conteúdo
3. 2 prompts para tomada de decisão
4. 2 prompts para automação

## Compromisso 48h

- Usar 3 prompts em situações reais
- Medir qualidade das respostas
- Iterar e melhorar pelo menos 1 prompt
