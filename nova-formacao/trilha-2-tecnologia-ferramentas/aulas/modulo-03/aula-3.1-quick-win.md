## Tipo: Quick Win | Duração: 10 minutos

## GPS

### Goal (30s)
Transformar um prompt ruim em um prompt excelente em 10 minutos.

### Position (60s)
A diferença entre "IA inútil" e "IA que resolve" está em como você pergunta. Vamos provar isso agora.

### Steps
1. Ver prompt ruim vs bom (3 min)
2. Aplicar fórmula simples (4 min)
3. Testar e comparar (3 min)

---

## Prompt Ruim vs Bom

### Exemplo Real

**Prompt RUIM:**
```
Escreve um email de vendas
```

**Resultado:** Genérico, sem personalidade, inútil.

**Prompt BOM:**
```
Você é um copywriter especializado em B2B SaaS.

Escreva um email de prospecção para:
- Persona: Diretor de Marketing de empresa de 50-200 funcionários
- Produto: Software de automação de marketing
- Dor principal: Equipe pequena, muitas tarefas manuais
- Tom: Profissional mas amigável
- Tamanho: Máximo 150 palavras
- CTA: Agendar demo de 15 minutos

Inclua uma linha de assunto com taxa de abertura acima de 40%.
```

**Resultado:** Email específico, pronto para usar.

## A Fórmula CRISP

| Letra | Significado | Exemplo |
|-------|-------------|---------|
| **C** | Contexto | Quem você é, situação |
| **R** | Role (Papel) | Que especialista a IA deve ser |
| **I** | Instrução | O que fazer exatamente |
| **S** | Specs (Especificações) | Formato, tamanho, restrições |
| **P** | Propósito | Para que será usado |

## Exercício: Transforme Agora

### Seu Prompt Ruim (copie um que você já usou)
```
[Cole aqui um prompt que deu resultado ruim]
```

### Aplique CRISP

**C - Contexto:**
```
Eu sou [seu cargo] de uma empresa de [setor] com [tamanho].
```

**R - Role:**
```
Você é um [especialista em...] com experiência em [área].
```

**I - Instrução:**
```
[Ação específica] sobre [tema específico].
```

**S - Specs:**
```
Formato: [lista/texto/tabela]
Tamanho: [X palavras/itens]
Tom: [formal/informal/técnico]
```

**P - Propósito:**
```
Isso será usado para [finalidade].
```
