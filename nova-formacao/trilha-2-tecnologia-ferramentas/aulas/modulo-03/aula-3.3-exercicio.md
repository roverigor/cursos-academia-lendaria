# Aula 3.3: Construindo sua Biblioteca de Prompts

## Tipo: Exercício | Duração: 20 minutos

---

## GPS

### Goal (30s)
Criar 10 prompts otimizados que você vai usar no dia a dia do seu negócio.

### Position (60s)
Prompts avulsos se perdem. Uma biblioteca organizada multiplica sua produtividade.

### Steps
1. Prompts de Análise (5 min)
2. Prompts de Criação (7 min)
3. Prompts de Decisão (4 min)
4. Prompts de Automação (4 min)

---

## Categoria 1: Análise (3 prompts)

### Prompt 1.1: Resumo Executivo

```
Você é um consultor de negócios especializado em [seu setor].

Analise o seguinte [documento/relatório/dados]:
[COLE AQUI]

Forneça um resumo executivo com:
1. Pontos principais (máximo 5)
2. Implicações para o negócio
3. Ação recomendada

Formato: Bullet points
Tamanho: Máximo 200 palavras
Tom: Direto e executivo
```

**Personalize:**
- Setor: _______________
- Tipo de documento mais comum: _______________

---

### Prompt 1.2: Análise de Concorrência

```
Analise o concorrente [NOME] considerando:

Informações disponíveis:
[COLE INFORMAÇÕES: site, redes sociais, reviews, etc.]

Forneça:
1. Proposta de valor principal
2. Público-alvo identificado
3. Pontos fortes (3)
4. Pontos fracos (3)
5. O que podemos aprender

Formato: Tabela + conclusão
```

**Seus principais concorrentes:**
1. _______________
2. _______________
3. _______________

---

### Prompt 1.3: Diagnóstico de Problema

```
Você é um especialista em resolução de problemas.

Problema: [DESCREVA O PROBLEMA]

Contexto:
- Quando começou: [data/período]
- Frequência: [sempre/às vezes/raramente]
- Impacto: [baixo/médio/alto]

Analise usando 5 Porquês:
1. Por que isso está acontecendo?
2. Por que [resposta 1]?
3. Por que [resposta 2]?
4. Por que [resposta 3]?
5. Por que [resposta 4]?

Forneça:
- Causa raiz provável
- 3 possíveis soluções
- Solução recomendada com justificativa
```

## Categoria 2: Criação de Conteúdo (3 prompts)

### Prompt 2.1: Post para Redes Sociais

```
Você é um social media manager especializado em [seu setor].

Crie um post para [LinkedIn/Instagram/outro]:

Tema: [TEMA]
Objetivo: [engajamento/autoridade/vendas/educação]
Público: [descrição do público]

Estrutura:
- Hook (primeira linha que prende atenção)
- Desenvolvimento (3-5 linhas)
- CTA (chamada para ação)

Tom: [profissional/descontraído/inspirador]
Inclua: [emojis sim/não] [hashtags sim/não]

Forneça 3 versões diferentes.
```

**Sua rede principal:** _______________
**Tom da sua marca:** _______________

---

### Prompt 2.2: Email de Follow-up

```
Você é um especialista em vendas B2B.

Crie um email de follow-up:

Contexto:
- Contato anterior: [reunião/proposta/cold email]
- Há quanto tempo: [X dias]
- Status: [sem resposta/pediu tempo/mostrou interesse]

Objetivo: [agendar reunião/fechar venda/manter relacionamento]

Tom: Profissional mas humano
Tamanho: Máximo 100 palavras
CTA: Claro e com baixa fricção

Inclua linha de assunto com alta taxa de abertura.
```

---

### Prompt 2.3: Descrição de Produto/Serviço

```
Você é um copywriter especializado em [seu setor].

Crie descrição para:
Produto/Serviço: [NOME]

Características:
- [característica 1]
- [característica 2]
- [característica 3]

Benefícios para o cliente:
- [benefício 1]
- [benefício 2]
- [benefício 3]

Público-alvo: [descrição]
Diferencial: [o que torna único]

Forneça:
1. Headline (máximo 10 palavras)
2. Descrição curta (50 palavras)
3. Descrição completa (150 palavras)
4. 3 bullet points de venda
```

## Categoria 3: Tomada de Decisão (2 prompts)

### Prompt 3.1: Análise de Oportunidade

```
Você é um consultor de estratégia de negócios.

Analise esta oportunidade:
[DESCREVA A OPORTUNIDADE]

Contexto do meu negócio:
- Setor: [seu setor]
- Tamanho: [faturamento/equipe]
- Momento: [crescimento/estabilidade/crise]
- Recursos disponíveis: [descreva]

Forneça:
1. Análise SWOT da oportunidade
2. Investimento estimado (tempo/dinheiro)
3. Retorno potencial
4. Riscos principais (top 3)
5. Recomendação: GO / NO-GO / INVESTIGAR MAIS

Justifique a recomendação em 3 frases.
```

---

### Prompt 3.2: Priorização de Tarefas

```
Você é um especialista em produtividade e gestão.

Preciso priorizar estas tarefas:
1. [tarefa 1]
2. [tarefa 2]
3. [tarefa 3]
4. [tarefa 4]
5. [tarefa 5]

Contexto:
- Prazo geral: [X dias/semanas]
- Recursos: [só eu / equipe de X]
- Critério mais importante: [urgência/impacto/facilidade]

Classifique usando matriz Eisenhower:
- Urgente + Importante: FAZER AGORA
- Importante + Não Urgente: AGENDAR
- Urgente + Não Importante: DELEGAR
- Nem Urgente + Nem Importante: ELIMINAR

Forneça ordem de execução com justificativa.
```

## Categoria 4: Automação (2 prompts)

### Prompt 4.1: Criar Workflow de Automação

```
Você é um especialista em automação com n8n.

Quero automatizar: [PROCESSO]

Atualmente:
- Ferramenta A: [nome] - faz [função]
- Ferramenta B: [nome] - faz [função]
- Ferramenta C: [nome] - faz [função]

Fluxo manual atual:
1. [passo 1]
2. [passo 2]
3. [passo 3]

Desenhe um workflow n8n com:
1. Trigger (o que inicia)
2. Nodes necessários
3. Lógica/condições
4. Output esperado

Formato: Diagrama em texto + explicação passo a passo
```

---

### Prompt 4.2: Script de Processamento

```
Você é um desenvolvedor especializado em automação.

Preciso de um script para:
[DESCREVA A TAREFA]

Input: [tipo de dado de entrada]
Output: [tipo de dado de saída]

Requisitos:
- Linguagem: [Python/JavaScript/outro]
- Tratamento de erro: [sim/não]
- Log de execução: [sim/não]

Forneça:
1. Código comentado
2. Instruções de uso
3. Possíveis problemas e soluções
```
