## O Que E o Mapa Tributario

### Definicao:
Um documento que consolida TODA sua carga tributaria em um unico lugar, dando visibilidade total para tomada de decisao.

### Por que usar:
- Ver o todo, nao pedacos isolados
- Identificar onde esta pesado
- Comparar com benchmarks do setor
- Fundamentar conversa com contador

## Como Preencher

### Passo 1: Coletar dados (30 min)
- Pedir extrato de impostos ao contador
- Reunir guias dos ultimos 6 meses
- Levantar folha de pagamento

### Passo 2: Preencher secoes (20 min)
- Comece pelos tributos explicitos
- Adicione encargos da folha
- Estime custos ocultos

### Passo 3: Calcular totais (10 min)
- Somar cada categoria
- Calcular percentuais
- Comparar com benchmarks

---

## Benchmarks por Regime

### Simples Nacional

| Atividade | Carga Tipica |
|-----------|--------------|
| Comercio | 4-11% |
| Servicos (baixa folha) | 6-15% |
| Servicos (alta folha) | 8-13% |
| Industria | 5-12% |

### Lucro Presumido

| Atividade | Carga Tipica |
|-----------|--------------|
| Servicos em geral | 11-17% |
| Comercio | 6-12% |
| Industria | 8-14% |

### Dica:
Se sua carga esta ACIMA do benchmark, investigue.
Se esta muito ABAIXO, pode haver risco (subnotificacao).

---

## Erros Comuns ao Preencher

| Erro | Problema | Como Evitar |
|------|----------|-------------|
| Usar valores "chutados" | Analise errada | Pedir dados reais ao contador |
| Ignorar encargos | Subestimar carga | Incluir TODA a folha |
| Esquecer multas | Nao ver custos ocultos | Revisar ultimos 12 meses |
| Preencher sozinho | Dados incompletos | Envolver contador |

## ARTEFATO DA AULA: Template do Mapa Tributario Empresarial

### O Que Voce Vai Criar
Template estruturado do Mapa Tributario Empresarial com todas as secoes (dados gerais, tributos por esfera, encargos, custos ocultos, resumo), pronto para preenchimento.

### Por Que Isso Importa
O Mapa e o ENTREGAVEL PRINCIPAL do Modulo 1:
- Consolida toda carga em um documento
- Base para decisoes de todo o curso
- Fundamenta conversa com contador
- Visibilidade total = decisao informada

**Linha do DRE:** Visibilidade de Despesas (ver 100% dos custos tributarios, nao so parte)

### Quando Usar
- Preenchimento completo na aula 1.7
- Revisao anual (todo Janeiro)
- Quando faturamento mudar >20%
- Antes de qualquer mudanca de regime

### Como Criar
1. Copie o template para seu formato preferido
2. Configure cabecalho com dados da empresa
3. Prepare lista de dados a coletar
4. Agende tempo para preenchimento (aula 1.7)

## IA NA PRATICA

### Prompt Principal: Assistente de Preenchimento do Mapa

```
Atue como assistente de planejamento tributario. Vou preencher meu Mapa Tributario Empresarial e preciso de ajuda para calcular valores.

MEUS DADOS:
- Razao Social: [nome]
- CNAE: [codigo] - [descricao]
- Regime: [Simples/Presumido/Real]
- Faturamento mensal: R$ [valor]
- Funcionarios: [numero]
- Folha bruta: R$ [valor]/mes

O QUE EU SEI:
- DAS/Impostos mensais: R$ [valor]
- [outros dados que tiver]

O QUE NAO SEI:
- [liste o que nao sabe, ex: "decomposicao do DAS", "encargos sobre folha"]

AJUDE-ME A:
1. Calcular os valores que nao tenho
2. Validar se os que tenho fazem sentido
3. Identificar o que preciso pedir ao contador
4. Estimar custos ocultos tipicos para meu perfil

FORMATO: Preencha o template do Mapa com meus dados + estimativas claras.
```

### Prompt Alternativo: Benchmark Personalizado

```
Qual o benchmark de carga tributaria para meu perfil?

PERFIL:
- Atividade: [descrever detalhadamente]
- CNAE: [codigo]
- Regime: [Simples/Presumido/Real]
- Faturamento: R$ [valor]/mes
- Regiao: [estado/cidade]
- Funcionarios: [numero]

INFORME:
1. Carga tributaria EXPLICITA tipica para meu perfil (%)
2. Carga IMPLICITA tipica (encargos sobre folha) (%)
3. Carga OCULTA estimada (%)
4. Carga TOTAL esperada (%)
5. Faixa aceitavel (minimo - maximo)
6. Se estou fora da faixa, o que pode significar

Compare com empresas similares do mesmo setor.
```

### Como Usar o Resultado

| Se a IA revelar... | Faca isso |
|--------------------|-----------|

| Gaps de informacao | Liste para pedir ao contador |
| Benchmark do setor | Compare com seu resultado |
| Alertas de incoerencia | Investigue antes de seguir |
