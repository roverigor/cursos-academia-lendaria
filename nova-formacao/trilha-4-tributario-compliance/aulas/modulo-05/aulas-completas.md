## Aula 5.3: Matriz de Priorizacao

> **Duracao:** 10 minutos

### Economia vs Risco

```
                    ECONOMIA ALTA
                         |
        FAZER COM      |      FAZER
        CUIDADO        |      AGORA
                       |
    RISCO ALTO --------|-------- RISCO BAIXO
                       |
        NAO            |      FAZER
        FAZER          |      DEPOIS
                       |
                    ECONOMIA BAIXA
```

### Classifique Suas Acoes

| Acao | Economia | Risco | Quadrante |
|------|----------|-------|-----------|
| | [ ] Alta [ ] Baixa | [ ] Alto [ ] Baixo | |
| | [ ] Alta [ ] Baixa | [ ] Alto [ ] Baixo | |
| | [ ] Alta [ ] Baixa | [ ] Alto [ ] Baixo | |
| | [ ] Alta [ ] Baixa | [ ] Alto [ ] Baixo | |

### Ordem de Execucao

1. **Fazer agora:** Economia alta + Risco baixo
2. **Fazer com cuidado:** Economia alta + Risco alto
3. **Fazer depois:** Economia baixa + Risco baixo
4. **Nao fazer:** Economia baixa + Risco alto

## Aula 5.5: Demo - Plano de Execucao Completo

> **Duracao:** 10 minutos

### Exemplo: Consultoria ABC

```markdown
# PLANO DE EXECUCAO TRIBUTARIA

Empresa: Consultoria ABC Ltda
Data: Dezembro/2025

## 1. DECISAO ESTRATEGICA

Estrategia principal: Otimizacao de Pro-labore
- Reduzir de R$ 50.000 para R$ 16.000
- Distribuir R$ 34.000 como lucros

Economia esperada: R$ 86.400/ano
Risco aceito: Baixo (base legal solida)

## 2. CRONOGRAMA 90 DIAS

| Mes | Acao | Responsavel | Entregavel |
|-----|------|-------------|------------|
| Jan | Formalizar nova estrutura | Contador | Ata de alteracao |
| Jan | Criar DRE mensal | Contador | Template DRE |
| Fev | Primeira distribuicao | Socio | Comprovante |
| Fev | Resolver gaps LGPD | TI | Checkbox + canal |
| Mar | Revisar contratos | Advogado | Contratos atualizados |
| Mar | Validar economia | Contador | Relatorio comparativo |

## 3. PROFISSIONAIS A ENVOLVER

| Profissional | Acao | Prazo |
|--------------|------|-------|
| Contador Jose | Validar plano, criar DRE | Janeiro |
| Advogado Maria | Revisar contratos | Marco |

## 4. METRICAS

| Metrica | Baseline | Meta 30d | Meta 90d |
|---------|----------|----------|----------|
| Carga tributaria | 22% | 20% | 18% |
| Compliance score | 57% | 75% | 90% |
| Economia acumulada | R$ 0 | R$ 7.200 | R$ 21.600 |

## 5. COMPROMISSO

Eu, Joao Silva, me comprometo a:
- Implementar as acoes conforme cronograma
- Envolver os profissionais necessarios
- Medir resultados mensalmente
- Ajustar o plano se necessario

Assinatura: Joao Silva
Data: 15/12/2025
```
