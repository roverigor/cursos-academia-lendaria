### Como Usar o Resultado

| Resultado da Validacao | Proxima Acao |
|-----------------------|--------------|
| Score > 80% | Levar para contador validar |
| Score 50-80% | Completar itens faltantes |
| Score < 50% | Revisar plano inteiro |
| Calculo com erro | Refazer com valores corretos |
| Base legal incompleta | Usar prompt de base legal |

## ARTEFATO DA AULA: Plano Tributario Defensavel Preenchido

### O Que Voce Vai Criar
Seu Plano Tributario Defensavel completo, com pelo menos 1 estrategia detalhada, pronto para validacao profissional.

### Por Que Isso Importa
Este e o ENTREGAVEL PRINCIPAL do Modulo 3:
- Estrategia documentada formalmente
- Economia calculada com metodologia
- Riscos mapeados
- Base para reuniao com contador

**Linha do DRE:** Economia Tributaria Projetada (valor que voce pode economizar)

### Quando Usar
- Agora: criar durante a aula
- Em reuniao com contador: apresentar
- Anualmente: revisar e atualizar
- Se fiscalizado: usar como defesa

### Como Criar
1. **Passo 1 (5 min):** Identificar oportunidades
2. **Passo 2 (15 min):** Preencher plano
3. **Passo 3 (5 min):** Validar com IA
