## O QUE IA NAO PODE FAZER

| Limitacao | Por que |
|-----------|---------|
| Substituir contador | Sem responsabilidade legal |
| Acessar seus dados | Voce precisa informar |
| Garantir resultado | Sugestao, nao certeza |
| Acompanhar mudancas | Legislacao muda constantemente |
| Assinar documentos | Nao tem personalidade juridica |

## ARTEFATO DA AULA: Guia de Uso de IA para Tributacao

### O Que Voce Vai Criar
Documento de referencia sobre quando e como usar IA para questoes tributarias, com limites claros e boas praticas.

### Por Que Isso Importa
Usar IA SEM entender limites e perigoso:
- Pode confiar demais
- Pode ignorar erros
- Pode tomar decisoes erradas
- Pode ter problemas legais

**Linha do DRE:** Custo de Erro (IA errada pode custar caro se nao validar)

### Quando Usar
- Antes de comecar a usar IA para tributacao
- Como referencia ao usar prompts
- Para lembrar de validar resultados

### Como Criar
1. Liste o que vai usar IA para fazer
2. Defina limites claros
3. Estabeleca processo de validacao

### Template

```
GUIA DE USO DE IA PARA TRIBUTACAO

VOU USAR IA PARA:
- [ ] Calcular cenarios
- [ ] Entender conceitos
- [ ] Gerar templates
- [ ] Revisar documentos
- [ ] Organizar informacoes

NAO VOU USAR IA PARA:
- [ ] Decisao final sem validar
- [ ] Substituir contador
- [ ] Enviar dados sensiveis sem cuidado

MEU PROCESSO DE VALIDACAO:
1. Gerar com IA
2. Revisar criticamente
3. Validar com contador se necessario
4. Implementar

MEUS LIMITES:
- Valores acima de R$ ___: sempre validar com contador
- Decisoes de regime: sempre validar
- Questoes legais: sempre validar
```

## IA NA PRATICA

### Prompt Principal: Verificador de Capacidades da IA

```
Preciso entender o que voce pode e nao pode fazer em relacao a tributacao. Me ajude:

MINHA SITUACAO:
- Tipo de empresa: [descrever]
- Regime: [Simples/Presumido/Real]
- Duvidas principais: [listar]

PARA CADA DUVIDA, CLASSIFIQUE:

| Duvida | IA Pode Ajudar? | Como | Precisa Validar? |
|--------|-----------------|------|------------------|
| | [ ] Sim [ ] Parcial [ ] Nao | | [ ] Sim [ ] Nao |

EXPLICACAO:
- Onde IA pode ajudar completamente: ___
- Onde IA pode dar um ponto de partida: ___
- Onde IA NAO deve ser usada sozinha: ___

RECOMENDACAO:
Para sua situacao, use IA para: ___
Valide com contador: ___
Nao use IA para: ___
```

### Prompt Alternativo: Criador de Fluxo de Validacao

```
Crie um fluxo de validacao para minhas decisoes tributarias usando IA:

MEU CONTEXTO:
- Faturamento: R$ [valor]/mes
- Decisoes que preciso tomar: [listar]
- Contador disponivel: [ ] Sim [ ] Nao
- Conhecimento tributario: [ ] Basico [ ] Intermediario [ ] Avancado

GERE FLUXO DE DECISAO:

PASSO 1: PESQUISA INICIAL COM IA
- O que perguntar: ___
- O que esperar: ___
- Sinais de alerta: ___

PASSO 2: ANALISE CRITICA
- Verificar: ___
- Questionar: ___
- Buscar segunda opiniao: ___

PASSO 3: VALIDACAO
- Quando validar com contador: ___
- O que enviar para ele: ___
- Perguntas a fazer: ___

PASSO 4: DECISAO
- Documentar: ___
- Implementar: ___
- Monitorar: ___

CHECKLIST DE SEGURANCA:
- [ ] Resultado faz sentido?
- [ ] Esta dentro da lei?
- [ ] Tenho base legal?
- [ ] Contador validou (se necessario)?
```

### Como Usar o Resultado

| Classificacao | Proxima Acao |
|---------------|--------------|
| IA pode ajudar | Seguir com prompts especificos |
| IA parcial | Usar como ponto de partida |
| Precisa validar | Agendar conversa com contador |
