### BLOCO 4: CTAs QUE CONVERTEM (2 minutos)

```
[OLHAR PARA CAMERA]

"Agora o CTA. A acao que voce quer.

[MOSTRAR ERROS]

CTAs fracos:
'Me siga para mais dicas' - fraco.
'Gostou? Deixa um like' - fraco.
'Link na bio' - generico.

[MOSTRAR FORMULA]

A formula do CTA forte:

ACAO ESPECIFICA + BENEFICIO + URGENCIA

Exemplo:
'Comenta EU QUERO (acao)
que eu te mando o template gratis (beneficio)
- so ate amanha (urgencia)'

[MOSTRAR LISTA]

10 CTAs de alta conversao:

1. 'Comenta EU QUERO que eu te mando no DM'
2. 'DM CONVERSAR pra agendar 15 min gratis'
3. 'Link na bio - baixe o guia completo'
4. 'DM PRECO que eu te mando as condicoes'
5. 'Salva esse post e me conta qual voce vai usar'
6. 'Marca aquele amigo que PRECISA ver isso'
7. 'Se fatura mais de 50K/mes, comenta SIM'
8. 'Ultimas 3 vagas - DM VAGA agora'
9. 'So pra quem comentar: mando bonus no DM'
10. 'Te dou 30 min de consultoria gratis - DM GRATIS'"
```

## PROMPT DE IA - GERADOR DE GANCHOS

```
Gere 10 ganchos para post de Instagram.

CONTEXTO:
- Tema: [seu tema]
- ICP: [seu cliente ideal]
- Tom: [profissional/descontraido/provocativo]

USE ESTAS FORMULAS:
1. Numero + Beneficio
2. Pergunta provocativa
3. Controversia
4. Historia pessoal
5. Resultado surpresa

PARA CADA GANCHO:
- Maximo 1 linha
- Deve parar o scroll
- Gerar curiosidade

RETORNE em formato de lista numerada.
```

## PROMPT DE IA - GERADOR DE CTAs

```
Gere 5 CTAs para post de [objetivo: lead/venda/engajamento].

CONTEXTO:
- Produto: [seu produto]
- ICP: [seu cliente ideal]
- Plataforma: [Instagram/LinkedIn]

FORMULA:
Acao especifica + Beneficio + Urgencia (opcional)

EXEMPLO:
"Comenta 'QUERO' que eu te mando o template gratis - so ate amanha"

RETORNE 5 CTAs prontos pra usar.
```
