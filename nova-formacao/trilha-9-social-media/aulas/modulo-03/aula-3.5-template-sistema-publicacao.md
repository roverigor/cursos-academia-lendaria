# Aula 3.5: Template - Sistema de Publicacao

---

> **Duracao:** 7 minutos

## ROTEIRO DE FALA

### ABERTURA (30 segundos)

```
[OLHAR PARA CAMERA]

"Este template vai organizar
toda sua publicacao em um unico lugar.

Banco de Conteudo.
Calendario.
Rotina.
Checklist.
Metricas.

Tudo integrado.

Abre o template que ta no material.
Vamos configurar juntos."
```

---

### BLOCO 1: VISAO GERAL DO SISTEMA (1 minuto)

```
[MOSTRAR ESTRUTURA NA TELA]

"O Sistema de Publicacao tem 5 partes.

[MOSTRAR]

1. BANCO DE CONTEUDO
De onde vem os posts.
Voce criou isso no Modulo 2.

2. CALENDARIO EDITORIAL
Quando cada post vai ao ar.
Dias e horarios.

3. ROTINA SEMANAL
Seu processo de criacao.
Ideacao, criacao, agendamento.

4. CHECKLIST PRE-PUBLICACAO
Qualidade garantida.
Antes de agendar, passa por aqui.

5. METRICAS DE ACOMPANHAMENTO
Como saber se ta funcionando.
Alcance, engajamento, leads.

[OLHAR PARA CAMERA]

5 partes. 1 sistema.
Vamos configurar cada uma."
```

---

### BLOCO 2: CALENDARIO EDITORIAL (2 minutos)

```
[MOSTRAR TEMPLATE NA TELA]

"O Calendario e o coracao do sistema.

[MOSTRAR ESTRUTURA]

Cada linha tem:
- Dia da semana
- Horario
- Etapa (Topo/Meio/Fundo)
- Formato
- ID do post (conecta com o Banco)

[MOSTRAR EXEMPLO]

Segunda | 11h | Topo | Carrossel | T01
Terca | 19h | Meio | Reels | M01
Quarta | 12h | Topo | Imagem | T02
Quinta | 18h | Meio | Stories | M02
Sexta | 11h | Fundo | Carrossel | F01

[OLHAR PARA CAMERA]

Preenche as 4 semanas do mes.

Se voce seguiu o Modulo 2,
ja tem 30 posts no Banco.
E so conectar os IDs.

[PAUSA - 30 SEGUNDOS]

Preenche agora.
Pelo menos a semana 1."
```

---

### BLOCO 3: ROTINA SEMANAL (1,5 minuto)

```
[MOSTRAR ROTINA NA TELA]

"A Rotina define QUANDO voce trabalha
em social media.

[MOSTRAR]

SEGUNDA - IDEACAO (20 min)
- Revisar performance semana anterior
- Gerar 10 ideias com IA
- Selecionar 5-7 para desenvolver

TERCA - CRIACAO (45 min)
- Desenvolver posts com IA
- Criar/selecionar visuais
- Revisar e ajustar tom

QUARTA - AGENDAMENTO (25 min)
- Agendar todos os posts
- Verificar previews
- Configurar lembretes

QUINTA A DOMINGO
- Responder comentarios (15 min/dia)
- ZERO criacao de conteudo

[OLHAR PARA CAMERA]

Preenche no template
quais dias e horarios funcionam pra voce.

[PAUSA - 20 SEGUNDOS]

Feito?"
```

---

### BLOCO 4: CHECKLIST PRE-PUBLICACAO (1,5 minuto)

```
[MOSTRAR CHECKLIST NA TELA]

"Antes de agendar qualquer post,
passa por esse checklist.

[MOSTRAR]

1. Gancho forte? (para nos 3 primeiros segundos)
2. CTA claro e especifico?
3. Imagem/video de qualidade?
4. Texto sem erros?
5. Horario otimizado?
6. Hashtags relevantes? (5-10)
7. Link na bio atualizado? (se necessario)

[OLHAR PARA CAMERA]

Se passar nos 7 criterios, agenda.
Se nao passar, ajusta.

Isso garante qualidade minima
em TODOS os posts.

Se quiser, pode usar a IA
pra validar antes de agendar.

Tem um prompt no material.
Mas e opcional."
```

## PROMPT DE IA - VALIDAR POST (OPCIONAL)

```
Avalie este post antes de publicar:

[Cole seu post aqui]

VERIFICAR:
1. Gancho (1-10): Esta forte o suficiente?
2. Clareza: A mensagem esta clara?
3. CTA: O proximo passo esta obvio?
4. Erros: Gramatica e digitacao OK?
5. Melhorias: 1 sugestao especifica

Retorne em formato de checklist.
```
