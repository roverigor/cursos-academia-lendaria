```
SISTEMA DE PUBLICACAO

├── 4. Checklist Pre-Publicacao (qualidade)
└── 5. Metricas de Acompanhamento (resultado)
```

## 2. CALENDARIO EDITORIAL (Quando)

### Template Semanal

| Dia | Horario | Etapa | Formato | Post ID |
|-----|---------|-------|---------|---------|
| Segunda | 11h | Topo | Carrossel | 001 |
| Terca | 19h | Meio | Reels | 002 |

| Quinta | 18h | Meio | Stories | 004 |
| Sexta | 11h | Fundo | Carrossel | 005 |

### Prompt IA: Gerar Calendario

```
Crie um calendario editorial para o proximo mes.

DADOS:
- Plataforma: [Instagram]
- Frequencia: [5 posts/semana]
- Distribuicao: 50% Topo, 30% Meio, 20% Fundo

CONSIDERE:
- Melhores horarios para [nicho]
- Variedade de formatos
- Datas comemorativas relevantes

FORMATO: Tabela com 4 semanas
Colunas: Semana | Dia | Horario | Etapa | Formato | Tema sugerido
```

---

## 3. ROTINA SEMANAL (Processo)

### Checklist por Dia

**Segunda - Ideacao (20 min)**
- [ ] Revisar performance semana anterior
- [ ] Gerar 10 ideias com IA
- [ ] Selecionar 5-7 para desenvolver

**Terca - Criacao (45 min)**
- [ ] Desenvolver posts com IA
- [ ] Criar/selecionar visuais
- [ ] Revisar e ajustar tom

**Quarta - Agendamento (25 min)**
- [ ] Agendar todos os posts
- [ ] Verificar previews
- [ ] Configurar lembretes

**Quinta a Domingo**
- [ ] Responder comentarios (15 min/dia)
- [ ] Monitorar Stories
- [ ] ZERO criacao de conteudo novo

## 4. CHECKLIST PRE-PUBLICACAO (Qualidade)

Antes de agendar cada post:

| Criterio | OK? |
|----------|-----|
| Gancho forte (para nos 3 primeiros segundos) | [ ] |
| CTA claro e especifico | [ ] |
| Imagem/video de qualidade | [ ] |
| Texto sem erros | [ ] |
| Horario otimizado | [ ] |
| Hashtags relevantes (5-10) | [ ] |
| Link na bio atualizado (se necessario) | [ ] |

### Prompt IA: Validar Post

```
Avalie este post antes de publicar:

[Cole seu post aqui]

VERIFICAR:
1. Gancho (1-10): Esta forte o suficiente?
2. Clareza: A mensagem esta clara?
3. CTA: O proximo passo esta obvio?
4. Erros: Gramatica e digitacao OK?
5. Melhorias: 1 sugestao especifica

Responda em formato de checklist.
```
