### BLOCO 1: O QUE E BATCH CONTENT (1,5 minuto)

```
[MOSTRAR DIAGRAMA NA TELA]

"Batch Content e criar tudo de uma vez.
Em blocos concentrados.

[MOSTRAR]

Em vez de criar 1 post por dia (7 sessoes),
voce cria 7 posts em 1 dia (1 sessao).

[MOSTRAR COMPARACAO]

MODO REATIVO:
Segunda: cria 1 post
Terca: cria 1 post
Quarta: cria 1 post
... 7 trocas de contexto

MODO BATCH:
Segunda: cria 7 posts
Terca a domingo: ZERO criacao

[OLHAR PARA CAMERA]

A diferenca?

No modo reativo, voce perde tempo
entrando e saindo do modo criativo.

No modo batch, voce entra UMA VEZ
e produz tudo."
```

## MATERIAL DE APOIO

### Rotina Semanal Batch

| Dia | Atividade | Tempo | Uso de IA |
|-----|-----------|-------|-----------|
| Segunda | Ideacao | 20 min | Gerar ideias |
| Terca | Criacao | 45 min | Desenvolver posts |
| Quarta | Revisao + Agendamento | 25 min | Revisar |
| Qui-Dom | ZERO criacao | 0 min | - |
| **TOTAL** | | **1h30** | |

### Comparativo de Tempo

| Modo | Tempo/Semana | Tempo/Mes |
|------|--------------|-----------|
| Reativo | 3h30 | 15h |
| Batch com IA | 1h30 | 6h |
| **Economia** | **2h** | **9h** |

### Beneficios do Batch

1. **FOCO** - Sem troca de contexto
2. **QUALIDADE** - Mais tempo por post
3. **CONSISTENCIA** - Nunca falta conteudo
4. **LIBERDADE** - 4-5 dias sem criar

## PROMPTS DE IA - BATCH CONTENT

### Prompt 1: Ideacao (Segunda)

```
Gere 10 ideias de posts para minha semana de conteudo.

CONTEXTO:
- Nicho: [seu nicho]
- Publico: [seu ICP]
- Plataforma: [Instagram/LinkedIn]

DISTRIBUICAO:
- 5 posts Topo (awareness)
- 3 posts Meio (consideracao)
- 2 posts Fundo (conversao)

PARA CADA IDEIA:
- Gancho (primeira linha)
- Formato (carrossel/reels/imagem)
- Dia sugerido para postar

Formato: Tabela organizada por dia.
```

### Prompt 2: Criacao (Terca)

```
Desenvolva estes posts para [plataforma]:

IDEIAS:
[Cole as ideias do prompt anterior]

PARA CADA POST:
1. Gancho (max 10 palavras)
2. Corpo (max 150 palavras)
3. CTA especifico
4. 5 hashtags relevantes
5. Sugestao de visual

TOM: [profissional/casual/provocativo]

Organize por dia da semana.
```

### Prompt 3: Revisao (Quarta)

```
Revise estes posts antes de agendar:

[Cole seus posts aqui]

PARA CADA POST, AVALIE:
1. Gancho forte? (1-10)
2. CTA claro? (Sim/Nao)
3. Legibilidade (facil/medio/dificil)
4. Algo pra melhorar? (1 sugestao)

Formato: Tabela com notas e sugestoes.
```
