## PROMPT IA: AUDITORIA DO SISTEMA

```
Faca uma auditoria do meu sistema de publicacao.

MEU SISTEMA:
- Ferramenta: [qual]
- Posts agendados: [quantidade]
- Frequencia: [posts/semana]
- Rotina: [descreva]

VERIFIQUE:
1. Pontos fortes do sistema
2. Gaps ou riscos
3. O que pode dar errado
4. Sugestoes de melhoria

RETORNE:
- Score de prontidao (0-100)
- Top 3 ajustes prioritarios
- Proximo passo recomendado
```
