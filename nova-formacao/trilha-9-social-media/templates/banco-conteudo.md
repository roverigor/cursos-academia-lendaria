## 8. PROMPTS DE IA PARA CRIACAO

### Gerar Ideias Topo

```
Gere 15 ideias de posts TOPO DE FUNIL para Instagram.
Tema: [seus pilares]
ICP: [seu cliente ideal]

Cada ideia deve ter:
- Gancho (primeira linha chamativa)
- Formato sugerido
- Objetivo (awareness/engajamento)

Foco em pessoas que NAO me conhecem ainda.
```

### Gerar Ideias Meio

```
Gere 9 ideias de posts MEIO DE FUNIL.
Para pessoas que ja me seguem.
Foco em: cases, tutoriais, bastidores, metodologia.

Cada ideia deve gerar CONFIANCA e mostrar expertise.
```

### Gerar Ideias Fundo

```
Gere 6 ideias de posts FUNDO DE FUNIL.
Para pessoas prontas para comprar.
Ofertas: [suas ofertas]

Cada post deve ter CTA claro.
Use gatilhos: escassez, urgencia, prova social.
```

### Desenvolver Post

```
Desenvolva este post para Instagram:

IDEIA: [titulo/gancho]
FORMATO: [carrossel X slides / reels / imagem]
TOM: [profissional/casual/inspirador]
CTA: [acao desejada]

Estrutura:
- Gancho forte
- Desenvolvimento
- CTA claro

Maximo 200 palavras + 5-10 hashtags relevantes.
```
