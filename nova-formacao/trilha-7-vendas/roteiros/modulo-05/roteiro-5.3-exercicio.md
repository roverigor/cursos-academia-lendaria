## CONTEÚDO [1:00 - 18:00]

### Passo 1: Dashboard do Funil [1:00 - 7:00]

**[TOM INSTRUCIONAL]**

Primeiro, cria a visão do funil do mês atual.

**[MOSTRAR NA TELA: Template]**

Copia essa estrutura:

DASHBOARD COMERCIAL - [Mês/Ano]

Funil do Mês:

| Etapa | Volume | Taxa | Meta | Gap | Status |
| Leads | ___ | - | ___ | ___ | |
| MQL | ___ | ___% | ___% | | |
| SQL | ___ | ___% | ___% | | |
| Call | ___ | ___% | ___% | | |
| Proposta | ___ | ___% | ___% | | |
| Venda | ___ | ___% | ___% | | |

**[INSTRUÇÃO]**

Preenche com seus números do mês atual. Coluna Volume é quantos leads você tem. Coluna Taxa é a conversão pra próxima etapa. Meta é o benchmark ou sua meta interna. Gap é a diferença.

**[PAUSA]**

Pausa e preenche. Eu espero.

**[ESPERAR 5 SEGUNDOS]**

---

### Passo 2: Calcule o Forecast [7:00 - 11:00]

**[TOM PRÁTICO]**

Agora calcula a previsão de receita.

**[MOSTRAR NA TELA: Template]**

Forecast Próximo Mês:

| Pipeline Atual | Volume | Taxa Histórica | Receita Prevista |
| Em qualificação | ___ | ___% | R$ ___ |
| Em negociação | ___ | ___% | R$ ___ |
| Proposta enviada | ___ | ___% | R$ ___ |
| TOTAL | | | R$ ___ |

Meta: R$ ___
Gap: R$ ___

**[INSTRUÇÃO]**

Preenche com seus números. Multiplica volume pela taxa e pelo ticket.

**[EXEMPLO]**

Se você tem 50 leads em qualificação, taxa histórica de 5%, ticket de 5 mil:
50 × 5% × 5.000 = 12.500 reais previstos dessa etapa.

Faz pra cada etapa e soma.

**[ESPERAR 5 SEGUNDOS]**

---

### Passo 3: Identifique o Gargalo [11:00 - 14:00]

**[TOM ANALÍTICO]**

Agora encontra o problema.

**[MOSTRAR NA TELA: Template]**

Análise de Gargalos:

Etapa com pior taxa: _______________
Taxa atual: ___%
Benchmark: ___%
Gap: ___%

Por que está baixo?
[ ] Qualificação fraca
[ ] Follow-up inconsistente
[ ] Calls sem preparo
[ ] Proposta fraca
[ ] Outro: ___

**[INSTRUÇÃO]**

Olha sua tabela do funil. Qual etapa tem a pior taxa comparada com o benchmark? Essa é seu gargalo.

Marca a causa provável.

**[ESPERAR 3 SEGUNDOS]**

---

### Passo 4: Defina o Plano de Ação [14:00 - 18:00]

**[TOM DECISIVO]**

Por fim, define o que fazer.

**[MOSTRAR NA TELA: Template]**

Plano de Otimização:

Prioridade 1: Corrigir Gargalo
- Etapa: ___
- Ação: ___
- Responsável: ___
- Prazo: ___
- Métrica de sucesso: Taxa de ___% → ___%

Prioridade 2: Quick Win
- Ação: ___
- Impacto esperado: ___

Prioridade 3: Médio Prazo
- Ação: ___
- Prazo: ___

**[INSTRUÇÃO]**

Preenche as 3 prioridades. Foca primeiro no gargalo.

**[DICA]**

Se não sabe que ação tomar, usa esse prompt na IA:

"Meu gargalo é [etapa]. Taxa atual [X]%, meta [Y]%. Que ações posso tomar pra melhorar essa taxa em 2 semanas?"
