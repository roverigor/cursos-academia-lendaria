# Aula 1.3: Monte Seu Funil Comercial Instrumentado

> **Tipo:** Exercício
> **Duração:** 20 minutos
> **Entregável:** Funil Comercial Instrumentado (tabela completa)

---

## Objetivo

Criar seu **Funil Comercial Instrumentado** - uma tabela que define exatamente como leads avançam no seu processo de vendas.

---

## Build Sprint (20 min)

### Passo 1: Copie o Template (2 min)

Crie uma planilha ou documento com estas colunas:

| Etapa | Critério de Passagem | Taxa Atual | Tempo Médio | Dono |
|-------|---------------------|------------|-------------|------|
| | | | | |

### Passo 2: Defina Suas Etapas (5 min)

Preencha com as etapas do SEU negócio:

```
Exemplo para consultoria:
1. Lead - Preencheu formulário
2. Qualificado - Respondeu perguntas iniciais
3. Agendado - Marcou call de diagnóstico
4. Call feita - Participou da call
5. Proposta - Recebeu proposta
6. Cliente - Pagou/Contratou
```

Adapte para seu negócio. Máximo 7 etapas.

### Passo 3: Defina Critérios Objetivos (5 min)

Para cada etapa, responda: **"O que o lead precisa FAZER para avançar?"**

| Ruim (subjetivo) | Bom (objetivo) |
|------------------|----------------|
| "Demonstrou interesse" | "Respondeu 'sim' para pergunta X" |
| "Parece qualificado" | "Score >= 70 pontos" |
| "Quer agendar" | "Clicou no link e escolheu horário" |

### Passo 4: Preencha Métricas (5 min)

Para cada etapa, estime:

- **Taxa atual:** % que avança (pode ser estimativa)
- **Tempo médio:** Quantos dias fica nessa etapa
- **Dono:** Quem decide? (SDR, Closer, IA, Automação)

### Passo 5: Identifique Onde IA Entra (3 min)

Marque na coluna "Dono" onde IA pode atuar:

| Tarefa | Pode ser IA? |
|--------|--------------|
| Qualificar lead inicial | ✅ Sim |
| Calcular score | ✅ Sim |
| Enviar follow-up | ✅ Sim |
| Fazer call de vendas | ❌ Humano |
| Analisar objeções | ✅ Sim (suporte) |
| Fechar contrato | ❌ Humano |

---

## Template Preenchido (Exemplo)

```
| Etapa | Critério de Passagem | Taxa | Tempo | Dono |
|-------|---------------------|------|-------|------|
| Lead | Entrou no sistema | - | 0 | Automação |
| MQL | Respondeu qualificação | 40% | 2 dias | IA (score) |
| SQL | Score >= 70 | 50% | 1 dia | SDR |
| Call | Agendou e compareceu | 60% | 3 dias | SDR |
| Proposta | Recebeu proposta | 70% | 2 dias | Closer |
| Venda | Pagou | 30% | 5 dias | Closer |
```
