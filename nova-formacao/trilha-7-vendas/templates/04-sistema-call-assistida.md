## FASE 2: DURANTE A CALL

### Estrutura da Call (30 min)

| Fase | Tempo | O que fazer |
|------|-------|-------------|
| **Abertura** | 3 min | Rapport + agenda |
| **Descoberta** | 10 min | Perguntas BANT |
| **Apresentacao** | 10 min | Solucao + valor |
| **Objecoes** | 5 min | Responder duvidas |
| **Fechamento** | 2 min | Proximo passo |

---

### Script Flexivel

#### ABERTURA (3 min)

```
"Oi [NOME], tudo bem?

Antes de comecar, queria confirmar:
voce tem os 30 minutos que combinamos?

Otimo! A ideia e:
1. Entender melhor sua situacao
2. Ver se [SOLUCAO] faz sentido
3. Definir proximo passo, se houver

Pode ser?"
```

#### DESCOBERTA (10 min)

**Perguntas de Situacao:**
- "Me conta um pouco sobre [AREA/PROCESSO] hoje..."
- "Como voces fazem [ATIVIDADE] atualmente?"

**Perguntas de Problema:**
- "Qual o maior desafio que voce enfrenta com isso?"
- "O que acontece quando [PROBLEMA]?"

**Perguntas de Implicacao:**
- "Quanto isso custa em tempo/dinheiro?"
- "O que acontece se continuar assim?"

**Perguntas de Necessidade:**
- "Se voce pudesse mudar uma coisa, o que seria?"
- "Como seria o cenario ideal?"

#### APRESENTACAO (10 min)

```
"Pelo que voce me contou, [RESUMO DO PROBLEMA].

O que fazemos e [SOLUCAO RESUMIDA].

Na pratica, funciona assim:
1. [PASSO 1]
2. [PASSO 2]
3. [PASSO 3]

Clientes como [EXEMPLO] conseguiram [RESULTADO].

Faz sentido pra sua situacao?"
```

#### OBJECOES (5 min)

| Objecao | Resposta |
|---------|----------|
| "E caro" | |
| "Preciso pensar" | |
| "Ja tenho solucao" | |
| "Nao e prioridade" | |
| "Preciso falar com X" | |

#### FECHAMENTO (2 min)

```
SE POSITIVO:
"Otimo! O proximo passo seria [PROPOSTA/DEMO/TRIAL].
Que tal [DATA/HORA]?"

SE INCERTO:
"Entendo. O que voce precisa pra tomar essa decisao?
Posso te ajudar com algo especifico?"

SE NEGATIVO:
"Sem problema! Posso te adicionar na nossa lista
pra mandar conteudos uteis? Quem sabe no futuro..."
```
