## Detalhamento por Estágio

```
┌────────────────────────────────────────────────────────────────────────────┐
│                              1. ABERTURA                                   │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   • Apresentação pessoal (15 seg)                                          │
│   • Agradecer pelo tempo                                                   │
│   • Dar contexto da conversa                                               │
│   • Confirmar tempo disponível                                             │
│   • Pedir permissão para fazer perguntas                                   │
│                                                                             │
│   "Fulano, obrigado por dedicar esse tempo. Vi que você [contexto].        │
│    Temos 25 min? Posso fazer algumas perguntas pra entender melhor?"       │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────────────┐
│                             2. DESCOBERTA                                  │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   PERGUNTAS DE DOR:                                                        │
│   • "Qual seu maior desafio em vendas hoje?"                               │
│   • "Quanto isso está custando por mês?"                                   │
│   • "O que já tentou pra resolver?"                                        │
│                                                                             │
│   PERGUNTAS DE IMPACTO:                                                    │
│   • "Se resolver isso, quanto vale?"                                       │
│   • "O que muda na operação?"                                              │
│                                                                             │
│   PERGUNTAS DE DECISÃO:                                                    │
│   • "Quem mais participa da decisão?"                                      │
│   • "Qual o prazo pra resolver?"                                           │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────────────┐
│                            3. APRESENTAÇÃO                                 │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   DOR ───▶ SOLUÇÃO ───▶ PROVA ───▶ RESULTADO                              │
│                                                                             │
│   "Você me disse que [dor]. O que fazemos é [solução].                     │
│    Por exemplo, o cliente X tinha o mesmo problema e conseguiu [resultado].│
│    Em média, nossos clientes veem [métrica] em [prazo]."                   │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────────────┐
│                              4. OBJEÇÃO                                    │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   VALIDAR ───▶ QUESTIONAR ───▶ RESPONDER ───▶ CONFIRMAR                   │
│                                                                             │
│   "Entendo perfeitamente. [validação]                                      │
│    Me ajuda a entender: o que especificamente preocupa? [questionar]       │
│    Na verdade, [resposta com dados/prova].                                 │
│    Faz sentido? [confirmar]"                                               │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────────────┐
│                            5. FECHAMENTO                                   │
├────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   RESUMIR ───▶ CONFIRMAR INTERESSE ───▶ PROPOR PRÓXIMO PASSO              │
│                                                                             │
│   "Então, recapitulando: [resumo dores e como resolve].                    │
│    Isso faz sentido pra você? [confirmar]                                  │
│    O próximo passo seria [proposta/demo/reunião com time].                 │
│    Que tal [data/hora específica]?"                                        │
│                                                                             │
└────────────────────────────────────────────────────────────────────────────┘
```
