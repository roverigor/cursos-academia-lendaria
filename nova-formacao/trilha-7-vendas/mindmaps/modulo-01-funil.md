## Benchmarks por Tipo de Venda

```
┌──────────────────────────────────────────────────────────────────────────┐
│                    TAXAS DE REFERÊNCIA (VENDAS CONSULTIVAS)              │
├──────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│   Lead → MQL        ████████████████████░░░░░░░░░░░░░░░░░░░░   50%      │
│                                                                          │
│   MQL → SQL         ████████████████████████░░░░░░░░░░░░░░░░   60%      │
│                                                                          │
│   SQL → Call        ████████████████████████████░░░░░░░░░░░░   70%      │
│                                                                          │
│   Call → Proposta   ████████████████████████░░░░░░░░░░░░░░░░   60%      │
│                                                                          │
│   Proposta → Venda  ██████████████░░░░░░░░░░░░░░░░░░░░░░░░░░   35%      │

├──────────────────────────────────────────────────────────────────────────┤

│   B2B: taxas tendem a ser menores, ciclo mais longo                     │
│   B2C Alto Ticket: taxas podem ser maiores, ciclo mais curto            │
│                                                                          │

```

## Fórmula Central

```
┌────────────────────────────────────────────────────────────────┐
│                                                                 │
│         VOLUME × TAXA = VOLUME DA PRÓXIMA ETAPA                │
│                                                                 │
│   Exemplo:                                                      │
│   200 Leads × 50% = 100 MQLs                                   │
│   100 MQLs × 60% = 60 SQLs                                     │
│   60 SQLs × 70% = 42 Calls                                     │
│   42 Calls × 60% = 25 Propostas                                │
│   25 Propostas × 35% = 9 Vendas                                │
│                                                                 │
│   Taxa Total: 9/200 = 4.5%                                     │
│                                                                 │
└────────────────────────────────────────────────────────────────┘
```
