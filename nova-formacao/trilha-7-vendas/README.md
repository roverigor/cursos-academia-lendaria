# Trilha 7: Vendas - Aumento de Receita

> **Formato:** Microlearning Híbrido (4 aulas por módulo)
> **Duração Total:** 5-6 horas (20 micro-aulas)
> **Versão:** 2.0 | Dezembro 2025

---

## Promessa de Transformação

> **"Construir um sistema de vendas previsível, que reduz dependência de SDRs, aumenta taxa de conversão, encurta ciclo de vendas e transforma IA em copiloto comercial."**

---

| 1 | Arquitetura do Funil | 4 | 55 min |

**Total:** 20 aulas | ~4h45 de conteúdo + implementação

## Formato Microlearning Híbrido

Cada módulo segue a estrutura **GPS Didática Lendária**:

```
┌─────────────────────────────────────────────────────┐
│  AULA 1: QUICK WIN (10 min)                         │
│  → Resultado imediato, gera momentum                │
├─────────────────────────────────────────────────────┤
│  AULA 2: TEORIA MÍNIMA (15 min)                     │
│  → Conceitos essenciais, sem enrolação              │
├─────────────────────────────────────────────────────┤
│  AULA 3: EXERCÍCIO (20 min)                         │
│  → Mão na massa, cria entregável                    │
├─────────────────────────────────────────────────────┤
│  AULA 4: VALIDAÇÃO (10 min)                         │
│  → Checklist + Prompt IA + Próxima ação 48h         │
└─────────────────────────────────────────────────────┘
```
