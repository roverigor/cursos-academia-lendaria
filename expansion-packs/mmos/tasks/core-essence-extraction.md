---

##  PRIMARY OBJECTIVE

Distill from extracted patterns the **nuclear elements** that form cognitive essence:
- Irreducible components (cannot be simplified further)
- Persistent traits (present across all contexts/time)
- Differentiating features (unique to this person)
- Generative principles (produce observable behaviors)

Build the minimal viable model of this mind.

---

##  MUNGER-INSPIRED EXTRACTION METHOD

### PHASE 1: FIRST-PRINCIPLES DECOMPOSITION

**Question everything until you reach atoms:**

1. **Strip Context:**
   - Remove domain-specific language
   - Extract underlying principles
   - Find universal patterns

2. **Inversion Test:**
   - "What if this trait was opposite?"
   - "Would this still be the same person?"
   - "Is this fundamental or derived?"

3. **Reducibility Check:**
   - Can this be broken into smaller pieces?
   - Is this a compound of simpler elements?
   - What's the atomic unit?

### PHASE 2: PERSISTENCE VALIDATION

**Apply temporal stress-test:**

```
Element: [Candidate core trait]
Time Period 1: [Evidence]
Time Period 2: [Evidence]  
Time Period 3: [Evidence]

Persistence Score: X/10
Variation Analysis: [How it manifests differently]
```

**Criteria:**
- Present 95%+ across temporal sources
- Manifests in multiple contexts
- Survives life changes

### PHASE 3: DIFFERENTIATION ANALYSIS

**Compare against reference class:**

> "Circle of Competence: Know what makes this person unique vs common traits"

**Questions:**
- Found in 100 similar people? → Not core
- Found in 10 similar people? → Possibly core
- Found in 0-1 similar people? → Likely core

**Does this element PRODUCE observable patterns?**

