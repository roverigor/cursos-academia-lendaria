# SEO Best Practices Knowledge Base

**Purpose:** Reference guide for search engine optimization techniques used in content creation.

---

## On-Page SEO Fundamentals

### 1. Keyword Research & Strategy

**Primary Keyword:**
- 1 main keyword per page/post
- Search volume: 100-10K/month (sweet spot)
- Difficulty: Match to domain authority
- Intent: Match to content type

**Secondary Keywords:**
- 2-5 related keywords
- LSI (latent semantic indexing) terms
- Long-tail variations

**Tools:**
- Google Keyword Planner
- Ahrefs, SEMrush, Moz
- Google Search Console
- AnswerThePublic

**Usage in CreatorOS:**
- Research before writing
- Include in CONTENT-BRIEF
- Track rankings post-publish

---

### 2. Title Tag Optimization

**Formula:**
```
[Primary Keyword] - [Benefit/Hook] | [Brand]
```

**Best Practices:**
- 50-60 characters (Google cutoff)
- Primary keyword near beginning
- Include power words (Complete, Ultimate, Essential)
- Match search intent
- Unique per page

**Examples:**
- ❌ "CreatorOS Documentation"
- ✅ "CreatorOS Tutorial: Create Courses 80% Faster | Academia Lendária"

---

### 3. Meta Description

**Best Practices:**
- 150-160 characters
- Include primary keyword
- Call-to-action
- Compelling, not keyword-stuffed
- Unique per page

**Formula:**
```
[Problem] + [Solution] + [Benefit] + [CTA]
```

**Example:**
"Struggling with course creation? CreatorOS automates curriculum design with AI, cutting production time by 80%. Try it free today."

---

### 4. Header Structure (H1-H6)

**H1:**
- One per page
- Contains primary keyword
- 60 characters max

**H2:**
- Main section headers
- Include secondary keywords where natural
- 3-7 per post (1500-2500 words)

**H3-H6:**
- Sub-sections
- Logical hierarchy
- Improve scannability

**Example Structure:**
```
H1: CreatorOS Course Generation Guide
  H2: What is CreatorOS?
  H2: How to Create Your First Course
    H3: Step 1: Initialize Project
    H3: Step 2: Generate Curriculum
  H2: Advanced Features
    H3: Market Research
    H3: Voice Cloning
```

---

### 5. Content Quality Signals

**Length:**
- Blog posts: 1500-2500 words (sweet spot)
- Guides: 3000+ words
- News/updates: 500-800 words

**Readability:**
- Flesch Reading Ease: 60-70 (8th grade)
- Short paragraphs (3-4 lines max)
- Bullet points and lists
- Subheadings every 300 words

**Keyword Density:**
- Primary: 0.5-2% (not stuffed)
- Natural placement
- Variations and synonyms

**Freshness:**
- Update regularly (especially high-traffic pages)
- Add "Last Updated" dates
- Refresh outdated stats/links

---

### 6. Internal Linking

**Best Practices:**
- 3-5 internal links per 1000 words
- Descriptive anchor text (not "click here")
- Link to relevant, high-value pages
- Deep links (not just homepage)

**Anchor Text Patterns:**
- Exact match: "CreatorOS course generation"
- Partial match: "generate courses with CreatorOS"
- Branded: "CreatorOS tutorial"
- Generic: "learn more about this"

**Link Juice Distribution:**
- Link from high-authority pages to newer pages
- Create topic clusters (hub and spoke)

---

### 7. Image Optimization

**File Name:**
- Descriptive, keyword-rich
- Use hyphens: `course-creation-workflow.png`

**Alt Text:**
- Describe image for accessibility
- Include keyword where natural
- 125 characters max

**File Size:**
- Compress (aim for <100KB)
- Use WebP format
- Lazy loading for below-fold images

**Dimensions:**
- Specify width/height in HTML
- Avoid CLS (cumulative layout shift)

---

## Technical SEO

### 8. Site Speed Optimization

**Core Web Vitals:**
- LCP (Largest Contentful Paint): <2.5s
- FID (First Input Delay): <100ms
- CLS (Cumulative Layout Shift): <0.1

**Techniques:**
- Minify CSS/JS
- CDN for static assets
- Compress images
- Defer offscreen images
- Browser caching

---

### 9. Mobile Optimization

**Requirements:**
- Responsive design
- Touch-friendly (buttons 48x48px min)
- Fast load on 3G
- No horizontal scroll
- Readable text (16px min)

**Testing:**
- Google Mobile-Friendly Test
- PageSpeed Insights
- Real device testing

---

### 10. URL Structure

**Best Practices:**
- Short, descriptive
- Include primary keyword
- Use hyphens (not underscores)
- Lowercase only
- Avoid parameters when possible

**Examples:**
- ❌ `/page?id=123&cat=courses`
- ✅ `/creator-os-course-tutorial`

## Off-Page SEO

### 11. Link Building

**Quality Signals:**
- Domain Authority (DA) of linking site
- Relevance to your content
- Anchor text diversity
- Dofollow vs nofollow ratio

**Tactics:**
- Guest posting (relevant blogs)
- Broken link building
- Unlinked brand mentions
- Resource pages
- HARO (Help a Reporter Out)

**Avoid:**
- Paid links (Google penalty)
- Link farms
- Reciprocal link schemes
- Irrelevant directories

---

### 12. Social Signals

**Platforms:**
- LinkedIn (B2B, professional)
- Twitter (tech, news)
- Reddit (niche communities)
- Facebook groups

**Best Practices:**
- Share new content immediately
- Encourage engagement
- Use Open Graph tags
- Monitor mentions

## Content Strategy for SEO

### 13. Topic Clusters (Pillar Pages)

**Structure:**
- 1 Pillar Page (comprehensive, 3000+ words)
- 8-12 Cluster Pages (specific subtopics, 1500 words)
- Interlink all cluster pages to pillar

**Example (CreatorOS):**
- Pillar: "Complete Guide to AI Course Creation"
- Clusters:
  - "How to Extract ICP from Legacy Materials"
  - "GPS Framework for Course Design"
  - "Market Research for Courses"
  - "Voice Cloning with MMOS"

---

### 14. Content Refresh Strategy

**Priority:**
- High traffic pages (>100 visits/month)
- High conversion pages
- Pages ranking 5-15 (low-hanging fruit)

**Updates:**
- Add new sections (increase word count)
- Update stats/data
- Improve title/meta
- Add multimedia
- Fix broken links

**Frequency:**
- High-traffic: Quarterly
- Medium-traffic: Biannually
- Low-traffic: Annually

---

**Benefits:**

## SEO Workflow for Content Creation

### Pre-Writing:
1. Keyword research (primary + secondary)
2. Analyze top 10 competitors
3. Identify content gaps
4. Outline structure (H2/H3)
5. Target word count

### Writing:
1. Include keyword in H1, first 100 words, conclusion
2. Use keyword variations naturally
3. Add internal links (3-5)
4. Optimize images
5. Write compelling meta description

### Post-Publishing:
1. Submit to Google Search Console
2. Share on social media
3. Build 2-3 backlinks
4. Monitor rankings (weekly for first month)
5. Update quarterly

---

## Common SEO Mistakes to Avoid

❌ **Keyword Stuffing:** Unnatural repetition
❌ **Duplicate Content:** Same content on multiple pages
❌ **Thin Content:** <300 words, no value
❌ **Ignoring Mobile:** Non-responsive design
❌ **Slow Site:** >3s load time
❌ **No Internal Links:** Siloed content
❌ **Missing Alt Text:** Inaccessible images
❌ **No Analytics:** Can't measure what you don't track
