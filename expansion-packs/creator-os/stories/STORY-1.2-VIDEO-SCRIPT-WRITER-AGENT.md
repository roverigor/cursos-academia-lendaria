# Story 1.2: Implement Video Script Writer Agent

**Epic:** 1
**Status:** To Do

## Description

As a content creator, I want to be able to generate video scripts for different platforms so that I can produce engaging video content.

## Acceptance Criteria

- A new agent `script-writer` is created.
- A new task `generate-video-script` is created.
- The task can generate scripts for YouTube (5-15min), TikTok (60s), and Explainer videos (3min).
- The scripts include a hook, story arc, and CTA.

## Tasks

- [ ] Create `script-writer.md` agent file.
- [ ] Create `generate-video-script.md` task file.
- [ ] Create templates for YouTube, TikTok, and Explainer videos.
- [ ] Implement the logic for the `generate-video-script` task.
