# Story 1.1: Implement Social Media Specialist Agent

**Epic:** 1
**Status:** To Do

## Description

As a content creator, I want to be able to generate social media content for different platforms so that I can maintain a consistent online presence.

## Acceptance Criteria

- A new agent `social-media-specialist` is created.
- A new task `generate-social-content` is created.
- The task can generate content for LinkedIn, Twitter, and Instagram.
- The task uses platform-specific templates.
- The task can generate a 30-day content calendar.
