# Aula 5.3: Quando Fazer Upgrade?

**Módulo:** 5 - Evitando Problemas
**Duração:** 5 minutos
**Tipo:** Guia de decisão

---

## Objetivo da Aula

Ao final desta aula, o aluno vai:
- Identificar sinais de que precisa do Pro
- Conhecer diferenças Hobby vs Pro
- Tomar decisão informada

---

## Roteiro

### ABERTURA (15 segundos)

```
[TELA: Hobby vs Pro]
```

**NARRAÇÃO:**

> Você precisa do plano Pro? Provavelmente não. Mas vamos ver quando faz sentido.

---

### BLOCO 1: SINAIS DE QUE VOCÊ PRECISA DO PRO (2 minutos)

```
[TELA: Lista de sinais]
```

**NARRAÇÃO:**

> **Sinal 1: Você quer trabalhar em equipe**

```
[DESTAQUE: Ícone de time]
```

> O Hobby é só pra você. Se você quer que outro desenvolvedor acesse o projeto, edite configurações, veja logs... precisa do Pro.
>
> Pro começa em $20/mês por membro.
>
> **Sinal 2: Bandwidth estourando**

```
[DESTAQUE: Gráfico de bandwidth]
```

> Se você tá consistentemente passando de 80-90% dos 100GB, é hora de pensar em Pro.
>
> Pro tem 1TB de bandwidth (10x mais).
>
> **Sinal 3: Projeto comercial sério**

```
[DESTAQUE: Ícone de empresa]
```

> Se você tá fazendo dinheiro com o projeto, ou é pra um cliente pagante, considera Pro.
>
> Não por exigência técnica, mas porque:
> - Suporte é melhor
> - SLA de uptime
> - Mais profissional
>
> **Sinal 4: Precisa de features avançadas**

```
[DESTAQUE: Lista de features]
```

> Algumas coisas só existem no Pro:
> - Password protection em previews
> - Firewall rules avançadas
> - Analytics detalhado
> - Logs de mais de 1 hora

---

### BLOCO 2: DIFERENÇAS PRINCIPAIS (1.5 minutos)

```
[TELA: Tabela comparativa]
```

**NARRAÇÃO:**

> Comparação rápida:

| Feature | Hobby | Pro |
|---------|-------|-----|
| Preço | Grátis | $20/mês/membro |
| Bandwidth | 100GB | 1TB |
| Function Execution | 100GB-h | 1000GB-h |
| Membros | 1 | Ilimitados |
| Suporte | Comunidade | Ticket |
| SLA | Não | 99.99% |
| Preview Protection | Não | Sim |

> **Traduzindo:**
>
> Se você trabalha sozinho e seu projeto é pessoal, Hobby é suficiente pra sempre.
>
> Se você tem time ou cliente pagando, Pro faz sentido.

---

### BLOCO 3: NÃO TENHA MEDO (1 minuto)

```
[TELA: Mensagem tranquilizadora]
```

**NARRAÇÃO:**

> Uma coisa importante:
>
> **Você não precisa decidir agora.**
>
> Começa no Hobby. Usa. Quando/se você esbarrar num limite, a Vercel vai te avisar.
>
> Você pode fazer upgrade a qualquer momento. É só trocar de plano, sem perder nada.

```
[DESTAQUE: Fluxo de upgrade]
```

> E se você fizer upgrade e perceber que não precisava? Faz downgrade no mês seguinte.
>
> Não tem multa, não tem contrato.
>
> **Resumo: Começa grátis, sobe quando precisar.**

---

### FECHAMENTO (30 segundos)

```
[TELA: Fluxograma de decisão]
```

**NARRAÇÃO:**

> Fluxograma final:
>
> - Trabalha sozinho? → Hobby
> - Projeto pessoal? → Hobby
> - Menos de 50k pageviews/mês? → Hobby
> - Time ou cliente? → Considera Pro
> - Bandwidth estourando? → Pro
>
> 95% de vocês vai ficar no Hobby pra sempre. E tá tudo bem.
>
> No próximo módulo, vamos juntar tudo e fazer o projeto final: seu portfolio online!

## Pontos-Chave para Memorização

- Pro = times ou tráfego alto
- Hobby = projetos pessoais
- Upgrade/downgrade a qualquer momento
- 95% fica no Hobby pra sempre
- Não tenha medo, comece grátis
