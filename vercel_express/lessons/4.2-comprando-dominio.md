# Aula 4.2: Comprando Domínio (Registro.br)

**Módulo:** 4 - Domínio Customizado
**Duração:** 5 minutos
**Tipo:** Screencast

---

## Objetivo da Aula

Ao final desta aula, o aluno vai:
- Buscar disponibilidade de domínio
- Completar compra no Registro.br
- Conhecer alternativas para .com

---

```
[TELA: Registro.br]
```

**NARRAÇÃO:**

> Hora de comprar seu domínio. Se você quer um `.com.br`, o lugar oficial é o Registro.br.
>
> Vamos lá.

---

### BLOCO 1: POR QUE REGISTRO.BR (1 minuto)

```
[TELA: Logo Registro.br]
```

**NARRAÇÃO:**

> O Registro.br é o órgão oficial que gerencia todos os domínios `.br` no Brasil.
>
> - `.com.br`
> - `.net.br`
> - `.org.br`
> - `.dev.br`
> - E vários outros
>
> Se você quer qualquer domínio que termina em `.br`, é aqui que você registra.
>
> **Preço atual:** R$ 40/ano (mais ou menos). Comparado com outros países, é barato.
>
> **Alternativas pra .com (sem BR):**
> - Namecheap
> - GoDaddy
> - Google Domains (descontinuado, agora é Squarespace)
>
> Mas pra `.com.br`, Registro.br é o caminho.

---

### BLOCO 2: BUSCANDO DISPONIBILIDADE (1.5 minutos)

```
[SCREENCAST: Registro.br]
```

**NARRAÇÃO:**

> Acessa **registro.br**.

```
[AÇÃO: Abrir registro.br]
```

> Na página inicial tem uma barra de busca. Digita o nome que você quer.

```
[AÇÃO: Digitar "seunome" ou exemplo]
```

> Vou tentar "joaosilva.com.br"...

```
[AÇÃO: Clicar em buscar]
```

> Se aparecer verde com "disponível", ótimo! Você pode registrar.
>
> Se aparecer vermelho com "não disponível", alguém já tem. Tenta variações:
> - joaosilvadev.com.br
> - joaosilva.dev.br
> - jsilvaportfolio.com.br

```
[TELA: Exemplos de variações]
```

> **Dica:** Domínios curtos e fáceis de lembrar são melhores. Evita hífens se puder.

---

### BLOCO 3: PROCESSO DE COMPRA (2 minutos)

```
[SCREENCAST: Fluxo de compra]
```

**NARRAÇÃO:**

> Achou um disponível? Clica em **Registrar**.

```
[AÇÃO: Clicar em Registrar]
```

> Você vai precisar de uma conta no Registro.br. Se não tem, cria uma.

```
[TELA: Tela de login/cadastro]
```

> O cadastro pede CPF (ou CNPJ pra empresa). É obrigatório porque domínios `.br` precisam de titular identificado.

```
[AÇÃO: Mostrar formulário rapidamente]
```

> Depois de logado, você confirma o domínio e vai pro pagamento.
>
> **Formas de pagamento:**
> - Boleto
> - Cartão de crédito
> - Pix (mais rápido)
>
> Com Pix, o domínio é liberado em minutos. Boleto pode demorar 1-2 dias úteis.

```
[TELA: Confirmação de pagamento]
```

> Pagou? Pronto! O domínio é seu por 1 ano. Depois você renova.
