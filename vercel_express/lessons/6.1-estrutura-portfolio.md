**Módulo:** 6 - Projeto Final

### BLOCO 1: O QUE VAMOS CONSTRUIR (1.5 minutos)

```
[TELA: Preview do portfolio]
```

**NARRAÇÃO:**

> Nosso portfolio vai ter:
>
> **Estrutura simples:**
> - Página principal com sua apresentação
> - Seção de projetos
> - Formulário de contato (simulado)
> - Links pro seu GitHub/LinkedIn

```
[DESTAQUE: Estrutura de pastas]
```

> **Estrutura de pastas:**

```
meu-portfolio/
├── index.html       # Página principal
├── style.css        # Estilos
├── script.js        # Interatividade (opcional)
└── assets/
    └── foto.jpg     # Sua foto
```

> Isso é tudo. Sem framework, sem complicação.
>
> Por que HTML/CSS puro? Porque:
> - Faz deploy instantâneo na Vercel
> - Você entende cada linha
> - Funciona pra sempre

---

### BLOCO 2: CRIANDO O PROJETO (3 minutos)

```
[SCREENCAST: VS Code ou editor de texto]
```

**NARRAÇÃO:**

> Vamos criar do zero. Abre seu terminal ou VS Code.

```
[AÇÃO: Criar pasta]
```

```bash
mkdir meu-portfolio
cd meu-portfolio
```

> Agora cria o arquivo `index.html`:

```
[AÇÃO: Criar index.html]
```

```html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Seu Nome - Dev</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <header>
        <h1>Olá, eu sou [Seu Nome]</h1>
        <p>Desenvolvedor(a) em formação</p>
    </header>

    <section id="sobre">
        <h2>Sobre Mim</h2>
        <p>Escreva um parágrafo sobre você...</p>
    </section>

    <section id="projetos">
        <h2>Projetos</h2>
        <div class="projeto">
            <h3>Projeto 1</h3>
            <p>Descrição do projeto...</p>
            <a href="#">Ver código</a>
        </div>
    </section>

    <section id="contato">
        <h2>Contato</h2>
        <p>Email: seuemail@exemplo.com</p>
        <a href="https://github.com/seuusuario">GitHub</a>
        <a href="https://linkedin.com/in/seuusuario">LinkedIn</a>
    </section>

    <footer>
        <p>&copy; 2024 - Feito com ❤️ e Vercel</p>
    </footer>
</body>
</html>
```

```
[DESTAQUE: Código HTML]
```

> Esse é o esqueleto. Personaliza com suas informações.

### BLOCO 3: ADICIONANDO ESTILO (2.5 minutos)

```
[AÇÃO: Criar style.css]
```

**NARRAÇÃO:**

> Agora o CSS pra deixar bonito:

```css
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
    line-height: 1.6;
    color: #333;
    max-width: 800px;
    margin: 0 auto;
    padding: 20px;
}

header {
    text-align: center;
    padding: 60px 20px;
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    color: white;
    border-radius: 10px;
    margin-bottom: 40px;
}

h1 {
    font-size: 2.5rem;
    margin-bottom: 10px;
}

section {
    margin-bottom: 40px;
}

h2 {
    border-bottom: 2px solid #667eea;
    padding-bottom: 10px;
    margin-bottom: 20px;
}

.projeto {
    background: #f5f5f5;
    padding: 20px;
    border-radius: 8px;
    margin-bottom: 15px;
}

a {
    color: #667eea;
    text-decoration: none;
}

a:hover {
    text-decoration: underline;
}

footer {
    text-align: center;
    padding: 20px;
    color: #666;
}
```

```
[DESTAQUE: Resultado visual]
```

> Simples, limpo, profissional.
>
> Você pode customizar as cores, fontes, layout depois. O importante agora é ter algo funcionando.

---

### BLOCO 4: SUBINDO PRO GITHUB (2 minutos)

```
[SCREENCAST: Terminal]
```

**NARRAÇÃO:**

> Agora vamos subir pro GitHub. Se você ainda não tem Git instalado, instala primeiro.

```
[AÇÃO: Comandos Git]
```

```bash
git init
git add .
git commit -m "Portfolio inicial"
```

> Vai no GitHub, cria um repositório novo chamado "meu-portfolio" (ou o nome que quiser).
>
> Não marca nenhuma opção, deixa vazio.

```
[AÇÃO: Mostrar GitHub - criar repo]
```

> Depois:

```bash
git remote add origin https://github.com/seuusuario/meu-portfolio.git
git branch -M main
git push -u origin main
```

```
[DESTAQUE: Repo no GitHub]
```

> Pronto! Seu código tá no GitHub.
