# Aula 6.3: Checklist Final - Validação Completa

**Módulo:** 6 - Projeto Final
**Duração:** 5 minutos
**Tipo:** Validação + ajustes

---

## Objetivo da Aula

Ao final desta aula, o aluno vai:
- Validar todos os componentes do deploy
- Identificar e corrigir problemas comuns
- Ter certeza de que tudo funciona

### BLOCO 3: ERROS COMUNS E CORREÇÕES (1.5 minutos)

```
[TELA: Problemas frequentes]
```

**NARRAÇÃO:**

> Erros que vejo frequentemente:
>
> **Erro 1: Página em branco**

```
[DESTAQUE: Erro 1]
```

> Causa: arquivo não se chama `index.html` (letra maiúscula ou extensão errada)
> Correção: renomeia pra `index.html` exatamente
>
> **Erro 2: CSS não carrega**

```
[DESTAQUE: Erro 2]
```

> Causa: caminho errado no `<link href="">`
> Correção: usa caminho relativo: `href="style.css"` ou `href="./style.css"`
>
> **Erro 3: Imagem não aparece**

```
[DESTAQUE: Erro 3]
```

> Causa: caminho errado ou imagem não foi commitada
> Correção: verifica se a imagem tá no Git e o caminho tá certo
>
> **Erro 4: Domínio não funciona**

```
[DESTAQUE: Erro 4]
```

> Causa: DNS ainda propagando
> Correção: espera 15-30 minutos, verifica no whatsmydns.net
>
> **Erro 5: Sem HTTPS (cadeado)**

```
[DESTAQUE: Erro 5]
```

> Causa: certificado ainda gerando
> Correção: espera mais 5-10 minutos após DNS validar
