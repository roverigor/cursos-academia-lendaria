# Prompts de Social Listening - Modulo 4

## Trilha 9 - Social Media em Piloto

---

## PROMPT 1: Gerar Lista de Palavras-Chave

```
Crie uma lista de palavras-chave para social listening.

MEU NEGOCIO:
- Nome da empresa: [nome]
- Meu nome: [nome pessoal]
- Produtos/servicos: [liste]
- Nicho: [seu nicho]
- ICP: [cliente ideal]

MEUS CONCORRENTES:
1. [Nome] - @[perfil]
2. [Nome] - @[perfil]
3. [Nome] - @[perfil]

PRINCIPAIS DORES DO MEU CLIENTE:
1. [Dor 1]
2. [Dor 2]
3. [Dor 3]

GERE 3 LISTAS:

CAMADA 1 - MINHA MARCA (5-7 termos):
- Variacoes do nome
- Erros comuns de digitacao
- Hashtags proprias

CAMADA 2 - CONCORRENTES (3-5 termos):
- Nomes
- Termos de reclamacao ("problema com X")

CAMADA 3 - MERCADO (5-7 termos):
- Dores expressadas ("preciso de ajuda com...")
- Perguntas comuns
- Tendencias

PRIORIZE por probabilidade de gerar leads.
```

## PROMPT 2: Analisar Sentimento de Mencoes

```
Analise o sentimento destas mencoes sobre [marca/tema].

MENCOES:
1. "[Cole mencao 1]"
2. "[Cole mencao 2]"
3. "[Cole mencao 3]"
[Continue...]

MINHA MARCA: [nome]
MEU NICHO: [nicho]

PARA CADA MENCAO, CLASSIFIQUE:
| # | Sentimento | Score (-10 a +10) | Urgencia | Topico | Acao Sugerida |

RESUMO:
- Total positivas: X (X%)
- Total negativas: X (X%)
- Total neutras: X (X%)
- Sentimento geral: [positivo/negativo/neutro]

INSIGHTS:
1. Tema mais recorrente nas positivas
2. Tema mais recorrente nas negativas
3. Oportunidade identificada
4. Risco identificado

PRIORIDADES:
- Top 3 mencoes para responder primeiro
- Por que cada uma e prioritaria

SUGESTAO DE RESPOSTA para cada prioritaria.
```

## PROMPT 3: Mapeamento de Concorrentes

```
Mapeie a presenca social dos meus concorrentes.

CONCORRENTES:
1. [Nome] - @[instagram] - [linkedin]
2. [Nome] - @[instagram] - [linkedin]
3. [Nome] - @[instagram] - [linkedin]

MEU NICHO: [nicho]

PARA CADA CONCORRENTE, ANALISE:
1. Frequencia de publicacao
2. Tipos de conteudo (educativo, vendas, etc)
3. Engajamento aparente
4. Tom de voz
5. Pontos fortes
6. Pontos fracos

TABELA COMPARATIVA:
| Aspecto | Eu | Conc. 1 | Conc. 2 | Conc. 3 |

OPORTUNIDADES:
- O que nenhum deles faz que eu poderia fazer
- Gaps no mercado
- Publicos nao atendidos

AMEACAS:
- O que eles fazem melhor que eu
- Tendencias que estao surfando

RECOMENDACOES: 3 acoes para me diferenciar
```

## PROMPT 4: Gerar Respostas para Mencoes

```
Gere respostas para estas mencoes nas redes sociais.

MENCOES:
1. [Positiva]: "[texto da mencao]"
2. [Negativa]: "[texto da mencao]"
3. [Neutra/Pergunta]: "[texto da mencao]"
[Continue...]

MINHA MARCA: [nome]
MEU TOM DE VOZ: [profissional/casual/amigavel]

PARA CADA RESPOSTA:
- Mantenha meu tom de voz
- Seja conciso (max 2-3 frases)
- Personalize se possivel
- Inclua CTA quando apropriado

FORMATO:
Mencao: [texto]
Sentimento: [pos/neg/neu]
Resposta: [sua sugestao]
Por que funciona: [1 frase]

REGRAS:
- Positivas: agradecer genuinamente, nao ser baba-ovo
- Negativas: reconhecer, resolver, nao ser defensivo
- Neutras: informar, engajar, abrir conversa
```

---

## PROMPT 5: Extrair Insights de Mencoes

```
Extraia insights estrategicos destas mencoes.

MENCOES DO PERIODO [semana/mes]:
[Cole todas as mencoes]

MEU CONTEXTO:
- Negocio: [tipo]
- Objetivo atual: [qual]
- Desafio atual: [qual]

ANALISE E IDENTIFIQUE:

1. TEMAS RECORRENTES
| Tema | Frequencia | Sentimento | Oportunidade |

2. DORES EXPRESSADAS
- O que as pessoas reclamam/pedem
- Frequencia de cada dor

3. OPORTUNIDADES DE CONTEUDO
- 3 ideias de post baseadas nas mencoes
- Por que cada uma e relevante

4. OPORTUNIDADES DE PRODUTO/SERVICO
- Gaps identificados
- Demandas nao atendidas

5. ALERTAS
- Riscos de reputacao
- Tendencias negativas

ACAO PRIORITARIA: A 1 coisa mais importante para fazer baseado nisso.
```

## PROMPT 6: Setup de Sistema de Listening

```
Me ajude a configurar um sistema de social listening.

MEU PERFIL:
- Tamanho do negocio: [solo/pequena equipe/empresa]
- Orcamento mensal: [zero/ate R$100/ate R$500]
- Tempo disponivel: [horas por semana]
- Nivel tecnico: [iniciante/intermediario/avancado]
- Canais prioritarios: [liste]

MEU OBJETIVO:
- [Monitorar marca/Encontrar leads/Analise de mercado/Tudo]

CRIE:

1. STACK RECOMENDADO
| Funcao | Ferramenta | Custo | Por que |

2. TERMOS PARA MONITORAR
- Lista inicial de 10 termos

3. SETUP PASSO A PASSO
- Como configurar cada ferramenta
- Tempo estimado de setup

4. ROTINA SEMANAL
| Dia | Tarefa | Tempo |

5. ALERTAS A CONFIGURAR
- Quais gatilhos
- Como ser notificado

CUSTO TOTAL: R$ X/mes
TEMPO TOTAL: X horas/semana
```

---

## DICAS DE USO

1. **Cole mencoes reais** - Quanto mais contexto, melhor a analise
2. **Faca semanalmente** - Social listening e rotina
3. **Aja rapido em negativos** - Use as respostas sugeridas
4. **Documente insights** - Crie um banco de aprendizados
