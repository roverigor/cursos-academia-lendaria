# Prompts de Pipeline de Publicacao - Modulo 3

## Trilha 9 - Social Media em Piloto

---

## PROMPT 1: Otimizar Horarios de Publicacao

```
Analise os melhores horarios para postar no [Instagram/LinkedIn/TikTok].

MEU CONTEXTO:
- Nicho: [seu nicho]
- Publico-alvo: [descreva - idade, profissao, rotina]
- Plataforma: [qual]
- Objetivo: [engajamento/leads/vendas]

CONSIDERE:
1. Horarios de maior atividade do meu publico
2. Dias da semana com melhor performance tipica
3. Concorrencia de conteudo (quando tem menos)
4. Fuso horario (Brasil)

RETORNE:
| Dia | Melhor Horario | Segundo Melhor | Tipo de Post Ideal |
|-----|----------------|----------------|-------------------|

PLUS:
- 3 horarios para evitar
- Diferenca entre dias uteis e fim de semana
- Dica especifica para meu nicho
```

---

## PROMPT 2: Criar Calendario Editorial

```
Crie um calendario editorial para o proximo mes.

DADOS:
- Plataforma: [Instagram/LinkedIn/etc]
- Frequencia: [X posts por semana]
- Dias de publicacao: [seg, ter, qua...]
- Horarios: [seus horarios]
- Meus pilares: [liste os 5]

DISTRIBUICAO:
- Topo: 50%
- Meio: 30%
- Fundo: 20%

CONSIDERE:
- Variedade de formatos ao longo da semana
- Datas importantes: [liste feriados, datas do seu nicho]
- Progressao logica de temas

FORMATO:
Tabela com 4 semanas:
| Semana | Dia | Horario | Etapa | Pilar | Tema Sugerido | Formato |
```

## PROMPT 3: Calcular Custo de Publicacao Manual

```
Calcule o custo da minha publicacao manual vs automatizada.

MINHA ROTINA ATUAL:
- Tempo para criar 1 post: [X] minutos
- Tempo para editar: [X] minutos
- Tempo para publicar: [X] minutos
- Tempo para responder: [X] minutos
- Posts por semana: [X]
- Meu valor/hora estimado: R$ [X]

CALCULE:
1. Tempo total por mes (manual)
2. Custo em R$ por mes
3. Custo de oportunidade (o que poderia fazer)

COMPARE COM BATCH CONTENT:
- Tempo de batch (2h/semana x 4 = 8h/mes)
- Economia de tempo
- Economia em R$
- ROI da mudanca

CONCLUSAO: Vale a pena mudar?
```

## PROMPT 4: Configurar Workflow de Automacao

```
Crie um workflow de automacao para meu conteudo.

MEU STACK ATUAL:
- Planilha: [Google Sheets/Notion/Airtable/nenhum]
- Automacao: [n8n/Zapier/Make/nenhum]
- IA: [ChatGPT/Claude/qual]
- Agendamento: [Buffer/Meta/Later/qual]

MEU PROCESSO ATUAL:
1. [Descreva como cria conteudo hoje]
2. [Como agenda]
3. [Como publica]

MEU OBJETIVO:
- Frequencia: [X posts/semana]
- Canais: [liste]
- Nivel de automacao desejado: [basico/intermediario/avancado]

CRIE:
1. Fluxo visual do workflow (diagrama em texto)
2. Ferramentas necessarias para cada etapa
3. Prompts/templates necessarios
4. Tempo estimado de setup
5. Tempo semanal apos setup

CUSTO ESTIMADO: Mensal em R$
```

## PROMPT 5: Validar Posts Antes de Agendar

```
Valide estes [X] posts antes de eu agendar.

POSTS:
1. [Cole post 1]
2. [Cole post 2]
3. [Cole post 3]
...

MEU OBJETIVO: [engajamento/leads/vendas]
MINHA PLATAFORMA: [Instagram/LinkedIn]
MEU TOM: [profissional/casual]

PARA CADA POST, AVALIE (1-10):
1. Gancho (chama atencao?)
2. Clareza (facil de entender?)
3. Valor (entrega algo util?)
4. CTA (acao clara?)
5. Adequacao (fit com plataforma?)

IDENTIFIQUE:
- Erros de texto/gramatica
- Pontos fracos
- 1 melhoria especifica

VEREDICTO POR POST:
- Publicar como esta
- Pequeno ajuste (especifique)
- Reescrever (motivo)

ORDEM SUGERIDA: Qual publicar primeiro, segundo...
```

---

## DICAS DE USO

1. **Use para planejar a semana** - Domingo a noite, 15 min
2. **Valide antes de agendar** - Evita erros embaracosos
3. **Itere no workflow** - Comece simples, complexifique depois
4. **Documente o que funciona** - Crie seu proprio SOP
