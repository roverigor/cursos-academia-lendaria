## PROMPT 1: Analisar Post Com Potencial Viral

```
Analise este post que esta performando bem.

POST:
- Tema: [tema do post]
- Formato: [carrossel/reels/imagem]
- Gancho: [primeira linha]
- Plataforma: [Instagram/LinkedIn/TikTok]

METRICAS ATUAIS:
- Alcance: [X]
- Curtidas: [X]
- Salvamentos: [X]
- Compartilhamentos: [X]
- Comentarios: [X]
- Tempo desde publicacao: [X horas]

ANALISE:

1. METRICAS VIRAIS
- Velocity: (Engajamento / Alcance) x 100 = ?
- Share Rate: (Compartilhamentos / Alcance) x 100 = ?
- Save Rate: (Salvamentos / Alcance) x 100 = ?

2. STATUS
- Normal / Aquecendo / Viral?
- Por que chegou a essa conclusao?

3. ANATOMIA
- Qual dos 5 elementos virais esta mais forte?
  (Emocao, Simplicidade, Utilidade, Historia, Timing)
- Por que esse elemento funcionou?

4. CAPITALIZACAO
- O que fazer AGORA (proxima 1h)?
- O que preparar para proximas 24h?
- Como transformar em vendas?

ENTREGUE: Plano de acao imediato
```

## DICAS DE USO

1. **Use em tempo real** - Quando um post esta aquecendo, aja rapido
2. **Documente tudo** - Cada viral e uma aula
3. **Nao dependa de viral** - Sistema > Sorte
4. **Capitalize sempre** - Viral sem acao = desperdicio
