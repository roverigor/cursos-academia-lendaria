# Prompts de Criacao de Conteudo - Modulo 2

## Trilha 9 - Social Media em Piloto

---

## PROMPT 1: Definir Pilares de Conteudo

```
Me ajude a definir meus 5 pilares de conteudo para redes sociais.

MEU NEGOCIO:
- O que vendo: [produto/servico]
- Para quem: [ICP - cliente ideal]
- Principais dores do meu cliente: [liste 3-5 dores]
- Principais desejos do meu cliente: [liste 3-5 desejos]
- Meu diferencial: [o que me diferencia]

MEU OBJETIVO NAS REDES:
- [Autoridade/Leads/Vendas/Engajamento]

CRIE:
5 pilares de conteudo, onde cada pilar:
- Nome do pilar
- Descricao (1 frase)
- 3-5 subtemas
- Tipo de post ideal (educativo, inspirador, etc)
- % do meu conteudo (total = 100%)

FORMATO: Tabela organizada
```

---

## PROMPT 2: Gerar Ideias Topo de Funil

```
Gere 15 ideias de posts TOPO DE FUNIL para [plataforma].

CONTEXTO:
- Meu nicho: [nicho]
- Meu ICP: [cliente ideal]
- Meus pilares: [liste seus 5 pilares]
- Tom de voz: [profissional/casual/inspirador]

OBJETIVO: Atrair pessoas que NAO me conhecem

PARA CADA IDEIA:
- Gancho (primeira linha chamativa, max 10 palavras)
- Tema/Assunto
- Formato sugerido (carrossel/reels/imagem)
- Emocao que provoca (curiosidade/surpresa/etc)

REGRAS:
- Ideias devem ser sobre problemas/dores gerais, nao sobre meu produto
- Gancho deve parar o scroll
- Conteudo deve ser compartilhavel

FORMATO: Tabela numerada
```

---

## PROMPT 3: Gerar Ideias Meio de Funil

```
Gere 10 ideias de posts MEIO DE FUNIL.

CONTEXTO:
- Meu nicho: [nicho]
- Meus produtos/servicos: [liste]
- Minha metodologia/abordagem: [descreva]
- Resultados que entrego: [liste]

OBJETIVO: Gerar confianca em quem ja me segue

TIPOS DE POST:
- Cases de sucesso (clientes)
- Tutoriais e how-to
- Bastidores do meu trabalho
- Antes/Depois
- Metodologia explicada
- FAQ respondido

PARA CADA IDEIA:
- Gancho
- Tipo (case/tutorial/bastidor/etc)
- Formato sugerido
- Por que gera confianca

FORMATO: Tabela numerada
```

---

## PROMPT 4: Gerar Ideias Fundo de Funil

```
Gere 6 ideias de posts FUNDO DE FUNIL para converter seguidores em clientes.

MINHAS OFERTAS:
1. [Produto/Servico 1] - R$ [valor] - [para quem e]
2. [Produto/Servico 2] - R$ [valor] - [para quem e]
3. [Servico gratuito/lead magnet se tiver]

GATILHOS QUE POSSO USAR:
- Escassez (vagas limitadas, prazo)
- Prova social (X clientes, resultados)
- Garantia (o que ofereco)
- Urgencia (por que agir agora)

PARA CADA IDEIA:
- Gancho focado em transformacao
- Oferta relacionada
- CTA especifico (ex: "Comenta EU", "Link na bio")
- Gatilho utilizado

REGRAS:
- Foco na transformacao, nao no produto
- CTA claro e unico
- Nao ser agressivo/vendedor demais

FORMATO: Tabela numerada
```

---

## PROMPT 5: Desenvolver Post Completo

```
Desenvolva este post para [Instagram/LinkedIn/TikTok]:

IDEIA: [titulo/gancho do post]
FORMATO: [carrossel de X slides / reels / imagem unica / stories]
ETAPA: [Topo/Meio/Fundo de funil]
OBJETIVO: [engajamento/leads/vendas]

MEU TOM DE VOZ: [profissional/casual/inspirador/provocador]
MEU PUBLICO: [descreva brevemente]

ESTRUTURA DESEJADA:
- Gancho forte (primeira linha)
- Desenvolvimento (corpo do conteudo)
- CTA claro

RESTRICOES:
- Maximo [X] palavras no total
- Linguagem acessivel (evitar jargoes)
- [Outras restricoes]

ENTREGUE:
1. Texto completo do post
2. Sugestao de visual (descricao)
3. 5-10 hashtags relevantes
4. Melhor horario para postar
```

---

## PROMPT 6: Criar Carrossel Completo

```
Crie um carrossel de [X] slides sobre: [tema]

CONTEXTO:
- Plataforma: [Instagram/LinkedIn]
- Publico: [seu ICP]
- Objetivo: [educar/inspirar/vender]

ESTRUTURA:
- Slide 1: Capa com gancho forte
- Slides 2-[X-1]: Conteudo (1 ponto por slide)
- Slide final: Resumo + CTA

PARA CADA SLIDE:
- Titulo/Headline
- Texto de apoio (2-3 linhas max)
- Sugestao visual

REGRAS:
- Cada slide deve fazer sentido sozinho
- Texto legivel (fontes grandes)
- Progressao logica

FORMATO: Separado por slide
```

---

## PROMPT 7: Adaptar Post para Outra Plataforma

```
Adapte este post para [nova plataforma].

POST ORIGINAL ([plataforma original]):
[Cole o post aqui]

ADAPTAR PARA: [Instagram/LinkedIn/TikTok/Twitter]

CONSIDERE:
- Tom de voz da nova plataforma
- Limites de caracteres
- Formato preferido (video/texto/imagem)
- Hashtags relevantes

ENTREGUE:
1. Post adaptado completo
2. O que mudou e por que
3. Sugestao de formato
4. Hashtags (se aplicavel)
```

---

## PROMPT 8: Criar Batch de 30 Posts

```
Crie um plano de 30 posts para o proximo mes.

CONTEXTO:
- Meu nicho: [nicho]
- Meus pilares: [liste os 5]
- Minhas ofertas: [liste]
- Frequencia: [X posts por semana]

DISTRIBUICAO:
- 50% Topo de Funil (15 posts)
- 30% Meio de Funil (9 posts)
- 20% Fundo de Funil (6 posts)

PARA CADA POST:
- Numero
- Semana/Dia sugerido
- Etapa do funil
- Pilar
- Gancho
- Formato

CONSIDERE:
- Variedade de formatos
- Variedade de pilares
- Progressao logica no mes
- Datas especiais: [liste se houver]

FORMATO: Tabela com 30 linhas
```

---

## DICAS DE USO

1. **Salve os prompts que funcionam** - Crie sua biblioteca
2. **Ajuste o tom** - Peca para a IA reescrever se nao soar como voce
3. **Itere** - "Faca mais [especifico]" ou "Menos [algo]"
4. **Valide** - Sempre revise antes de publicar
