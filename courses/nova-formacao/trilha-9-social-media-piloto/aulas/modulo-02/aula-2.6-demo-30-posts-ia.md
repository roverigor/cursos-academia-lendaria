# Aula 2.6: Demo - 30 Posts Criados com IA em 20 Minutos

## Trilha 9 - Social Media em Piloto | Modulo 2

---

> **Duracao:** 10 minutos
> **Tipo:** Demonstracao
> **Caso:** Criacao em massa com ChatGPT/Claude

---

## GANCHO DE ABERTURA

**"Voce vai ver, passo a passo, como criar um mes inteiro de conteudo em menos de 20 minutos. Sem desculpas."**

---

## CONTEXTO DA DEMO

**Empresa:** Consultoria de Marketing Digital
**Objetivo:** 30 posts para o proximo mes
**Proporcao:** 50% Topo, 30% Meio, 20% Fundo
**Tempo total:** 20 minutos

---

## PASSO 1: DEFINIR PILARES (2 min)

Prompt usado:

```
Sou consultor de marketing digital para PMEs.
Meu ICP: empresarios que faturam 50-250K/mes.
Principais dores: nao sabem usar IA, gastam muito em ads, nao tem tempo.

Sugira 5 pilares de conteudo para meu Instagram.
Para cada pilar, de 3 subtemas.
```

**Resultado:**

| Pilar | Subtemas |
|-------|----------|
| 1. IA para Negocios | ChatGPT, automacao, clones |
| 2. Trafego Pago | ROI, reducao de CAC, criativos |
| 3. Produtividade | Ferramentas, rotina, delegacao |
| 4. Cases de Sucesso | Clientes, resultados, bastidores |
| 5. Ofertas | Mentoria, consultoria, cursos |

---

## PASSO 2: GERAR IDEIAS TOPO (5 min)

Prompt usado:

```
Gere 15 ideias de post TOPO DE FUNIL para Instagram.
Tema: Marketing digital e IA para PMEs.
Pilares: [os 5 pilares]

Cada ideia deve ter:
- Gancho (primeira linha)
- Formato sugerido (carrossel, reels, etc)
- Objetivo (awareness, engajamento)

Ideias devem ser para pessoas que NAO me conhecem.
```

**Resultado (exemplo):**

| # | Gancho | Formato |
|---|--------|---------|
| T1 | "5 ferramentas de IA que todo empresario deveria usar" | Carrossel |
| T2 | "Voce ainda paga agencia? Faca voce mesmo com isso" | Reels |
| T3 | "O erro que esta queimando seu dinheiro em ads" | Carrossel |
| ... | ... | ... |

---

## PASSO 3: GERAR IDEIAS MEIO (4 min)

Prompt usado:

```
Gere 10 ideias de post MEIO DE FUNIL.
Para pessoas que ja me seguem.
Foco em: cases, tutoriais, bastidores.

Cada ideia deve gerar CONFIANCA e mostrar minha metodologia.
```

**Resultado (exemplo):**

| # | Gancho | Formato |
|---|--------|---------|
| M1 | "Como meu cliente saiu de 0 para 50 leads/mes" | Carrossel case |
| M2 | "Passo a passo: configurando ChatGPT para vendas" | Reels tutorial |
| M3 | "Bastidores: como eu crio 30 posts em 20 min" | Stories |
| ... | ... | ... |

---

## PASSO 4: GERAR IDEIAS FUNDO (3 min)

Prompt usado:

```
Gere 5 ideias de post FUNDO DE FUNIL.
Para pessoas prontas para comprar.
Ofertas: mentoria, consultoria, curso.

Cada post deve ter CTA claro.
Use gatilhos: escassez, urgencia, prova social.
```

**Resultado (exemplo):**

| # | Gancho | CTA |
|---|--------|-----|
| F1 | "Ultimas 5 vagas para mentoria de marco" | DM "VAGA" |
| F2 | "500 empresarios ja usam meu metodo" | Link na bio |
| F3 | "Consultoria gratis para os 3 primeiros" | Comenta "EU" |
| ... | ... | ... |

---

## PASSO 5: DESENVOLVER POSTS (5 min)

Prompt para desenvolver cada post:

```
Desenvolva este post para Instagram:

IDEIA: [T1 - 5 ferramentas de IA]
FORMATO: Carrossel de 7 slides
TOM: Profissional mas acessivel
CTA: Salvar o post

Estrutura:
- Slide 1: Capa com gancho
- Slides 2-6: Uma ferramenta por slide
- Slide 7: Resumo + CTA

Para cada ferramenta: nome, para que serve, exemplo pratico.
```

---

---

## DICAS DA DEMO

1. **Batch por etapa** - Faca todo Topo, depois todo Meio, etc.
2. **Nao edite agora** - Primeiro gera tudo, depois refina
3. **Salve prompts** - Reutilize mes a mes
4. **Ajuste a voz** - Peca para IA imitar seu tom
5. **Valide com dados** - Apos publicar, veja o que funcionou
