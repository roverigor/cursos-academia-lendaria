## Objetivo do Modulo

Criar um **sistema de rastreamento de viralizacao** e um plano de escala baseado em dados reais.

| Aula | Titulo | Duracao | Tipo |

- [ ] Sistema de Social Listening funcionando (Modulo 4)

**Viral Tracker + Plano de Escala** com:
- Dashboard de metricas de viralizacao

- Playbook de capitalizacao (o que fazer quando viraliza)

- ROI projetado
