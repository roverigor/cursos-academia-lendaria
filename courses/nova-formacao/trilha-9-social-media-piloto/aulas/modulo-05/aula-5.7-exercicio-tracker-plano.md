# Aula 5.7: Exercicio - Seu Tracker + Plano de 30 Dias

## Trilha 9 - Social Media em Piloto | Modulo 5

---

> **Duracao:** 28 minutos
> **Tipo:** Pratica Guiada
> **Entregavel:** Viral Tracker + Plano de Escala

---

## GANCHO DE ABERTURA

**"Nos proximos 28 minutos, voce vai sair com um sistema de identificacao de virais E um plano concreto para escalar o que funciona."**

---

## INSTRUCOES GERAIS

- Use os dados reais da sua conta
- Nao busque perfeicao, busque acao
- Melhor um tracker simples que voce usa do que um complexo que abandona
- O plano de 30 dias e um ponto de partida, nao uma camisa de forca

---

## BLOCO 1: CRIAR TRACKER (8 min)

### Tarefas

- [ ] Criar planilha ou documento
- [ ] Adicionar estrutura do template
- [ ] Listar seus 5 posts recentes
- [ ] Preencher metricas basicas

### Estrutura Minima do Tracker

**Aba 1: Posts em Monitoramento**

| Post | Data | Alcance | Salvamentos | Compartilhamentos | Score Viral |
|------|------|---------|-------------|-------------------|-------------|
| | | | | | |

| Gatilho | Threshold | Acao |
|---------|-----------|------|
| Velocity > 10% | 1h | Responder comentarios |
| Share Rate > 3% | 24h | Ativar playbook |
| Save Rate > 5% | 24h | Criar variacao |

**Aba 3: Historico**

| Post Viral | Data | Resultado | Licao |
|------------|------|-----------|-------|
| | | | |

### Prompt IA: Gerar Estrutura

```
Crie a estrutura do meu Viral Tracker.

MEU PERFIL:
- Posts por semana: [numero]
- Plataforma principal: [qual]
- Tempo para monitorar: [horas/semana]

PRECISO:
1. Template simples de monitoramento
2. Lista de alertas/gatilhos
3. Checklist de acao quando viralizar

FORMATO: Descricao que eu possa replicar em planilha
```

### Prompt IA: Analisar Top Post

```
Analise meu post com maior score viral.

POST:
- Tema: [tema]
- Formato: [formato]
- Metricas: Alcance: ___, Salvamentos: ___, Compartilhamentos: ___

### Prompt IA: Identificar Padroes

```
Analise estes 5 posts e identifique padroes.

POST 1: [tema, formato, horario, metricas]
POST 2: [tema, formato, horario, metricas]

## BLOCO 4: PLANO DE ESCALA 30 DIAS (5 min)

### Tarefas

- [ ] Definir foco de cada semana
- [ ] Estabelecer metas simples
- [ ] Agendar revisoes

### Template do Plano

| Semana | Foco | Acao Concreta | Meta |
|--------|------|---------------|------|
| 1 | Replicar top post | Criar 1 variacao do melhor post | 1 post variacao |
| 2 | Testar formato | Experimentar formato diferente | 2 posts novos formatos |
| 3 | Otimizar horario | Postar no melhor horario identificado | +10% engajamento |
| 4 | Criar oferta | Usar tema viral para campanha | 1 campanha com CTA |
