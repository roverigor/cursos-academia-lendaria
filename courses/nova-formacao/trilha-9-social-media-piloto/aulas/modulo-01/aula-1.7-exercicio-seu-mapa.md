### BLOCO 3: Investimento de Tempo (5 min)

Seja HONESTO sobre quanto tempo gasta:

| Canal | Horas/Semana | Seu Valor/Hora | Custo Mensal |
|-------|--------------|----------------|--------------|
| | h | R$ | R$ |
| | h | R$ | R$ |
| | h | R$ | R$ |
| | h | R$ | R$ |
| **TOTAL** | h | - | **R$** |

**Calculo:**
```
Custo Mensal = Horas/Semana x 4 x Valor/Hora
```

**Nao sabe seu valor/hora?**
Use: Faturamento mensal / 160 horas

### BLOCO 4: ROI por Canal (5 min)

| Canal | Receita Atribuida | Custo (Tempo) | ROI |
|-------|-------------------|---------------|-----|
| | R$ | R$ | x |
| | R$ | R$ | x |
| | R$ | R$ | x |
| | R$ | R$ | x |

**Calculo:**
```
ROI = Receita / Custo
```

**Interpretacao:**
- ROI > 3x = Escalar
- ROI 1-3x = Manter
- ROI < 1x = Otimizar ou Abandonar
