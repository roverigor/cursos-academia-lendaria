## Aula 1.2: Piramide de Conteudo - Teoria Minima (10 min)

### A Piramide

```
         /\
        /  \  FUNDO (20%)
       / $$ \  Venda direta
      /------\
     /        \  MEIO (30%)
    /  Cases   \  Gera confianca
   /  Tutoriais \
  /--------------\
 /                \  TOPO (50%)
/   Dicas, Trends  \  Atrai novos
/   Entretenimento  \
--------------------
```

### Resumo Pratico

| Etapa | % | O Que | Exemplo |
|-------|---|-------|---------|
| TOPO | 50% | Atrai desconhecidos | "5 erros que todo empresario comete" |
| MEIO | 30% | Gera confianca | "Como meu cliente aumentou 50% vendas" |
| FUNDO | 20% | Converte em venda | "Ultimas 3 vagas para mentoria" |

### Regra 5-3-2

Se voce posta 10x por semana:
- 5 posts Topo
- 3 posts Meio
- 2 posts Fundo

## Aula 2.2: Batch Content + Ferramentas - Teoria Minima (10 min)

### O Que E Batch Content

```
ERRADO (diario):
Seg: pensar + criar + publicar
Ter: pensar + criar + publicar
= 3h/semana, stress diario

CERTO (batch):
1 dia: pensar + criar TUDO
Resto: so agendar
= 2h/mes, zero stress
```

### Formula do Batch

```
1 sessao de 2h = 30 dias de conteudo

- 30 min: Definir temas (piramide T/M/F)
- 60 min: Gerar com IA
- 30 min: Revisar e agendar
```

### Regra 2-1-1

- **2 horas/mes:** Sessao batch de criacao
- **1 hora/semana:** Revisao e ajustes
- **1 min/dia:** Responder comentarios

## Aula 3.2: Viral Tracking - Teoria Minima (10 min)

### Anatomia do Viral

```
VIRAL = Emocao + Timing + Formato
```

### 5 Gatilhos de Compartilhamento

| Gatilho | O Que E | Exemplo |
|---------|---------|---------|
| Utilidade | Resolve problema | "5 ferramentas gratis para X" |
| Identificacao | "Isso sou eu" | "POV: voce como empresario" |
| Controversia | Opiniao forte | "Por que X esta errado" |
| Novidade | Algo novo | "Nova funcao do Instagram" |
| Emocao | Faz sentir | Historia de superacao |

### Framework RAPI

- **R**econhecer: Identifique a trend
- **A**nalisar: Por que esta viralizando?
- **P**ersonalizar: Adapte pro seu nicho
- **I**mplementar: Publique rapido (< 48h)

### Timing

```
Dia 1-2: Nascimento ← IDEAL
Dia 3-5: Crescimento ← BOM
Dia 6-10: Pico ← TARDE
Dia 11+: Declinio ← NAO FACA
```
