# AULA 3.2: TEORIA MINIMA (10 min)

## Viral Tracking

### Anatomia do Viral

```
VIRAL = Emocao + Timing + Formato

Emocao: Faz sentir algo forte
Timing: No momento certo
Formato: Facil de consumir/compartilhar
```

### 5 Gatilhos de Compartilhamento

| Gatilho | O Que E | Exemplo |
|---------|---------|---------|
| Utilidade | Resolve problema | "5 ferramentas gratis" |
| Identificacao | "Isso sou eu" | "POV: voce como empresario" |
| Controversia | Opiniao forte | "Por que X esta errado" |
| Novidade | Algo novo | "Nova funcao do Instagram" |
| Emocao | Faz rir/chorar | Historia de superacao |

### Framework RAPI

```
R - Reconhecer: Identifique a trend
A - Analisar: Por que esta viralizando?
P - Personalizar: Adapte pro seu nicho
I - Implementar: Publique rapido (< 48h)
```

### Timing e Tudo

```
Dia 1-2: Nascimento ← IDEAL
Dia 3-5: Crescimento ← BOM
Dia 6-10: Pico ← TARDE
Dia 11+: Declinio ← NAO FACA
```

**Regra:** Se viu 10+ vezes, ja esta saturado.

### Quando NAO Surfar

Evite trends que:
- Nao tem relacao com seu nicho
- Podem prejudicar sua imagem
- Exigem muito tempo de producao
- Ja estao saturadas

## Adapte 3 Trends

### Prompt: Adaptador de Trends

```
Adapte esta trend para o meu nicho:

TREND:
[descreva a trend que voce viu]

MEU CONTEXTO:
- Nicho: [seu nicho]
- Publico: [seu ICP]
- Tom: [profissional/casual/bem-humorado]
- Plataforma: [Instagram/TikTok/LinkedIn]

GERE:
1. Versao adaptada da trend
2. Roteiro/texto completo
3. Dicas de execucao
4. Hashtags relevantes
5. Melhor horario para postar
```

### Prompt Bonus: Trend Original

```
Crie uma trend original para meu nicho:

NICHO: [seu nicho]
PUBLICO: [seu ICP]

A trend deve:

Gere 3 opcoes com roteiro completo.
```
