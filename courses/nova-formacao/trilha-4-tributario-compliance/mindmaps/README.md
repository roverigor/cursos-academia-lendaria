# Mind Maps - Trilha 4: Tributario & Compliance

Arquivos para importar no Whimsical e criar mapas mentais.

## Como Usar no Whimsical

1. Abra o Whimsical e crie um novo Mind Map
2. Copie o conteudo do arquivo desejado
3. Selecione o no central e cole (Ctrl+V / Cmd+V)
4. O Whimsical converte automaticamente a hierarquia

## Formato

- Cada linha = 1 no
- Indentacao por TAB = hierarquia (filho)
- Sem markdown especial (Whimsical interpreta texto puro)

## Metricas da Trilha

- **Duracao Total:** ~7.5 horas (450 min)
- **Aulas:** 40

- **Quick Wins:** 5
