## ARTEFATO DA AULA: Plano de Manutencao Anual

### O Que Voce Vai Criar
Checklist de manutencao anual para garantir que os ganhos da trilha sejam mantidos e otimizados ano apos ano.

### Por Que Isso Importa
Otimizacao tributaria nao e evento unico:
- Leis mudam
- Faturamento muda
- Cenario muda
- Revisao anual mantem ganhos

**Linha do DRE:** Economia Recorrente (ganhos que se repetem todo ano)

### Quando Usar
- Todo Janeiro
- Quando faturamento mudar significativamente
- Quando legislacao mudar
- Antes de expansao/contratacao

### Como Criar
1. Defina data de revisao anual
2. Liste itens a revisar
3. Configure lembrete no calendario

### Template

```
PLANO DE MANUTENCAO ANUAL

Data de revisao: ___/01/______ (todo Janeiro)

CHECKLIST ANUAL:

TRIBUTARIO:
- [ ] Atualizar Mapa Tributario
- [ ] Comparar carga atual vs ano anterior
- [ ] Revisar regime (faturamento mudou?)
- [ ] Checar Fator R se Simples
- [ ] Avaliar se estrategias ainda valem

COMPLIANCE:
- [ ] Atualizar Checklist de Compliance
- [ ] Verificar mudancas na LGPD
- [ ] Renovar contratos se necessario
- [ ] Treinar novos funcionarios

EXECUCAO:
- [ ] Medir economia realizada no ano
- [ ] Definir metas do proximo ano
- [ ] Agendar reuniao com contador

RESULTADO DO ANO ANTERIOR:
- Economia realizada: R$ ___
- Score compliance: ___%
- Metas atingidas: ___/___

METAS PARA ESTE ANO:
- Economia projetada: R$ ___
- Score compliance meta: ___%
- Acoes prioritarias: _______________

LEMBRETE CONFIGURADO: [ ] Sim
```
