## O Que É a Reforma Tributária?

### Resumo em 1 Minuto

A Reforma Tributária (EC 132/2023) vai **simplificar** o sistema brasileiro:

| ANTES (Atual) | DEPOIS (2033) |
|---------------|---------------|
| PIS | → |
| Cofins | → | **CBS** (Federal) |
| IPI | → |
| ICMS | → | **IBS** (Estados/Municípios) |
| ISS | → |

**5 tributos viram 2** (+ Imposto Seletivo para produtos nocivos)

## Linha do Tempo: O Que Acontece Quando

```
2025 ──────── Preparação (você está aqui)
    │

    │         (impacto mínimo)

    │
```

| **2025** | Entender, mapear, preparar |
| **2026** | Adaptar sistemas, baixo impacto |
| **2027-2028** | Maior complexidade, atenção redobrada |
| **2029+** | Simplificação gradual |
| **2033** | Sistema mais simples |

## As 3 Grandes Mudanças

### 1. Imposto no Destino (Não na Origem)

**Antes:** Imposto cobrado onde a empresa está
**Depois:** Imposto cobrado onde o cliente está

**Impacto:** Fim da "guerra fiscal" entre estados. Localização por eficiência, não por incentivo.

---

### 2. Crédito Pleno (Não-Cumulatividade)

**Antes:** Crédito parcial, muitas exceções
**Depois:** Crédito de TUDO que compra para o negócio

**Impacto:** Beneficia quem tem cadeia longa de fornecedores. Documentação fiscal vira ouro.

---

### 3. Transparência Total

**Antes:** Imposto "escondido" no preço
**Depois:** Imposto destacado na nota fiscal

**Impacto:** Consumidor vê quanto paga de imposto. Pode usar como argumento de marketing.

## Impacto Por Tipo de Negócio

### Serviços (Atenção!)

```
ISS atual: ~2% a 5%
IBS futuro: ~13% (estimativa)

⚠️ AUMENTO PROVÁVEL DE CARGA
```

**O que fazer:**
- Avaliar repasse ao preço
- Buscar eficiência operacional
- Revisar se Simples ainda compensa

---

### Comércio

```
ICMS atual: 7% a 18% (varia por estado)
IBS futuro: ~13% (estimativa, uniforme)

➡️ NEUTRO A POSITIVO (depende do estado atual)
```

**O que fazer:**
- Comparar carga atual vs futura
- Aproveitar crédito pleno de compras
- Manter documentação fiscal em dia

---

### Indústria

```
Carga atual: Alta, com cumulatividade
Carga futura: Menor, com crédito pleno

✅ REDUÇÃO PROVÁVEL DE CARGA
```

**O que fazer:**
- Aproveitar crédito de todos os insumos
- Investir em sistema de gestão integrado
- Documentar todas as compras

## E o Simples Nacional?

### Boa Notícia: Simples CONTINUA

| Aspecto | Situação |
|---------|----------|
| Regime diferenciado | ✅ Mantido |
| Limite R$ 4,8M | ✅ Mantido |
| Guia única (DAS) | ✅ Mantida |
| Alíquotas progressivas | ✅ Mantidas |

### Atenção: Crédito Para Clientes

**Problema potencial:**
- Quem compra de empresa do Simples terá crédito **limitado**
- No regime normal, o crédito é **integral**

**Impacto:**
- Se você vende para **consumidor final**: Simples continua vantajoso
- Se você vende para **outras empresas (B2B)**: Avaliar sair do Simples

### Nova Opção: Tributação Normal no Simples

Empresas do Simples poderão optar por apurar CBS/IBS no regime normal para dar crédito integral aos clientes.

**Quando considerar:**
- Maioria dos clientes são empresas
- Clientes reclamam de não ter crédito
- Margem permite absorver diferença

## Mudança no Imposto de Renda (2026)

### Nova Isenção: R$ 5.000/mês

A partir de **janeiro de 2026**:

| Renda Mensal | IRPF |
|--------------|------|
| Até R$ 5.000 | **ISENTO** |
| Acima de R$ 5.000 | Tabela progressiva |

### Impacto na Estratégia Pró-labore vs Lucros

**Estratégia clássica (até 2025):**
- Pró-labore mínimo (paga menos IRPF/INSS)
- Restante em lucros (isento)

**Nova realidade (2026+):**
- Pró-labore até R$ 5.000 = isento de IRPF
- Estratégia precisa ser recalculada
- Converse com seu contador!
