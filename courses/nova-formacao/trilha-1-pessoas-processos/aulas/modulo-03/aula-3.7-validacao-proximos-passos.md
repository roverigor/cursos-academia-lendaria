## ARTEFATO DA AULA: SOP Validado + Feedback de Teste

### O Que Você Vai Criar
SOP Inteligente revisado pela IA, ajustado após Teste do Novato, com feedback documentado e pronto para uso em produção.

### Por Que Isso Importa
SOP validado = SISTEMA confiável:
- IA revisou estrutura e completude
- Pessoa real testou na prática
- Ajustes documentados
- Pronto para delegar

**Linha do DRE:** ROI do SOP (horas economizadas x pessoas que usarão x meses de uso)

### Quando Usar
- Após criar e testar qualquer SOP
- Como entregável para validação de módulo
- Ao oficializar processo na equipe
- Como base para treinamento

### Como Criar
1. Rode o prompt de revisão com IA
2. Aplique as sugestões
3. Documente o feedback do Teste do Novato
4. Faça ajustes finais
