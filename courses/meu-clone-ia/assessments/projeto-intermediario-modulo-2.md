# Projeto Intermediário - Construa Seu Clone (MVA)

## Objetivo do Projeto

Ao final deste projeto, você terá um clone funcional (MVA - Minimum Viable Architecture) que:
- Pensa como você (reasoning documentado)
- Escreve como você (fidelidade de voz 80%+)
- Funciona em cenários comerciais reais (aprovado em 2/3 testes)

**Tempo estimado:** 2-3 horas (não 40 horas - foco em MVA)

---

## Contexto

Este não é um projeto teórico. Você vai construir seu clone DE VERDADE e testar em situações reais.

Se você seguir as instruções e completar os 4 deliverables, você sai daqui com um clone operacional que pode economizar 5-10h/semana ou gerar R$2-10k/mês.

**Não pule etapas. Cada deliverable é crítico.**

## FAQ do Projeto

**P: Posso usar ferramentas de IA para documentar Knowledge/Voice/Reasoning?**

R: Sim, mas o conteúdo tem que ser SEU. Use IA para acelerar extração (ex: transcrever áudio de você falando), mas não para gerar conhecimento genérico. Clone genérico vale R$0.

**P: Não tenho leads reais para testar. Posso simular?**

R: Não. Use leads passados (emails antigos), perguntas de discovery calls, ou DMs reais. Se realmente não tem, faça post público pedindo 3 perguntas e teste com essas.

**P: Meu clone falhou nos 3 testes. O que fazer?**

R: Identifique gaps específicos e itere:
- Falhou em qualificação → adicione 10 exemplos de como VOCÊ qualifica
- Falhou em voz → adicione 5 amostras do seu estilo + revise vocabulário
- Falhou em reasoning → documente 3 exemplos de como VOCÊ raciocina nessa situação

Depois reteste. Não passe para Módulo 3 sem aprovar em 2/3 cenários.

**P: Quanto tempo leva para completar o projeto?**

R: 2-3 horas se você seguir as instruções. Não perfeccione - foque em MVA. Clone 80% bom gerando resultado > clone 100% perfeito nunca lançado.

**P: Posso trabalhar em dupla?**

R: Não. Clone é pessoal - documenta SEU conhecimento, SUAS voice, SEU raciocínio. Não dá para terceirizar isso.
