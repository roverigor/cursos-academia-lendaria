Instalacao + Configuracao
	Instalacao por OS
		Windows
			Git for Windows
			git-scm.com/download/win
			Escolher VS Code como editor
			Branch inicial: main
			Usar Git Bash
		macOS
			Homebrew: brew install git
			Xcode Command Line Tools
			Verificar versao
		Linux (Ubuntu/Debian)
			sudo apt update
			sudo apt install git
		Linux (Fedora)
			sudo dnf install git
		Verificacao
			git --version
			Versao 2.30+ recomendada
	Configuracao Essencial
		Nome e Email
			git config --global user.name
			git config --global user.email
			Usar email do GitHub
			Aparece em todos commits
		Email Privado GitHub
			Settings â†’ Emails
			Keep email private
			username@users.noreply.github.com
		Editor Padrao
			VS Code
				git config --global core.editor "code --wait"
			Nano
				Simples para iniciantes
			Vim
				Se sabe sair dele
		Branch Padrao
			git config --global init.defaultBranch main
			Padrao atual da industria
		Cores no Terminal
			git config --global color.ui auto
			Facilita leitura
	Verificando Config
		Listar todas
			git config --list
		Config especifica
			git config user.name
		Onde fica salvo
			Global: ~/.gitconfig
			Local: .git/config
	Configuracoes Extras
		Pull com rebase
			git config --global pull.rebase true
			Evita merge commits
		Credential helper
			Windows: wincred
			macOS: osxkeychain
			Linux: cache
