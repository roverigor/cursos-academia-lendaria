# GitHub Essencial para Devs - Course Structure

## Overview

Complete course on Git and GitHub essentials for professional developers.

**Duration:** 2h30 (150 minutes)  
**Level:** Beginner  
**Language:** Portuguese (PT-BR)  
**Lessons:** 15 practical lessons

---

## Course Structure

### Module 1: Setup Moderno (25 min)
1. **1.1 - Git vs GitHub** (5 min)
   - Difference between Git and GitHub
   - Why companies require both

2. **1.2 - Instalação + Config** (10 min)
   - Git installation
   - Configuration (name, email, editor)

3. **1.3 - SSH com Ed25519** (10 min)
   - Modern SSH setup
   - GitHub authentication

---

### Module 2: Seu Primeiro Repo Real (50 min)
4. **2.1 - Criando repo certo** (10 min)
   - Professional repository setup
   - README, .gitignore, License

5. **2.2 - O ciclo básico** (15 min)
   - clone, status, add, commit workflow
   - Mental model of Git

6. **2.3 - Push e Pull** (10 min)
   - Synchronizing local ↔ remote
   - When to use each

7. **2.4 - Commits que comunicam** (10 min)
   - Clear commit messages
   - Atomic commits

8. **2.5 - .gitignore essencial** (5 min)
   - What never to version
   - Templates by language

---

### Module 3: Colaboração Profissional (55 min)
9. **3.1 - Branches** (12 min)
   - Why and how to use branches
   - Naming conventions

10. **3.2 - Pull Request** (15 min)
    - Professional PR creation
    - Description templates

11. **3.3 - Code Review** (13 min)
    - Reviewing colleagues' code
    - Constructive feedback

12. **3.4 - Merge + Conflitos** (15 min)
    - Merging branches
    - Resolving conflicts

---

### Module 4: Sobrevivência + Próximos Passos (20 min)
13. **4.1 - Desfazendo erros** (10 min)
    - reset, revert, stash
    - When to use each

14. **4.2 - Perfil profissional** (5 min)
    - Profile README
    - Pinned repositories

15. **4.3 - O que vem depois** (5 min)
    - GitHub Actions, Copilot
    - Open source contributions
    - Next learning steps

1. **Modern setup** - SSH Ed25519, branch `main`, 2025 best practices
2. **Portfolio-first** - Every repo is real, not throwaway
3. **Career-focused** - Direct connection to job market
4. **Team workflow** - Simulates real company processes
5. **No-fluff approach** - Short lessons, actionable content
6. **Real examples** - Practical code and scenarios

## Learning Outcomes

After completing this course, students will be able to:

1. Configure Git and GitHub with modern SSH Ed25519 setup
2. Manage repositories with versioning and best practices
3. Work in teams using branches, PRs, and code review
4. Resolve merge conflicts confidently
5. Undo mistakes using reset, revert, and stash
6. Build a professional GitHub profile
7. Know next steps for continuous improvement
