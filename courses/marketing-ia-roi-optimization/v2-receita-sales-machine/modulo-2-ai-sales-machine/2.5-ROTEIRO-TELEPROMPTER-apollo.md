## [12:30 - 14:30] REDIS CACHE

**[SCREENCAST: Redis integration]**

Problema: Enriquecer mesmo lead 2x = desperdício.

Solução: Cache Redis.

```javascript
// Check cache first
const cached = await redis.get(`apollo:${email}`);
if (cached) return cached; // 0 créditos

// Se não tem, enriquecer
const enriched = await apollo.enrich(email); // 1 crédito
await redis.set(`apollo:${email}`, enriched, 2592000); // 30 dias
```
