## [03:00-07:00] SETUP A/B TEST

**[SCREENCAST: n8n]**

```javascript
const variant = Math.random() < 0.5 ? 'A' : 'B';

const subjects = {
  A: '{{ company }} aumentar vendas 800%?',
  B: '{{ firstName }}, pergunta sobre {{ company }}'
};

sendEmail({
  subject: subjects[variant],
  ab_variant: variant
});
```

**[MOSTRA tracking]**

```sql
INSERT INTO ab_test_results (variant, opened, clicked)
VALUES ({{ variant }}, {{ opened }}, {{ clicked }})
```

## [07:00-10:00] STATISTICAL SIGNIFICANCE

**[MOSTRA cálculo]**

Variant A: 45 opens / 100 sends = 45%
Variant B: 32 opens / 100 sends = 32%

Chi-square test: p-value = 0.03

p < 0.05 = statistically significant!

Winner: Variant A.

**[AUTO-DEPLOY]**
