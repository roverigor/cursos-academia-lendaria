### PARTE 1: ANATOMIA DA INTERFACE (0:45 - 3:00)

**[CENA: Screencast - interface completa do NotebookLM]**

A interface tem 3 áreas principais:

#### ÁREA 1: PAINEL DE SOURCES (Esquerda)
- Lista de todas as sources carregadas
- Clique para selecionar/desselecionar
- Checkbox para incluir/excluir do contexto

**[DESTACAR NA TELA]**

**DICA IMPORTANTE:** Você pode desmarcar sources para focar a conversa apenas em documentos específicos. Se você tem 20 sources mas quer perguntar sobre apenas 2, desmarque as outras.

---

#### ÁREA 2: CHAT PRINCIPAL (Centro)
- Onde você faz perguntas
- Histórico de conversas
- Respostas com citações

**[DESTACAR NA TELA]**

**DICA:** Clique nas citações (números em azul) para ver exatamente de onde veio a informação.

---

#### ÁREA 3: STUDIO (Direita)
- Gera outputs automáticos
- Audio Overview, FAQ, Study Guide, etc.
- Área de notas e salvos

**[DESTACAR NA TELA]**

Vamos explorar o Studio em detalhes nos próximos módulos.

---

### PARTE 2: GERENCIANDO SOURCES (3:00 - 5:30)

**[CENA: Screencast - painel de sources]**

#### 2.1 - Renomeando Sources

Por padrão, sources têm nomes automáticos (nome do arquivo, título do site).

**Para renomear:**
1. Clique nos 3 pontos ao lado da source
2. Selecione "Rename"
3. Digite nome descritivo

**[DEMONSTRAR]**

**Convenção sugerida:**
```
[TIPO] Nome Descritivo - Autor/Fonte

Exemplos:
[PDF] Relatório Vendas Q3 - Equipe Comercial
[YT] Entrevista CEO Apple - CNBC
[WEB] Artigo Precificação SaaS - Indie Hackers
```

---

#### 2.2 - Deletando Sources

**Para deletar:**
1. Clique nos 3 pontos ao lado da source
2. Selecione "Delete"
3. Confirme

**[DEMONSTRAR]**

**ATENÇÃO:** Deletar é permanente. Se for uma source importante, pense duas vezes.

---

#### 2.3 - Visualizando Source Original

**Para ver o conteúdo original:**
1. Clique na source
2. Painel lateral abre com preview
3. Role para navegar pelo documento

**[DEMONSTRAR]**

**USO:** Quando o NotebookLM cita algo, você pode verificar o contexto original.

---

#### TÉCNICA 2: Seleção Estratégica de Sources

Nem sempre você quer que o NotebookLM considere todas as sources.

**Cenário:** Notebook com 30 sources, mas você quer perguntar sobre um tema específico.

**Solução:**
1. Desmarque todas (clique no checkbox geral)
2. Marque apenas as 3-5 sources relevantes
3. Faça sua pergunta
4. Resposta mais focada e precisa

**[DEMONSTRAR]**

---

#### TÉCNICA 3: Limpeza Periódica

**A cada 2-4 semanas:**
1. Revise suas sources
2. Delete as que não usa mais
3. Reorganize nomes se necessário
4. Considere criar novos notebooks para novos projetos

**REGRA:** Se você não consultou uma source em 30 dias, provavelmente pode deletar.

---

### PARTE 4: ATALHOS E PRODUTIVIDADE (8:00 - 9:30)

**[CENA: Screencast - demonstração de atalhos]**

#### Atalhos Úteis:

| Ação | Como Fazer |
|------|------------|
| Nova pergunta | Clique no campo de chat |
| Ver citação original | Clique no número azul |
| Copiar resposta | Botão "Copy" abaixo da resposta |
| Selecionar todas sources | Checkbox no topo da lista |
| Deselecionar todas | Checkbox no topo (segunda vez) |

---

#### Dica de Produtividade: Perguntas Salvas

Se você faz perguntas similares frequentemente, mantenha um arquivo de texto com seus prompts favoritos.

**Exemplo - Meus Prompts Padrão:**
```
EXTRAÇÃO:
"Liste os 5 principais insights deste documento em bullet points"

SÍNTESE:
"Resuma em 3 parágrafos: contexto, principais pontos, conclusões"

APLICAÇÃO:
"Crie um plano de ação de 5 passos baseado nas recomendações"
```
