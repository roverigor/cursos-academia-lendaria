### 3.2. Objetivos de Aprendizagem

**Ao final do curso, o aluno sera capaz de:**

```
1. Instalar Git e configurar conexao segura com GitHub via SSH Ed25519

2. Criar repositorio com estrutura organizada para automacoes, SOPs e prompts

3. Executar ciclo completo: adicionar arquivos, salvar versao, sincronizar com nuvem

4. Escrever mensagens de commit claras que explicam o que foi feito

5. Configurar gitignore para proteger senhas, chaves e informacoes sensiveis

6. Criar branches para testar mudancas sem afetar versao principal

7. Abrir Pull Request com descricao clara do que mudou e por que

8. Revisar trabalho de outras pessoas com comentarios construtivos

9. Resolver conflitos quando duas pessoas editam o mesmo arquivo

10. Recuperar versoes anteriores usando restore, stash, reset ou revert
```
