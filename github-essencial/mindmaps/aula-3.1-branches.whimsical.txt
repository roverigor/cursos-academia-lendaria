Versoes Paralelas (Branches)
	O Que Sao Branches
		Branch = ramo em ingles
		Analogia
			main = tronco da arvore
			Versao estavel que funciona
			branch = galho que sai do tronco
			Versao para experimentar
		Isolamento
			Mudancas no galho nao afetam o tronco
			Se funcionar: junta de volta (merge)
			Se nao funcionar: apaga o galho
	Exemplo Pratico
		Manual de processos funcionando bem
		Quer reorganizar completamente
		Em vez de editar direto
			Cria branch
			Faz todas as mudancas la
			Se ficar melhor: integra
			Se ficar pior: volta para original
	Quando Usar Branches
		Testar mudanca grande
			Reorganizar documento
			Mudar estrutura de pastas
			Nova abordagem
		Pedir ajuda para alguem
			Freelancer ou membro da equipe
			Trabalha em branch separada
			Voce revisa antes de integrar
		Manter main sempre funcionando
			Enquanto trabalha em mudancas
			Versao principal continua estavel
		Regra simples
			Mudanca pequena: edite direto
			Mudanca maior: crie branch
	Nomes de Branches
		Bons nomes
			nova-tabela-precos
			reorganiza-manual
			adiciona-faq
			corrige-processos-venda
			Explicam o que esta fazendo
		Nomes ruins
			teste (teste de que?)
			branch1 (o que tem nessa branch?)
			nova (nova o que?)
	Comandos Basicos
		Ver branches
			git branch
			Asterisco indica qual voce esta
		Criar branch e entrar
			git checkout -b nome-da-branch
			-b = criar nova branch
		Voltar para main
			git checkout main
		Enviar branch para GitHub
			git push -u origin nome-da-branch
			-u origin configura conexao
			Depois: apenas git push
		Apagar branch
			git branch -d nome-da-branch
			Usar depois de integrar
	Exercicio
		Ver branches atuais
			git branch
			Provavelmente so main
		Criar nova branch
			git checkout -b teste-nova-secao
		Confirmar que mudou
			git branch
			Asterisco em teste-nova-secao
		Fazer mudanca
			Editar README.md
			Adicionar nova secao
		Salvar versao
			git add .
			git commit -m "Adiciona nova secao para teste"
		Voltar para main
			git checkout main
		Verificar isolamento
			Abrir README.md
			Secao nao esta la!
		Voltar para branch
			git checkout teste-nova-secao
			Secao aparece de novo
		Enviar para GitHub
			git push -u origin teste-nova-secao
			Ver branch no GitHub
	Resumo
		Branch = versao paralela do projeto
		Use para mudancas grandes ou testes
		git checkout -b nome (criar e entrar)
		git checkout main (voltar)
		git push -u origin nome (enviar)
