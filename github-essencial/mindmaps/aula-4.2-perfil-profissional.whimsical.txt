Organizando Seu Hub
	Por Que Organizar
		Acesso rapido
			Abre GitHub e ve projetos principais
		Clareza
			Sabe o que cada projeto contem
		Compartilhamento
			Mostrar para parceiros ou equipe
		Investir minutos
			Economiza tempo toda vez que acessa
	Configuracao Basica
		Foto
			Sua foto ou logo da empresa
			Algo que identifique voce
		Bio (descricao curta)
			Uma frase do que voce faz
			Exemplos
				Empresario | Organizando projetos com GitHub
				Dono de agencia | Documentos e processos organizados
		Links uteis
			Site, LinkedIn
			Forma de contato
			Fica visivel no perfil
	README de Perfil
		O que e
			Apresentacao no topo do perfil
			Funcao especial do GitHub
		Como criar
			Criar repositorio com nome do seu usuario
			Se voce e joaosilva
			Crie repositorio joaosilva
			Marque Add a README file
		GitHub avisa
			This is a special repository
			Confirma que fez certo
		Conteudo sugerido
			# Ola! Eu sou [Seu Nome]
			## Sobre
			Empresario focado em [sua area]
			Uso GitHub para organizar projetos
			## Meus Projetos Principais
			- Projeto 1: descricao
			- Projeto 2: descricao
			## Contato
			Site, Email
		Aparece automaticamente
			Quando alguem visita seu perfil
	Projetos em Destaque
		O que e
			Ate 6 projetos no topo do perfil
			Primeiros que aparecem
		Como fazer
			Ir no perfil
			Clicar em Customize your pins
			Selecionar ate 6 repositorios
			Clicar em Save pins
		O que destacar
			Projetos que acessa com frequencia
			Documentos importantes
			Manuais e processos
			Projetos compartilhados com equipe
		O que NAO destacar
			Testes que nao usa mais
			Repositorios com nomes confusos
			Projetos abandonados
	Descricoes Claras
		Cada repositorio pode ter descricao
		Em vez de: docs
			Nao explica nada
		Coloque: Manuais de processos da equipe de vendas
			Ajuda a encontrar rapidamente
	Exercicio
		Abrir perfil GitHub
		Clicar em Edit profile
		Adicionar bio curta
		Salvar
		Ir em Customize your pins
		Selecionar projetos importantes
		Opcional: criar README de perfil
	Resumo
		Bio clara que explica quem voce e
		README de perfil para apresentacao visual
		Ate 6 projetos em destaque
		Descricoes claras em cada repositorio
		Menos tempo procurando, mais tempo trabalhando
