Instalacao Passo a Passo
	Por Que Configurar Nome e Email
		Git registra quem fez cada mudanca
		Voce ve no historico
		Outras pessoas sabem quem fez
		Configuracao feita uma vez so
	Verificar Se Ja Tem Git
		Abrir terminal
			Windows: cmd ou Prompt de Comando
			Mac: Terminal (pasta Utilitarios)
		Comando
			git --version
		Resultado
			Versao aparece: ja tem Git
			Erro: precisa instalar
	Instalacao por Sistema
		Windows
			Acesse git-scm.com
			Download for Windows
			Execute o instalador
			Opcoes importantes
				Branch padrao: main
				Terminal: Git Bash
			Depois: abrir Git Bash
		Mac
			Forma simples
				Terminal: git --version
				Mac pergunta se quer instalar
				Aceite Command Line Tools
				Aguarde 2-3 minutos
			Forma alternativa (Homebrew)
				brew install git
		Linux
			Ubuntu/Debian
				sudo apt update && sudo apt install git
			Fedora
				sudo dnf install git
	Configuracao
		Nome
			git config --global user.name "Seu Nome"
		Email
			git config --global user.email "seu@email.com"
			Usar mesmo email do GitHub
		Branch padrao
			git config --global init.defaultBranch main
			Padrao que todo mundo usa
	Verificar Configuracao
		Comando
			git config --list
		Deve mostrar
			user.name=Seu Nome
			user.email=seu@email.com
			init.defaultbranch=main
	Checklist
		Git instalado (git --version funciona)
		Nome configurado
		Email configurado (mesmo do GitHub)
		Branch padrao: main
