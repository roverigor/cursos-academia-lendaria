Voltando no Tempo (Desfazendo Erros)
	Erros Sao Normais
		Todo mundo erra
		Git foi feito para permitir voltar atras
		Nao precisa ter medo
	git restore (Descartar Mudancas)
		Quando usar
			Editou um arquivo
			Quer voltar como estava antes
		Arquivo especifico
			git restore nome-do-arquivo.txt
			Arquivo volta ao estado anterior
		Tirar do staging
			git restore --staged nome-do-arquivo.txt
			Remove da preparacao
			Mantem suas mudancas no arquivo
		Descartar TODAS mudancas
			git restore .
			CUIDADO: mudancas sao perdidas
	git stash (Guardar Temporariamente)
		Quando usar
			Precisa mudar de branch urgente
			Mudancas nao estao prontas para salvar
		Guardar
			git stash
			Mudancas vao para uma "gaveta"
			Pasta de trabalho fica limpa
		Recuperar
			git stash pop
			Mudancas voltam
			Gaveta e esvaziada
		Com descricao
			git stash save "Trabalhando na tabela de precos"
		Ver o que esta guardado
			git stash list
	git reset (Desfazer Versao Nao Enviada)
		Quando usar
			Salvou versao (commit)
			Ainda NAO enviou para GitHub
		Manter mudancas
			git reset --soft HEAD~1
			Desfaz commit
			Mudancas continuam no arquivo
		Descartar tudo
			git reset --hard HEAD~1
			CUIDADO: apaga tudo
			Usar so quando tem certeza
		HEAD~1
			Uma versao antes da atual
			HEAD~2 = duas versoes
		IMPORTANTE
			So usar se NAO enviou para GitHub
	git revert (Desfazer Versao Ja Enviada)
		Quando usar
			Salvou E enviou para GitHub
			Precisa desfazer
		Comando
			git revert HEAD
		O que acontece
			Cria NOVA versao que desfaz a anterior
			Historico e preservado
			Todo mundo pode ver o que aconteceu
		Depois do revert
			git push
			Envia a correcao
	git commit --amend (Corrigir Ultimo Commit)
		Quando usar
			Acabou de salvar
			Mensagem ficou errada
			Esqueceu arquivo
		Corrigir mensagem
			git commit --amend -m "Nova mensagem corrigida"
		Adicionar arquivo esquecido
			git add arquivo-esquecido.txt
			git commit --amend --no-edit
			--no-edit mantem mensagem atual
		IMPORTANTE
			So usar se NAO enviou para GitHub
	Quando Usar Cada Comando
		Mudanca NAO salva ainda?
			git restore: descartar
			git stash: guardar temporariamente
		Versao salva mas NAO enviada?
			git reset --soft: desfazer (manter mudancas)
			git commit --amend: corrigir mensagem
		Versao ja enviada para GitHub?
			git revert: criar versao que desfaz
		Regra principal
			Ja enviou para GitHub: use revert
			Nao enviou: pode usar reset
	Exercicio
		Teste restore
			Editar README.md
			git status (mostra modificado)
			git restore README.md
			Arquivo voltou ao estado anterior
		Teste stash
			Editar README.md
			git stash
			git status (pasta limpa)
			git stash pop
			Mudanca voltou
	Resumo
		git restore: descartar mudancas nao salvas
		git stash: guardar temporariamente
		git reset --soft: desfazer versao nao enviada
		git revert: desfazer versao ja enviada
		git commit --amend: corrigir ultima versao
		Regra de ouro: ja enviou para GitHub, use revert
