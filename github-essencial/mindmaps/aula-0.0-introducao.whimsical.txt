Por Que Voce Precisa de GitHub
	O Problema Sem GitHub
		Versoes confusas
			proposta_v1, v2, final, final_revisado
			Qual era a versao aprovada?
		Mudancas perdidas
			Editou e ficou pior
			Nao lembra como era antes
		Arquivos vulneraveis
			Computador trava
			HD queima
			Notebook roubado
		Colaboracao confusa
			Arquivos por email/WhatsApp
			Nao sabe o que mudou
	GitHub: A Solucao
		Google Drive mais inteligente
			Guarda arquivos
			Guarda historico de mudancas
		Comparativo
			Google Drive
				Guarda arquivos
				Versoes confusas
				Backup quando lembra
			GitHub
				Guarda arquivos + historico
				Cada mudanca registrada
				Backup automatico
		Beneficios
			Nunca perde uma versao
			Volta para qualquer momento
			Colaboracao documentada
			Menos retrabalho
	Exemplos do Dia a Dia
		Documento importante
			Proposta comercial
			Cada versao salva automatico
			Volta em segundos
		Instrucoes de trabalho
			SOP de atendimento
			Ve quando alterou
			Pode voltar atras
		Trabalho com outras pessoas
			Entrega pelo GitHub
			Ve o que foi feito
			Comenta, aprova ou ajusta
		Seguranca
			Computador morre
			Acessa de outro em minutos
	Estrutura do Curso
		Modulo 1
			Instalacao e configuracao
		Modulo 2
			Salvar versoes e sincronizar
		Modulo 3
			Colaborar com outras pessoas
		Modulo 4
			Recuperar versoes e organizar
