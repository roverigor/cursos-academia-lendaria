Juntando Versoes (Merge e Conflitos)
	Como Conflitos Acontecem
		Duas pessoas editam
			Mesma parte do mesmo arquivo
			De formas diferentes
		Exemplo
			Voce muda telefone para 99999-1111
			Outra pessoa muda para 99999-2222
			Mesma linha do documento
		Ao tentar juntar (merge)
			Git nao sabe qual manter
			Voce precisa decidir
		Importante
			Conflito so na mesma parte
			Partes diferentes: Git junta automatico
	Identificando Conflitos
		Mensagem ao tentar merge
			CONFLICT in documento.txt
			Automatic merge failed
		Nao e erro grave
			Git pedindo sua decisao
		git status
			Mostra arquivos em conflito
			both modified: documento.txt
	Os Marcadores de Conflito
		Quando abrir arquivo
			<<<<<<< HEAD
			(11) 99999-1111
			=======
			(11) 99999-2222
			>>>>>>> outra-branch
		Significado
			<<<<<<< HEAD = inicio, sua versao
			======= = separador
			>>>>>>> branch = fim, outra versao
		Seu trabalho
			Escolher qual versao manter
			Ou combinar as duas
			Apagar todos os marcadores
	Resolvendo Passo a Passo
		Passo 1: Identificar arquivos
			git status
			Ver quais estao em both modified
		Passo 2: Abrir arquivo e escolher
			Opcao 1: Manter sua versao
				(11) 99999-1111
			Opcao 2: Manter outra versao
				(11) 99999-2222
			Opcao 3: Combinar as duas
				Telefone matriz: (11) 99999-1111
				Telefone filial: (11) 99999-2222
		Passo 3: Apagar marcadores
			Remover <<<<<< ====== >>>>>>
			Todas as linhas de marcador
		Passo 4: Salvar arquivo
		Passo 5: Marcar como resolvido
			git add documento.txt
		Passo 6: Finalizar merge
			git commit
			Git sugere mensagem padrao
		Passo 7: Enviar
			git push
		Conflito resolvido!
	Se Quiser Desistir
		Comando
			git merge --abort
		O que faz
			Cancela o merge
			Volta tudo como estava antes
			Nenhum dano
		Quando usar
			Conflito muito complicado
			Quer pensar melhor
			Fez merge da branch errada
			Quer pedir ajuda antes
	Exercicio
		Criar conflito de proposito
			Duas branches mudando mesma linha
		Criar branch 1
			git checkout main
			git checkout -b versao-matriz
			Editar README.md: Telefone 99999-1111
			git add . && git commit -m "Telefone matriz"
		Criar branch 2
			git checkout main
			git checkout -b versao-filial
			Editar README.md: Telefone 99999-2222
			git add . && git commit -m "Telefone filial"
		Causar conflito
			git checkout main
			git merge versao-matriz (funciona)
			git merge versao-filial (CONFLITO!)
		Resolver
			Abrir README.md (ver marcadores)
			Editar para combinar ambos
			Remover marcadores
			git add README.md
			git commit
			git push
	Resumo
		Conflitos: duas pessoas editam mesma parte
		Marcadores: <<<<<<< ======= >>>>>>>
		Resolver: escolher versao ou combinar
		Apagar todos os marcadores
		git merge --abort para voltar atras
		Modulo 3 completo!
