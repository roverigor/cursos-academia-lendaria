Revisando Trabalho de Outros
	O Que e Revisao
		Quando alguem abre Pull Request
			Voce pode revisar
		O que voce pode fazer
			Ver o que foi modificado
			Fazer comentarios
			Aprovar ou pedir ajustes
		Mentalidade
			Nao e para criticar
			E para garantir que esta correto
			Conversa para melhorar o trabalho
			Foque nas mudancas, nao na pessoa
	Acessando a Revisao
		Abrir o Pull Request
		Clicar na aba Files changed
		Ver mudancas
			Linhas em verde = adicionado
			Linhas em vermelho = removido
		Comentar em linha especifica
			Passar mouse no numero da linha
			Clicar no + azul
			Escrever comentario
			Add single comment ou Start a review
	Tipos de Comentarios
		Perguntas
			Quando algo nao esta claro
			Exemplo: Por que essa secao foi movida?
		Sugestoes
			Quando tem ideia de melhoria
			Exemplo: Que tal adicionar um titulo aqui?
		Problemas
			Quando encontra algo errado
			Exemplo: O telefone aqui esta desatualizado
		Elogios
			Quando algo ficou muito bem feito
			Exemplo: Ficou otima essa organizacao!
			Importantes: reconhecer bom trabalho
	Como Escrever Bom Feedback
		Use perguntas em vez de ordens
			Em vez de: Mude isso
			Escreva: Que tal mudar isso?
		Seja especifico
			Em vez de: Isso nao esta bom
			Escreva: Essa secao ficou confusa porque mistura dois assuntos
		Explique o motivo
			Em vez de: Adicione uma introducao
			Escreva: Sugiro adicionar introducao para o leitor saber o que vai encontrar
		Quem recebe entende melhor e aprende
	Finalizando a Revisao
		Clicar em Review changes
		Tres opcoes
			Approve
				Esta tudo certo, pode integrar
			Request changes
				Tem problemas que precisam corrigir
			Comment
				Apenas feedback, sem aprovar nem bloquear
		Quando usar Request changes
			Apenas quando algo precisa ser corrigido
		Para sugestoes menores
			Use Approve
			Escreva: Aprovado! Deixei algumas sugestoes
	Exercicio
		Abrir um Pull Request
			Pode ser o seu proprio
		Ir em Files changed
		Fazer comentarios de teste
			Uma pergunta
			Uma sugestao
			Um elogio
		Clicar em Review changes
			Escrever: Revisado! As mudancas estao boas
			Selecionar Approve
			Enviar
	Resumo
		Revisao melhora o trabalho, nao e critica
		Comentarios especificos com motivo
		Perguntas em vez de ordens
		Reconheca o que esta bom com elogios
		Aprove quando pronto, peca mudancas so se necessario
