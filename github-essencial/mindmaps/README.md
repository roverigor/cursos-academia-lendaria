# Mind Maps - GitHub Essencial para Devs

This directory contains 17 comprehensive mind maps in Whimsical format for the "GitHub Essencial para Devs" course.

## Whimsical Import Format

These files use the Whimsical text import format:
- First line = central topic
- TAB character = child node (hierarchy)
- Simple text lines
- No special formatting needed

## How to Import

1. Go to [Whimsical](https://whimsical.com)
2. Create new Mind Map
3. Click "Import" or paste text directly
4. Copy content from any `.whimsical.txt` file
5. Paste into Whimsical
