.gitignore Essencial
	Por Que Isso Importa
		Caso Uber 2019
			57 milhoes de dados vazados
			Credenciais AWS no GitHub
		Desenvolvedor
			Chave AWS commitada
			24 horas: $6.000 prejuizo
		Nao pode acontecer com voce
	O Que NUNCA Versionar
		Credenciais e segredos
			Arquivos .env
			Senhas, chaves API, tokens
			Se commitar, fica no historico PARA SEMPRE
		Dependencias
			node_modules (JavaScript)
			venv (Python)
			Baixaveis com npm install ou pip install
			Podem ter 200MB
		Arquivos de build
			Pasta dist, build
			Arquivos minificados
			Gerados automaticamente
		Configuracoes de IDE
			Pasta .vscode, .idea
			Preferencias pessoais
			Cada dev tem a sua
		Arquivos do sistema
			.DS_Store (Mac)
			Thumbs.db (Windows)
			Nada a ver com seu codigo
	Criando .gitignore
		Exemplo basico JavaScript
			# Variaveis de ambiente
			.env
			.env.local
			# Dependencias
			node_modules/
			# Build
			dist/
			build/
			# IDEs
			.vscode/
			.idea/
			# Sistema
			.DS_Store
			Thumbs.db
		Sintaxe
			# = comentario
			barra no final = diretorio
			asterisco = coringa (*.log)
	Templates Prontos
		github.com/github/gitignore
			Templates por linguagem
			Node, Python, Java, tudo
		gitignore.io
			Escolhe linguagem + IDE + sistema
			Gera automaticamente
	Padrao .env.example
		Problema
			.env tem senhas
			Nao pode versionar
		Solucao
			.env.example sem senhas reais
		.env (NAO versionar)
			DATABASE_URL=postgres://user:SENHA_SECRETA@localhost/db
			API_KEY=chave_secreta_real_123
		.env.example (versionar)
			DATABASE_URL=postgres://user:password@localhost/dbname
			API_KEY=your_api_key_here
		No .gitignore
			.env
			!.env.example
			Exclamacao = NAO ignore
		Quem clonar
			Copia .env.example para .env
			Preenche credenciais reais
	Exercicio
		Criar .env fake
			echo "API_KEY=secret123" > .env
		git status
			.env aparece (perigoso!)
		Criar .gitignore
			echo ".env" > .gitignore
			echo "node_modules/" >> .gitignore
		git status de novo
			.env sumiu!
		Criar .env.example
			echo "API_KEY=your_api_key_here" > .env.example
		Commitar
			git add .gitignore .env.example
			git commit -m "chore: add .gitignore and env template"
			git push
	Resumo
		.gitignore lista o que Git ignora
		Nunca versione .env, node_modules, builds
		Use .env.example como template
		Credenciais no historico ficam la PARA SEMPRE
		Configuracao critica de seguranca
