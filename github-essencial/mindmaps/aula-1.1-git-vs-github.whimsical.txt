Git vs GitHub
	Git: Programa no Computador
		O que e
			Programa que voce instala
			Como Word, Excel
			Funciona no terminal
		O que faz
			Guarda historico de mudancas
			Super Ctrl+Z infinito
			Mostra o que mudou
			Recupera arquivos deletados
		Funciona offline
			Nao precisa de internet
			Roda localmente
	GitHub: Servico na Nuvem
		O que e
			Site: github.com
			Servico na internet
			Como Google Drive
		O que faz
			Guarda projetos na nuvem
			Backup seguro
			Acesso de qualquer lugar
			Colaboracao com outras pessoas
		Workflow offline/online
			Baixa projeto do GitHub
			Trabalha offline com Git
			Sincroniza quando tem internet
	Como Trabalham Juntos
		Git
			Programa no computador
			Guarda historico
		GitHub
			Servico na nuvem
			Guarda copia de tudo
		Analogia
			Git = Camera do celular
				Tira e guarda fotos no aparelho
				Funciona sozinha
			GitHub = Google Fotos
				Guarda na nuvem
				Backup e acesso remoto
	Por Que Isso Importa
		Organizacao economiza tempo
			Nao procura versao antiga
			Nao refaz trabalho perdido
			Nao explica o que mudou
		Tempo e dinheiro
			Quem se organiza produz mais
			Menos estresse
		Git + GitHub
			Ferramentas de produtividade
	Checklist
		Git: programa no computador
		GitHub: servico na nuvem
		Git funciona offline
		GitHub sincroniza com internet
		Juntos: historico + backup + organizacao
