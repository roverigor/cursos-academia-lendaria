Conexao Segura (SSH)
	O Que e SSH
		Forma segura de conectar dois computadores
		Secure Shell
		Depois de configurar
			Nunca mais digita senha
			Funciona para sempre
	Como Funciona
		Par de chaves
			Chave privada
				Fica no seu computador
				Nunca compartilha
				Como chave da sua casa
			Chave publica
				Envia para o GitHub
				Como um cadeado
		Teste de conexao
			GitHub testa se sua chave abre o cadeado
			Se combina, libera conexao
		Tipo Ed25519
			Mais moderno e seguro
			GitHub recomenda desde 2022
	Passo 1: Verificar Se Ja Tem
		Comando
			ls ~/.ssh/id_ed25519.pub
		Resultado
			Arquivo existe: pular para adicionar no GitHub
			Erro: criar a chave
	Passo 2: Criar a Chave
		Comando
			ssh-keygen -t ed25519 -C "seu@email.com"
		Perguntas
			Onde salvar: Enter (aceitar padrao)
			Passphrase (senha)
				Computador so seu: deixar em branco
				Computador compartilhado: criar senha
	Passo 3: Ativar a Chave
		Mac/Linux
			eval "$(ssh-agent -s)"
			ssh-add ~/.ssh/id_ed25519
		Windows (Git Bash)
			eval `ssh-agent -s`
			ssh-add ~/.ssh/id_ed25519
	Passo 4: Copiar Chave Publica
		Mac
			pbcopy < ~/.ssh/id_ed25519.pub
		Windows (Git Bash)
			cat ~/.ssh/id_ed25519.pub | clip
		Linux
			cat ~/.ssh/id_ed25519.pub
			Copiar manualmente o texto
		Formato da chave
			Comeca com ssh-ed25519
			Termina com seu email
	Passo 5: Adicionar no GitHub
		github.com → sua foto → Settings
		SSH and GPG keys
		New SSH key
		Preencher
			Title: nome do computador
			Key: colar a chave copiada
		Add SSH key
	Passo 6: Testar Conexao
		Comando
			ssh -T git@github.com
		Primeira vez
			Pergunta sobre fingerprint
			Digitar yes
		Sucesso
			"Hi SeuUsuario! You've successfully authenticated..."
		Conexao funcionando
			Nunca mais digita senha
	Modulo 1 Completo
		Git instalado
		Nome e email configurados
		Conexao segura com GitHub
		Pronto para trabalhar
