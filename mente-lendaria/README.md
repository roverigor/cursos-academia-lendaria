# Mente Lendaria

**Construa seu Segundo Cerebro com o Metodo 4C**

## Visao Geral

Mente Lendaria e um curso que ensina como construir um Segundo Cerebro usando o Metodo 4C: **Consumir**, **Capturar**, **Conectar** e **Criar**. O curso transforma a forma como voce processa informacao, transformando consumo passivo em criacao ativa.

## Estrutura do Curso

| Modulo | Nome | Aulas | Foco |
|--------|------|-------|------|
| M0 | Introducao | 01-03 | Visao geral do metodo 4C |
| M1 | Consumir | 04-08 | Mentalidade Cacador |
| M2 | Capturar | 09-17 | Mentalidade Curador |
| M3 | Conectar | 18-21 | Sexo de Ideias e MOCs |
| M4 | Criar | 22-24 | Suas Ideias no Mundo |

**Total:** 24 aulas | **Duracao estimada:** 8-10 horas

## Framework 4C

```
CONSUMIR ──► CAPTURAR ──► CONECTAR ──► CRIAR
   │            │            │           │
   ▼            ▼            ▼           ▼
 Cacador     Curador      MOCs       Output
 Filtrar     Anotar     Conectar    Publicar
```

## Conceitos Principais

### Modulo 1: Consumir
- **Mentalidade Cacador**: Consumo intencional vs acumulacao
- **Horizontal vs Vertical**: Amplitude vs profundidade
- **Geladeira de Ideias**: Sistema de triagem temporaria

### Modulo 2: Capturar
- **12 Problemas de Feynman**: Filtros mentais para relevancia
- **Anotacao Lego**: Notas atomicas (de/para/similar/oposto)
- **Anotacao Tetris**: Notas expandidas
- **Captura com IA**: ChatGPT para processamento

### Modulo 3: Conectar
- **Sexo de Ideias**: Inovacao por recombinacao
- **MOCs**: Maps of Content (Nick Milo)
- **Divergir/Convergir**: Ritmo criativo

### Modulo 4: Criar
- **Modo Consumidor x Criador**: Alternancia consciente
- **Output ativo**: Transformar conhecimento em criacao
