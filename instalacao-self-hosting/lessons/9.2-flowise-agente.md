# AULA 9.2: Criando Agente com ChatGPT

---

## ROTEIRO COMPLETO

### [CONEXÃO DRE] - 15 segundos

**[INSTRUTOR NA CÂMERA]**

> Vamos criar um agente de IA que responde perguntas sobre seu negócio.
>
> Atendimento 24h, respostas inteligentes, sem você precisar estar lá.

---

### [CONFIGURANDO API OPENAI] - 60 segundos

**[TELA: Flowise]**

> Primeiro, vamos cadastrar a API key do OpenAI.
>
> **[NAVEGA]**
>
> Menu → Credentials → Add Credential
>
> **[SELECIONA]**
>
> Tipo: OpenAI API
>
> **[COLA]**
>
> Nome: "OpenAI"
> API Key: sua-chave-da-openai
>
> **[SALVA]**
>
> **[EXPLICA]**
>
> Se não tem, cria em platform.openai.com → API Keys.
>
> Custa alguns centavos por uso, mas muito menos que ferramentas pagas.

---

### [CRIANDO CHATFLOW] - 90 segundos

**[TELA: Flowise]**

> **[NAVEGA]**
>
> Chatflows → Add New
>
> **[MOSTRA CANVAS VAZIO]**
>
> Vamos montar o agente.
>
> **[ADICIONA BLOCO]**
>
> Na esquerda, busca: "ChatOpenAI"
>
> **[ARRASTA PRO CANVAS]**
>
> Esse é o cérebro do agente — o ChatGPT.
>
> **[CONFIGURA]**
>
> - Credential: OpenAI (que criamos)
> - Model: gpt-3.5-turbo (mais barato) ou gpt-4
> - Temperature: 0.7

---

### [ADICIONANDO MEMÓRIA] - 60 segundos

**[TELA: Flowise]**

> Agora vamos dar memória pro agente.
>
> **[BUSCA]**
>
> "Buffer Memory"
>
> **[ARRASTA]**
>
> **[CONECTA]**
>
> Conecta a saída do Memory na entrada "Memory" do ChatOpenAI.
>
> **[EXPLICA]**
>
> Com isso, o agente lembra da conversa anterior.
>
> "Você falou sobre X" — ele vai saber.

---

### [ADICIONANDO PROMPT] - 60 segundos

**[TELA: Flowise]**

> Agora o prompt — as instruções do agente.
>
> **[BUSCA]**
>
> "Chat Prompt Template"
>
> **[ARRASTA E CONECTA]**
>
> **[CONFIGURA]**
>
> System Message:
>
> ```
> Você é um assistente da empresa [SUA EMPRESA].
> Seu papel é responder dúvidas sobre nossos serviços de forma educada e profissional.
> Se não souber a resposta, diga que vai encaminhar para um atendente humano.
> ```
>
> **[CONECTA]**
>
> Prompt → ChatOpenAI

---

### [TESTANDO] - 60 segundos

**[TELA: Flowise]**

> **[CLICA NO CHAT]**
>
> Botão de chat no canto inferior direito.
>
> **[TESTA]**
>
> "Olá, quais serviços vocês oferecem?"
>
> **[MOSTRA RESPOSTA]**
>
> Olha! O agente respondeu baseado no prompt.
>
> **[TESTA MEMÓRIA]**
>
> "Pode repetir o que você disse?"
>
> **[MOSTRA]**
>
> Lembrou! A memória funciona.

---

### [PUBLICANDO API] - 45 segundos

**[TELA: Flowise]**

> **[SALVA]**
>
> Clica "Save" → dá um nome: "Agente Atendimento"
>
> **[MOSTRA API]**
>
> Vê esse ícone de código? Clica.
>
> **[MOSTRA ENDPOINT]**
>
> Esse é o endpoint da API do seu agente:
>
> `https://ia.seudominio.com.br/api/v1/prediction/xxx`
>
> Você pode integrar com N8N, WhatsApp, site...

---

### [CHECKPOINT] - 15 segundos

> **CHECKPOINT:**
>
> Se o agente respondeu e lembrou da conversa, funcionou!
>
> Você tem um agente de IA próprio.

---

### [PRÓXIMA AÇÃO] - 30 segundos

**[INSTRUTOR NA CÂMERA]**

> Isso foi o básico. Com Flowise você pode:
>
> - Conectar documentos (RAG)
> - Criar agentes com ferramentas
> - Integrar com qualquer API
>
