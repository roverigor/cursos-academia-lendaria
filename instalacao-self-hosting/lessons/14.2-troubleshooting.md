### [PROBLEMA 1: SITE NÃO ABRE (502/503)] - 60 segundos

**[TELA: Navegador com erro 502]**

> **Erro: "502 Bad Gateway" ou "503 Service Unavailable"**
>
> **Causa:** O container parou ou não subiu direito.
>
> **Solução:**
>
> 1. Vai no Portainer → Containers
> 2. Procura o container da ferramenta
> 3. Vê o status: está "Running" ou "Exited"?
>
> **[MOSTRA NO PORTAINER]**
>
> Se tiver "Exited":
> - Clica no container → Logs
> - Lê o erro (geralmente no final)
> - Reinicia: clica em "Restart"
>
> **[DEMONSTRA]**
>
> 90% das vezes, reiniciar resolve.

---

### [PROBLEMA 2: CERTIFICADO SSL INVÁLIDO] - 45 segundos

**[TELA: Navegador com aviso de certificado]**

> **Erro: "Sua conexão não é particular"**
>
> **Causa:** Certificado SSL não foi gerado.
>
> **Solução:**
>
> 1. Verifica se o DNS propagou (dnschecker.org)
> 2. Verifica logs do Traefik:
>
> ```bash
> docker logs traefik
> ```
>
> **[MOSTRA]**
>
> Procura por erros de "acme" ou "certificate".
>
> Se o DNS tá certo, reinicia o Traefik:
>
> ```bash
> docker restart traefik
> ```
>
> Aguarda 1-2 minutos e tenta de novo.

---

### [PROBLEMA 3: CONTAINER NÃO SOBE] - 45 segundos

**[TELA: Portainer - Deploy failed]**

> **Erro: Stack não consegue fazer deploy**
>
> **Causa comum:** Erro no docker-compose.yml
>
> **Solução:**
>
> 1. Lê a mensagem de erro com atenção
> 2. Erros comuns:
>    - Indentação errada (YAML é chato com espaços)
>    - Domínio com typo
>    - Porta já em uso
>
> **[MOSTRA EXEMPLO]**
>
> Use um validador de YAML online: `yamlvalidator.com`
>
> Cola seu compose, ele mostra onde tá o erro.

---

### [PROBLEMA 4: MEMÓRIA CHEIA] - 45 segundos

**[TELA: Terminal]**

> **Erro: Servidor lento ou containers morrendo**
>
> **Causa:** Memória RAM esgotada.
>
> **Diagnóstico:**
>
> ```bash
> free -h
> ```
>
> **[MOSTRA]**
>
> Se "available" estiver muito baixo, é isso.
>
> **Soluções:**
>
> 1. Reinicia containers que não usa:
>    - Portainer → Container → Stop
>
> 2. Ou faz upgrade da VPS (mais RAM)
>
> 3. Ou limpa imagens não usadas:
>
> ```bash
> docker system prune -a
> ```

---

### [PROBLEMA 5: ESQUECI A SENHA] - 30 segundos

**[INSTRUTOR NA CÂMERA]**

> **Erro: Não lembro a senha do Portainer/ferramenta**
>
> **Solução Portainer:**
>
> ```bash
> docker stop portainer
> docker rm portainer
> rm -rf /opt/portainer/portainer_data
> ```
>
> Depois, sobe o compose de novo. Vai pedir pra criar senha nova.
>
> **Outras ferramentas:** Cada uma tem processo diferente. Consulta a documentação oficial ou nossa comunidade.
