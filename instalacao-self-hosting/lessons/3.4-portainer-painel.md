## ROTEIRO COMPLETO

### [CONEXÃO DRE] - 20 segundos

**[INSTRUTOR NA CÂMERA]**

> Você instalou Docker, mas quem quer ficar digitando comandos?
>
> Portainer é um painel visual pra gerenciar Docker.
>
> Instalar ferramentas vira: colar, clicar, pronto.
>
> É o que você vai usar pra tudo daqui pra frente.

---

### [INSTALANDO PORTAINER] - 2 minutos

**[TELA: Terminal]**

> Conecta na VPS e cria a pasta:
>
> ```bash
> mkdir -p /opt/portainer && cd /opt/portainer
> ```
>
> **[EXECUTA]**
>
> Cria o arquivo:
>
> ```bash
> nano docker-compose.yml
> ```
>
> **[COLA O CONTEÚDO]**

```yaml
version: "3.8"

services:
  portainer:
    image: portainer/portainer-ce:latest
    container_name: portainer
    restart: unless-stopped
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock
      - portainer_data:/data
    labels:
      - "traefik.enable=true"
      - "traefik.http.routers.portainer.rule=Host(`portainer.seudominio.com.br`)"
      - "traefik.http.routers.portainer.entrypoints=websecure"
      - "traefik.http.routers.portainer.tls.certresolver=letsencrypt"
      - "traefik.http.services.portainer.loadbalancer.server.port=9000"
    networks:
      - traefik_proxy

volumes:
  portainer_data:

networks:
  traefik_proxy:
    external: true
```

> **[EXPLICA]**
>
> Troca `seudominio.com.br` pelo seu domínio real.
>
> Salva: `Ctrl+O`, Enter, `Ctrl+X`
>
> **Sobe o Portainer:**
>
> ```bash
> docker compose up -d
> ```
>
> **[EXECUTA]**
>
> Aguarda alguns segundos...

### [PRIMEIRO ACESSO] - 90 segundos

**[TELA: Navegador]**

> Agora acessa no navegador:
>
> `https://portainer.seudominio.com.br`
>
> **[ACESSA]**
>
> **Primeira vez: criar admin**
>
> - Username: admin (ou o que preferir)
> - Password: senha forte (mínimo 12 caracteres)
>
> **[PREENCHE E CLICA]**
>
> Clica em "Create user"
>
> **Selecionar ambiente:**
>
> Clica em "Get Started" → Seleciona "local"
>
> **[CLICA]**
>
> Pronto! Você está no painel do Portainer.

---

### [TOUR PELO PAINEL] - 90 segundos

**[TELA: Portainer Dashboard]**

> Deixa eu te mostrar o que tem aqui:
>
> **[NAVEGA E MOSTRA]**
>
> **Home:** Visão geral dos ambientes
>
> **Stacks:** Onde você vai instalar as ferramentas
> - Cada "stack" é um conjunto de containers
> - N8N vai ser uma stack, Chatwoot outra, etc.
>
> **Containers:** Lista de containers rodando
> - Aqui você vê Traefik e Portainer
>
> **Images:** Imagens Docker baixadas
>
> **Networks:** Redes (você já criou a traefik_proxy)
>
> **Volumes:** Onde ficam os dados persistentes
>
> **[DESTACA]**
>
> O mais importante: **Stacks**. É lá que você vai trabalhar.

---

### [TESTANDO: CRIANDO UMA STACK] - 60 segundos

**[TELA: Portainer]**

> Vamos testar criando uma stack simples:
>
> Stacks → Add Stack
>
> **[CLICA]**
>
> - Nome: `teste`
> - Cola esse compose:
>
> ```yaml
> version: "3.8"
> services:
>   whoami:
>     image: traefik/whoami
>     labels:
>       - "traefik.enable=true"
>       - "traefik.http.routers.whoami.rule=Host(`teste.seudominio.com.br`)"
>       - "traefik.http.routers.whoami.entrypoints=websecure"
>       - "traefik.http.routers.whoami.tls.certresolver=letsencrypt"
>     networks:
>       - traefik_proxy
>
> networks:
>   traefik_proxy:
>     external: true
> ```
>
> **[COLA]**
>
> Troca o domínio e clica em "Deploy the stack"
>
> **[ACESSA teste.seudominio.com.br]**
>
> Se aparecer informações do servidor, **tudo funcionando!**
>
> Pode deletar essa stack depois (é só teste).
