## Common Curriculum Patterns

### Core Topics (90%+ of courses)
1. **Module 1: Introduction & Installation** (all courses)
   - Why Obsidian / second brain concept
   - Installation (desktop)
   - Basic setup

2. **Module 2: Markdown & Note-taking** (10/12 courses)
   - Markdown syntax
   - Creating notes
   - File organization basics

3. **Module 3: Organization** (9/12 courses)
   - Folders vs tags
   - Naming conventions
   - File management

4. **Module 4: Links & Connections** (11/12 courses)
   - Internal links
   - Backlinks
   - Graph view

5. **Module 5: Plugins** (7/12 courses)
   - Community plugins overview
   - Installing plugins
   - 2-3 recommended plugins (Calendar, Dataview most common)
