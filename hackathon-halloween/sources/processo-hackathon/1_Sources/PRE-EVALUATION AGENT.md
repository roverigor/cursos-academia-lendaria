- Demonstrates book knowledge
- Can explain popular ideas
- References other experts' work
- No evidence of personal application

Evidence required:
- Only references others' work
- No original application or experience
- Vague generalities
- Possibly misunderstands concepts

no depth, no application."

TYPE 1: TEMPORAL EVOLUTION (valuable)
Pattern: Belief changes across time, person acknowledges shift

Example:
- Source A (2015): "Work-life balance is essential to my values"
- Source B (2023): "Work-life balance is a myth I no longer believe"
- Source C (2024): "I evolved on this. I used to think... now I think..."

Assessment: GOOD. Shows genuine evolution. Look for bridge source (C).
Value: High (reveals change process, not just endpoints)
Action: Document shift, find source explaining why

TYPE 2: CONTEXTUAL VARIANCE (valuable)
Pattern: Person behaves differently in different contexts

Example:
- Podcast (casual): "I'm deeply introverted, I recharge alone"
- Profile article: Friends say "He's the life of every party"
- Essay: "I've learned to perform extroversion for professional settings"

Assessment: GOOD. Reveals complexity, context-dependent behavior.
Value: High (shows person has range, adapts)
Action: Map contexts, document variance

TYPE 3: SURFACE INCONSISTENCY (neutral)
Pattern: Minor contradictions on non-core topics

Example:
- Interview A: "I wake up at 5am"
- Interview B: "I wake up at 6am"

Assessment: NEUTRAL. Habits change, or not strictly consistent.
Value: Low (not psychologically significant)
Action: Note but don't emphasize

TYPE 4: CORE INCOHERENCE (problematic)
Pattern: Contradictory core values with no evolution explanation

Example:
- Source A (2020): "Transparency is my core value, I share everything"
- Source B (2022): "Privacy is essential, some things should never be shared"
- No bridge source explaining the shift

Assessment: COMPLEX. Could be evolution OR rationalization OR context-dependent.
Value: High (needs resolution)
Action: FLAG. Need source that bridges the gap or explains

TYPE 5: SELF-REPORT vs OBSERVER MISMATCH (critical)
Pattern: Person says X, observers report Y, no acknowledgment

Example:
- Person: "I value collaboration, I'm a team player"
- Multiple observers: "He's extremely competitive, dominates every meeting"
- Person never discusses this tension

Assessment: CRITICAL. Potential blind spot or self-deception.
Value: Very high (reveals shadow aspects)
Action: HIGHLIGHT. This is gold for authentic clone.

- Reflective (person thinking aloud, not scripted)
- Specific (names, dates, concrete examples)
- Vulnerable (admits failures, uncertainties)
- Dense (every paragraph has insight)

LOW SIGNAL sources:
- Short-form (<15min / <500 words)
- Scripted (rehearsed, media-trained)
- Vague (generic advice, no specifics)
- Performative (marketing mode)
- Thin (mostly filler, few insights)

SCORING:
Calculate average signal score per source:
- Count total sources
- Classify each: High (3) / Medium (2) / Low (1)
- Average = (sum of scores) / (number of sources)

Target: Average 2.3+ (mostly high/medium signal)

Example:
- 10 high-signal sources = 30 points
- 8 medium-signal sources = 16 points
- 2 low-signal sources = 2 points
- Total: 48 points / 20 sources = 2.4 average (good)
