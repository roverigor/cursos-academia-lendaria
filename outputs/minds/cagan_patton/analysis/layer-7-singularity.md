## Marty Cagan's Singularity

### The Empowerment Litmus Test

**Unique Insight:** Empowerment is not about autonomy or trust or culture - it's about **problem assignment with outcome accountability**.

**The Formula:**
```
True Empowerment =
  Assigned meaningful problems (not features)
  + Authority to decide HOW to solve
  + Accountability for outcomes (not output)
```

**Why This is Singular:**
- Everyone talks about "empowerment" vaguely
- Cagan provides a **testable definition**
- The litmus test: "Can the team decide the best way to solve the problem?"
- If leaders prescribe solutions → Not empowered (regardless of "culture")

**Evidence:** Source 03 - "The litmus test for empowerment is that the team is able to decide the best way to solve the problems they have been assigned"

**Impact:** Converted fuzzy cultural concept into operational criteria

## Jeff Patton's Singularity

### User Story Mapping as Universal Decomposition Language

**Unique Insight:** 2D spatial organization (horizontal = narrative flow, vertical = priority) is not just a technique - it's a **cognitive tool that reveals system structure invisible in linear formats**.

**The Discovery:**
- Patton didn't "invent" story mapping
- He **recognized an emerging pattern** others were independently discovering
- Gave it language, made it teachable
- Showed it's universal across product types

**Why This is Singular:**
- Not a new idea but a discovered pattern (like Newton and gravity)
- Works because it matches how humans naturally think about journeys
- The 2D structure isn't arbitrary - it maps to:
  - Horizontal = temporal flow (how users move through system)
  - Vertical = importance/refinement (what's essential vs. nice-to-have)

**Evidence:** Source patton/04 - "This isn't an innovation but rather an emerging pattern...citing parallels in Indi Young's Mental Models and Todd Warfel's Task Analysis Grid"

**Impact:** Gave teams a visual language for decomposition that creates shared understanding

## Combo Singularity

### Strategic Discovery → Visual Decomposition Protocol

**Unique to the Combo:**

The Cagan-Patton handoff protocol creates something neither provides alone:

1. **Cagan:** Strategic filter identifies THE problem worth solving
2. **Patton:** Story map visually decomposes HOW to solve it
3. **Integration:** Problem clarity (Cagan) + Decomposition tool (Patton) = Executable strategy

**Formula:**
```
Opportunity Assessment (filter)
  → Empowered Team (assigned problem)
  → Story Mapping Workshop (visual decomposition)
  → Walking Skeleton (first value delivery)
  → Outcome Measurement (validate results)
```

**Why This Combo is Singular:**
- Cagan alone: Great at WHAT problems, less prescriptive on HOW to decompose
- Patton alone: Great at decomposition, but which problems deserve the effort?
- Together: End-to-end from strategy to execution
