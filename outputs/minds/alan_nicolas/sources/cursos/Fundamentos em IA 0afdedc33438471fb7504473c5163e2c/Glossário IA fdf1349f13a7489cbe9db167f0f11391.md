# Glossário IA

1. **Algoritmo**: Um algoritmo é um conjunto de instruções ou regras definidas que são seguidas para realizar uma tarefa ou resolver um problema, especialmente por um computador. Na IA, os algoritmos são fundamentais para processar dados, aprender com eles e tomar decisões ou fazer previsões com base em padrões identificados.
2. **ANPD** (Autoridade Nacional de Proteção de Dados): No Brasil, a ANPD é um órgão responsável pela fiscalização e aplicação da Lei Geral de Proteção de Dados (LGPD). Ela atua no estabelecimento de normas, orientações e procedimentos sobre como os dados pessoais devem ser tratados, garantindo a privacidade e proteção dos dados dos cidadãos.
3. **API** (Interface de Programação de Aplicações): Uma API é um conjunto de regras e definições que permite que diferentes softwares ou componentes de software se comuniquem entre si. Na IA, APIs são frequentemente usadas para integrar algoritmos de aprendizado de máquina e funcionalidades de IA em aplicações e sistemas existentes.
4. **AWS** (Amazon Web Services): AWS é uma plataforma de serviços de computação em nuvem oferecida pela Amazon. Ela fornece uma variedade de serviços, incluindo armazenamento de dados, poder de processamento e ferramentas de IA, que ajudam as empresas e desenvolvedores a criar e escalar aplicações.
5. **Backpropagation (Retropropagação)**: Na IA, backpropagation é um método utilizado em redes neurais artificiais para ajustar os pesos das conexões na rede durante o treinamento. Esse processo minimiza o erro entre a saída esperada e a saída real da rede, otimizando seu desempenho na tarefa de aprendizado.
6. **Backend**: O backend refere-se à parte da arquitetura de software que lida com a lógica de negócios, o armazenamento e o processamento de dados, geralmente no servidor. Na IA, o backend é crucial para processar dados complexos, realizar cálculos e fornecer os resultados para a interface do usuário (frontend).
7. **Big Data**: Big Data é um termo que descreve grandes volumes de dados - tanto estruturados quanto não estruturados - que são processados e analisados para revelar padrões, tendências e associações. Na IA, Big Data é essencial para alimentar algoritmos de aprendizado de máquina com a variedade e quantidade de dados necessários para o treinamento eficaz.
8. **BPM** (Business Process Management): BPM é uma abordagem disciplinada para identificar, desenhar, executar, documentar, medir, monitorar e controlar processos de negócios automatizados e não automatizados para alcançar resultados consistentes e alinhados com os objetivos estratégicos da organização.
9. **BYOD** (Bring Your Own Device): BYOD é uma política empresarial que permite aos funcionários usar seus dispositivos pessoais - como smartphones, tablets e laptops - para acessar sistemas e dados da empresa. Embora ofereça flexibilidade, também apresenta desafios em termos de segurança e gerenciamento de dados.
10. **Cache**: Cache é uma forma de armazenamento temporário que ajuda a acelerar o acesso a dados e informações. Em computadores e dispositivos móveis, os caches armazenam dados que são acessados frequentemente, permitindo um carregamento mais rápido em usos subsequentes.
11. **ChatGPT**: ChatGPT é um modelo de linguagem baseado em IA desenvolvido pela OpenAI. Ele é capaz de gerar respostas textuais de forma coerente e relevante, baseando-se em uma vasta quantidade de informações e aprendizado de máquina. É amplamente utilizado para responder perguntas, criar conteúdo e em diversas outras aplicações de processamento de linguagem natural.
12. **Cloud (Nuvem)**: A computação em nuvem refere-se ao armazenamento e acesso de dados e programas pela Internet, em vez de em um computador ou servidor local. Ela oferece escalabilidade, eficiência e economia de custos, permitindo aos usuários acessar recursos de TI sob demanda.
13. **CMYK (Cyan, Magenta, Yellow e Black Key)**: CMYK é um modelo de cores usado em impressão colorida. Ele utiliza quatro tintas (ciano, magenta, amarelo e preto) sobrepostas em diferentes intensidades para criar uma ampla gama de cores. É o padrão na indústria gráfica para impressão em papel.
14. **Cookie**: Cookies são pequenos arquivos de dados armazenados no navegador do usuário ao visitar websites. Eles são usados para lembrar informações sobre a visita, como preferências de navegação, login, e ajudam na personalização da experiência do usuário na web.
15. **Criptografia**: Criptografia é o processo de converter informações ou dados em um código para prevenir acessos não autorizados. É uma prática essencial para garantir a segurança da informação em comunicações digitais e armazenamento de dados.
16. **CRM (Customer Relationship Management)**: CRM é um sistema que ajuda as empresas a gerenciar e analisar as interações com clientes atuais e potenciais. O objetivo é melhorar as relações comerciais, auxiliando na retenção de clientes e no aumento das vendas.
17. **CMS (Content Management System)**: Um CMS é uma plataforma de software que permite a criação, edição, gerenciamento e publicação de conteúdo digital. É amplamente utilizado em websites e blogs para gerenciar o conteúdo de forma eficiente e intuitiva.
18. **CPU (Central Processing Unit)**: A CPU, ou Unidade Central de Processamento, é o cérebro do computador, onde a maioria dos cálculos ocorre. É um componente crucial que executa as instruções de um programa de computador interpretando e processando as operações básicas.
19. **CSV (Valores Separados por Vírgula)**: Um arquivo CSV é um tipo de arquivo de texto que utiliza uma vírgula para separar valores. É comumente usado para trocar dados entre diferentes aplicações, como bancos de dados e planilhas.
20. **Dados de Treinamento**: Em IA, dados de treinamento são conjuntos de dados usados para treinar modelos de aprendizado de máquina. Esses dados são essenciais para que o modelo aprenda a realizar tarefas específicas, como reconhecimento de imagem, processamento de linguagem natural, entre outras.
21. **Data Compliance (Conformidade de Dados)**: Refere-se à adesão às leis e regulamentações que governam a coleta, armazenamento e uso de dados. A conformidade de dados é crucial para assegurar a privacidade do usuário e a proteção contra o uso indevido de dados.
22. **Deep Learning (Aprendizado Profundo)**: Deep learning é um subcampo da aprendizagem de máquina baseado em redes neurais artificiais com várias camadas. É usado para modelar padrões complexos em grandes conjuntos de dados, sendo fundamental em aplicações como reconhecimento de fala, visão computacional e tradução automática.
23. **DNS (Domain Name System)**: O DNS é um sistema que traduz nomes de domínio legíveis por humanos (como openai.com) em endereços IP numéricos. Isso facilita a navegação na Internet, permitindo que os usuários encontrem sites usando nomes de domínio em vez de endereços
24. **DPO (Data Protection Officer)**: O Data Protection Officer é um cargo criado pela GDPR (Regulamento Geral sobre a Proteção de Dados) da União Europeia, responsável por garantir que uma organização esteja em conformidade com as leis de proteção de dados. O DPO é o ponto de contato entre a empresa e as autoridades de proteção de dados.
25. **Embed**: Embed, em tecnologia, refere-se à prática de integrar conteúdo de uma fonte externa dentro de um site ou aplicativo. Isso pode incluir incorporar vídeos, mapas, feeds de redes sociais ou outros conteúdos usando um código HTML específico.
26. **Embedar**: É um termo informal derivado de "embed", significando o ato de incorporar conteúdo de uma fonte externa em um website ou aplicativo. Por exemplo, embedar um vídeo do YouTube em um blog.
27. **Fine-Tuning (Ajuste Fino)**: No contexto de IA, fine-tuning é o processo de fazer ajustes refinados em um modelo de aprendizado de máquina pré-treinado para melhorar seu desempenho em uma tarefa específica ou conjunto de dados.
28. **Fintech**: Fintech, uma combinação de "financeiro" e "tecnologia", refere-se às empresas e serviços que usam tecnologia para automatizar e melhorar os serviços financeiros. Isso inclui tudo, desde pagamentos móveis e transferências online até crowdfunding e gestão de investimentos.
29. **Flaw Point**: Termo não amplamente reconhecido no contexto de IA ou tecnologia. Pode se referir a um ponto de falha ou uma fraqueza em um sistema ou processo.
30. **Frontend**: O frontend de um aplicativo ou site é a parte com a qual o usuário interage diretamente. Inclui a interface do usuário, design gráfico e alguns aspectos de programação que funcionam no navegador do usuário.
31. **Full Stack**: Um desenvolvedor full stack é alguém que tem a habilidade de trabalhar tanto no frontend quanto no backend de um aplicativo ou site. Eles possuem conhecimentos abrangentes que cobrem várias tecnologias e plataformas.
32. **GDPR (General Data Protection Regulation)**: A GDPR é um regulamento da União Europeia que estabelece diretrizes para a coleta e processamento de informações pessoais de indivíduos dentro da UE. É um dos marcos legais mais rigorosos em termos de proteção de dados.
33. **GPT (Generative Pre-training Transformer)**: GPT é um tipo de modelo de linguagem baseado em IA desenvolvido pela OpenAI. Utiliza o aprendizado profundo para gerar texto, podendo ser adaptado para diversas tarefas de processamento de linguagem natural, como tradução, resumo e geração de conteúdo.
34. **GPU (Graphics Processing Unit)**: Uma GPU é um processador especializado otimizado para acelerar a manipulação e criação de imagens e vídeos. Em IA, GPUs são fundamentais para processar grandes conjuntos de dados e realizar cálculos complexos de forma mais eficiente do que as CPUs tradicionais.
35. **IA (Inteligência Artificial)**: Inteligência Artificial é a simulação de processos de inteligência humana por sistemas computacionais. Isso inclui aprendizado (a aquisição de informações e regras para usar a informação), raciocínio (usando regras para chegar a conclusões aproximadas ou definitivas) e autocorreção.
36. **IA Generativa**: Inteligência Artificial Generativa refere-se a sistemas de IA que podem gerar novos conteúdos ou dados que não existiam anteriormente, como imagens, textos, música, etc. Isso é frequentemente realizado através de técnicas como redes generativas adversárias (GANs).
37. **IaaS (Infrastructure as a Service)**: IaaS é um serviço de computação em nuvem que oferece recursos de infraestrutura básica - como servidores, rede, armazenamento e espaço de data center - sobre uma base de pagamento conforme o uso.
38. **IFTTT (If This Then That)**: IFTTT é um serviço gratuito que permite aos usuários criar cadeias de comandos simples, chamadas de "applets". Estes applets são baseados na lógica: se isso acontecer (um gatilho), então faça aquilo (uma ação). É usado para automatizar tarefas entre diferentes serviços e dispositivos.
39. **IMEI (International Mobile Equipment Identity)**: IMEI é um número único de identificação para cada dispositivo móvel. Funciona como uma espécie de identidade digital do aparelho, usado principalmente para rastreamento e segurança, especialmente em casos de roubo ou perda.
40. **I/O (Input/Output)**: Input/Output refere-se ao processo de entrada (input) e saída (output) de dados em um sistema computacional. Input é a informação inserida no sistema, enquanto output é a informação processada e produzida pelo sistema.
41. **IoT (Internet of Things)**: Internet of Things é a rede de dispositivos físicos ("coisas") que estão conectados à internet, coletando e compartilhando dados. Isso inclui tudo, desde eletrodomésticos inteligentes até carros conectados e sistemas de monitoramento urbano.
42. **IP (Internet Protocol)**: IP é o principal protocolo de comunicação para a transmissão de dados através da internet. Cada dispositivo conectado à internet tem um endereço IP único, que é usado para identificar e localizar o dispositivo na rede.
43. **LGPD (Lei Geral de Proteção de Dados)**: A LGPD é a legislação brasileira que regula o tratamento de dados pessoais, incluindo seu armazenamento, manipulação e compartilhamento. É semelhante ao GDPR europeu e visa proteger a privacidade e a segurança dos dados pessoais dos cidadãos.
44. **LLM (Large Language Models)**: Modelos de linguagem de grande escala são sistemas de IA avançados capazes de entender, gerar e interpretar linguagem humana com um alto nível de proficiência. Eles são treinados com enormes conjuntos de dados textuais e são fundamentais em aplicações como tradução automática, resumo de texto e geração de conteúdo.
45. **LMS (Learning Management System)**: Um sistema de gestão de aprendizagem é uma plataforma de software usada para fornecer, acompanhar e gerenciar cursos e treinamentos educacionais. É amplamente usado em educação e treinamento corporativo.
46. **Mineração de Dados**: Mineração de dados é o processo de [](http://insights.ml/)descobrir padrões e conhecimentos a partir de grandes conjuntos de dados. Utiliza métodos de IA, estatística e sistemas de banco de dados para extrair informações úteis e insights.
47. **ML (Machine Learning)**: Machine Learning, ou aprendizado de máquina, é um subcampo da IA que foca no desenvolvimento de sistemas capazes de aprender e melhorar a partir da experiência, sem serem explicitamente programados.
48. **NFC (Near Field Communication)**: NFC é uma tecnologia de comunicação sem fio de curto alcance que permite a troca de dados entre dispositivos próximos. É amplamente utilizada em pagamentos móveis e tags inteligentes.
49. **NLP (Natural Language Processing)**: Processamento de linguagem natural é uma área da IA que se concentra na interação entre computadores e linguagem humana. Ela envolve a compreensão, interpretação e geração de linguagem natural por máquinas[.](http://xn--pesquisveis-q7a.open/)
50. **OCR (Optical Character Recognition)**: OCR é uma tecnologia que converte diferentes tipos de documentos, como imagens digitalizadas de texto impresso ou manuscrito, em dados de texto editáveis e pesquisáveis.
51. **Open Code**: O termo "Open Code" geralmente se refere a "código aberto", que é um modelo de desenvolvimento de software onde o código-fonte é disponibilizado publicamente para que qualquer pessoa possa estudar, modificar e distribuir.
52. **Pixel**: Pixel é a menor unidade que compõe uma imagem digital. São pontos minúsculos que, quando combinados em milhões, criam a ilusão óptica de formas, cores e movimentos nas telas eletrônicas. Os pixels são a base fundamental para a visualização de imagens em dispositivos digitais.
53. **Prompt**: Em IA, um prompt é o "empurrãozinho" dado para iniciar a operação de um modelo de IA, especialmente em IA generativa. É o comando ou solicitação (geralmente em forma de texto) que orienta a IA a realizar uma tarefa específica.
54. **Query**: Uma query, ou consulta, é um pedido de informação a um banco de dados. É como uma pergunta formulada em uma linguagem que o banco de dados pode entender e responder, retornando as informações solicitadas.
55. **RPA (Robotic Process Automation)**: RPA é a tecnologia que permite a automação de processos empresariais por meio de "robôs" de software. Estes robôs podem realizar tarefas repetitivas de forma rápida e precisa, superando a capacidade humana em consistência e eficiência.
56. **SaaS (Software as a Service)**: SaaS é um modelo de distribuição de software onde os aplicativos são hospedados por um provedor de serviços e disponibilizados aos usuários pela internet, geralmente sob um modelo de assinatura. Este modelo é popular entre startups e empresas de tecnologia.
57. **Sandbox**: Em desenvolvimento de software, um sandbox é um ambiente de teste isolado que imita o ambiente de produção. Aqui, os desenvolvedores podem experimentar e testar novas funcionalidades sem afetar o ambiente operacional principal.
58. **SEO (Search Engine Optimization)**: SEO envolve um conjunto de técnicas e estratégias para otimizar sites a fim de melhorar seu posicionamento nos resultados de busca orgânica de motores de busca, como o Google. O objetivo é aumentar a visibilidade e atrair mais tráfego para o site.
59. **SSL (Secure Sockets Layer)**: SSL é um protocolo de segurança que estabelece uma conexão criptografada entre um servidor web e um navegador. Garante que todos os dados transmitidos sejam seguros e privados. É indicado pela presença de "https" e um cadeado na barra de endereços do navegador.
60. **Text-to-Speech**: Text-to-Speech é uma tecnologia de IA que converte texto em fala falada. É amplamente utilizada em assistentes virtuais, leitores de tela para deficientes visuais e outras aplicações que requerem interação vocal.
61. **UI (User Interface)**: User Interface é a interface através da qual um usuário interage com um dispositivo ou aplicativo. Inclui elementos visuais, como botões, ícones, menus e tudo que ajuda o usuário a navegar e interagir com o software ou hardware.
62. **UX (User Experience)**: User Experience é o processo de criação de produtos que proporcionam experiências significativas e relevantes aos usuários. Inclui aspectos de branding, design, usabilidade e função.
63. **URL (Uniform Resource Locator)**: URL é o endereço web de um recurso disponível na internet. É a string de texto que você digita na barra de endereços do navegador para visitar um site ou página específica.
64. **VPN (Virtual Private Network)**: VPN é uma tecnologia que cria uma conexão segura e criptografada sobre uma rede menos segura, como a internet. Permite que usuários enviem e recebam dados de forma privada, mesmo que estejam conectados a uma rede pública.
65. **Wireframe**: Wireframe é um esboço visual básico que mostra a estrutura de um site ou aplicativo. Funciona como um plano, mostrando a disposição dos elementos de interface e a navegação básica do site ou aplicativo, antes do desenvolvimento completo e do design final.
