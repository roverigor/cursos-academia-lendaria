4. **ambiente** (11): Espaço virtual isolado para desenvolvimento de projetos Python. Ex: "Para começar a usar o ambiente virtual, você precisa ativá-lo"
5. **python** (10): Linguagem de programação usada no desenvolvimento com Crew AI. Ex: "Certifique-se de que o Python está instalado em sua máquina."
6. **virtual** (10): Tipo de ambiente isolado para desenvolvimento Python. Ex: "Criar um ambiente virtual em Python com venv"

9. **instalar** (8): Processo de configurar as dependências necessárias, como Python, bibliotecas, etc. Ex: "Passo 1: Instalar o Python"
10. **processo** (7): Sequência de tarefas a serem realizadas pelos agentes no Crew AI. Ex: "Isso significa que as tarefas serão executadas uma após a outra."
