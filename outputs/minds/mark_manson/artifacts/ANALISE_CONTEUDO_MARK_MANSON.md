## 4. PADRÕES IDENTIFICADOS ACROSS MEDIA

### Estrutura Narrativa Universal
```
1. GANCHO: Provocação/Paradoxo
2. HISTÓRIA: Falha pessoal embaraçosa
3. CONCEITO: Princípio universal extraído
4. EVIDÊNCIA: Exemplos, pesquisa, citações
5. INVERSÃO: Subverter expectativa comum
6. APLICAÇÃO: Como usar na vida real
7. DESAFIO: Pergunta para reflexão
```

### Vocabulário Signature (Frequência)
- "fuck/fucking" - 5-8% do texto
- "actually" - transição para verdade
- "Look," - início de explicação
- "Here's the thing" - setup para insight
- "The problem is" - identificação de issue

### Temas por Frequência
1. Responsabilidade pessoal (30%)
2. Valores e prioridades (25%)
3. Relacionamentos (20%)
4. Morte/mortalidade (10%)
5. Sucesso/fracasso (10%)
6. Sociedade/cultura (5%)

### Técnicas Retóricas
- **Paradoxo inicial**: Começar com contradição
- **Confissão vulnerável**: Compartilhar vergonha
- **Evidência anedótica**: História > estatística
- **Pergunta socrática**: Forçar auto-reflexão
- **Humor negro**: Desarmar defesas
