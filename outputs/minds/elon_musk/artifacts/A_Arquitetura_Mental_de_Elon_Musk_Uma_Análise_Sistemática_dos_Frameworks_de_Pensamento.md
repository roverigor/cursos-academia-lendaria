#### Fase 3: Reconstrução do Zero

Constrói nova solução baseada apenas em verdades fundamentais:

- Física permite pouso controlado? Sim
- Podemos criar algoritmos de controle? Sim
- Podemos fazer isso economicamente? Vamos calcular...

```
Idiot Index = (Custo Total Componente) / (Custo Materiais Brutos)

Onde:
Custo Total = Materiais + Mão de Obra + Overhead + Margem + Transporte
Custo Materiais = Preço spot market dos elementos básicos
```

### A Origem no Poker

**História PayPal (2000)**: Peter Thiel relata Musk em mesa de poker:

### Estrutura Matemática do Risco

Musk opera com cálculo modificado de Kelly Criterion:

```
f* = (bp - q) / b

Onde:
f* = fração do bankroll para apostar
b = odds (decimal)
p = probabilidade de ganhar
q = probabilidade de perder (1-p)
```

**Modificação Musk**: Se p > 0.1 (10% chance), aposte TUDO se o payoff for grande o suficiente.

### Casos Documentados de Double Down
