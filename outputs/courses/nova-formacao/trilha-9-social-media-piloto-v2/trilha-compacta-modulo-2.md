# AULA 2.2: TEORIA MINIMA (10 min)

## Batch Content

### O Conceito

```
ERRADO (diario):
Seg: pensar + criar + publicar
Ter: pensar + criar + publicar
Qua: pensar + criar + publicar
= 3h/semana, stress diario

CERTO (batch):
1 dia: pensar + criar TUDO
Resto: so agendar
= 2h/mes, zero stress
```

### Formula do Batch

```
1 sessao de 2h = 30 dias de conteudo

- 30 min: Definir temas (piramide T/M/F)
- 60 min: Gerar com IA
- 30 min: Revisar e agendar
```

### Ferramentas

**Criacao (IA):**
- ChatGPT / Claude - Textos
- Canva AI - Imagens

**Agendamento:**
- Creator Studio - IG/FB
- Buffer - Multi
- Later - IG foco

**Organizacao:**
- Notion - Calendario
- Google Sheets - Simples

### Regra 2-1-1

- **2 horas/mes:** Sessao batch
- **1 hora/semana:** Revisao
- **1 min/dia:** Responder comentarios

# AULA 2.3: EXERCICIO (25 min)

## Sistema 30 Dias

### Prompt: Calendario 30 Dias

```
Crie um calendario editorial de 30 dias para [seu nicho].

CONTEXTO:
- Frequencia: [3x/semana ou 5x/semana]
- Plataforma principal: [Instagram/LinkedIn]
- Objetivo: [engajamento/leads/vendas]

DISTRIBUICAO (Piramide):
- 50% Topo (atrair)
- 30% Meio (conectar)
- 20% Fundo (converter)

PARA CADA DIA:
- Data
- Etapa (T/M/F)
- Tema/Titulo
- Formato (carrossel/reels/imagem)
- Horario sugerido

Organize em tabela.
```

### Prompt: Completar Posts

```
Gere mais [X] posts seguindo a Piramide de Conteudo:

- [X] posts TOPO
- [X] posts MEIO
- [X] posts FUNDO

Mesmo formato:
1. Etapa
2. Gancho
3. Corpo
4. CTA
5. Formato
6. Hashtags
```

### Passo a Passo

**Bloco 1: Gerar Calendario (8 min)**
1. Cole o prompt
2. Preencha dados
3. Copie resultado

**Bloco 2: Ajustar (7 min)**
- [ ] Datas fazem sentido?
- [ ] Distribuicao T/M/F correta?
- [ ] Temas relevantes?

**Bloco 3: Completar (10 min)**

Calcule:
- 3x/semana = 12 posts/mes
- 5x/semana = 20 posts/mes
