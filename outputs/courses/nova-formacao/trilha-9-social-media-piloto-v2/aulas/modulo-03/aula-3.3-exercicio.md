# Aula 3.3: Exercicio - Adapte 3 Trends

## Modulo 3 | 25 minutos | Pratica

---

**Transforme tendencias genericas em conteudo do seu nicho.**

---

## PROMPT: ADAPTADOR DE TRENDS

Cole no ChatGPT/Claude:

```
Adapte esta trend para o meu nicho:

TREND:
[descreva a trend que voce viu]

MEU CONTEXTO:
- Nicho: [seu nicho]
- Publico: [seu ICP]
- Tom: [profissional/casual/bem-humorado]
- Plataforma: [Instagram/TikTok/LinkedIn]

GERE:
1. Versao adaptada da trend
2. Roteiro/texto completo
3. Dicas de execucao
4. Hashtags relevantes
5. Melhor horario para postar
```

---

## PASSO A PASSO (25 min)

### Bloco 1: Identificar Trends (8 min)

1. Abra o Explore do Instagram/TikTok
2. Role por 5 minutos
3. Anote 3 trends que podem funcionar:

| # | Trend | Plataforma | Por Que Escolhi |
|---|-------|------------|-----------------|
| 1 | | | |
| 2 | | | |
| 3 | | | |

**Criterios:**
- Tem relacao com seu nicho
- Voce consegue executar
- Ainda nao esta saturada

### Bloco 2: Adaptar com IA (12 min)

Para cada trend:
1. Cole o prompt
2. Descreva a trend
3. Copie o resultado

**4 min por trend = 12 min total**

### Bloco 3: Revisar e Organizar (5 min)

Para cada adaptacao:
- [ ] Faz sentido pro meu publico?
- [ ] Consigo executar hoje/amanha?
- [ ] Mantem minha identidade?

---

## PROMPT BONUS: Criar Trend Original

```
Crie uma trend original para meu nicho:

NICHO: [seu nicho]
PUBLICO: [seu ICP]
OBJETIVO: [engajamento/viralizar]
FORMATO: [reels/carrossel/imagem]

A trend deve:
1. Ser facil de replicar
2. Gerar identificacao
3. Ter potencial de compartilhamento

Gere 3 opcoes com roteiro completo.
```
