## BLOCO 1: TAREFA + BRIEFING [12 min]

[TELA: "BLOCO 1 - TAREFA + BRIEFING" com timer 12 min]

### Instrucao do Bloco

**Locucao:**
"Bloco 1: Definir tarefa e expandir briefing.

**Parte 1 - TAREFA (4 min):**
Escreve a tarefa com mais detalhes:
- O que FAZ parte (escopo)
- O que NAO faz parte

**Parte 2 - BRIEFING (8 min):**
Expande o briefing:
- O QUE fazer (objetivo claro)
- POR QUE fazer (contexto/importancia)
- PRA QUANDO (deadline)
- O que NAO pode acontecer (erros graves)
- Onde achar recursos (links reais)

**Dica:** Se travar, pensa: 'O que eu falaria pra um estagiario no primeiro dia?'

**Timer: 12 minutos. Comeca agora.**"

**Locucao:**
"Bloco 3: Criar output esperado e validar tudo.

**Parte 1 - OUTPUT ESPERADO (6 min):**
Cria um exemplo REAL de entrega correta.
Pode ser:
- Print de tela
- Texto pronto
- Template preenchido
- Qualquer coisa que mostre 'assim que fica certo'

**Parte 2 - TESTE DO MODELO (4 min):**
Revisa o modelo completo.
Imagina: 'Se eu entregar isso pra pessoa, ela consegue fazer?'

**Parte 3 - AJUSTES (2 min):**
Se faltou algo, adiciona.
Se esta confuso, simplifica.

**Timer: 12 minutos. Comeca agora.**"

## ARTEFATO DA AULA: Modelo de Delegação Assistida Completo

### O Que Você Vai Criar
Modelo completo com todos os 5 componentes: tarefa, briefing, checklist, prompt de validação e output esperado - pronto para implementar.

### Por Que Isso Importa
Este modelo é SISTEMA de delegação:
- Qualquer pessoa nova consegue fazer
- IA valida antes de chegar em você
- Você só intervém quando necessário
- Escala sem aumentar sua carga

**Linha do DRE:** ROI de Delegação (horas liberadas x valor/hora x semanas de uso)

### Quando Usar
- Para treinar qualquer pessoa
- Ao delegar tarefas repetitivas
- Como padrão para equipe
- Revisão trimestral de processos

### Como Criar
1. **Bloco 1 (12 min):** Tarefa + Briefing
2. **Bloco 2 (12 min):** Checklist + Prompt
3. **Bloco 3 (12 min):** Output + Validação
