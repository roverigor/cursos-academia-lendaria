## Simples Nacional

### O que e:
Regime unificado que junta 6+ tributos em uma unica guia (DAS).

### Caracteristicas:

| Aspecto | Detalhe |
|---------|---------|
| **Limite** | R$ 4.8M/ano |
| **Tributos incluidos** | IRPJ, CSLL, PIS, COFINS, CPP, ISS/ICMS |
| **Calculo** | Tabela progressiva por faixa |
| **Anexos** | I (comercio), II (industria), III-V (servicos) |

### Quando faz sentido:

- Faturamento ate R$ 4.8M/ano
- Servicos com ALTA folha de pagamento (Fator R >= 28%)
- Comercio em geral
- Industria pequena
- Quem quer SIMPLICIDADE

### Quando NAO faz sentido:

- Faturamento proximo do teto (pode estourar)
- Servicos com BAIXA folha (Fator R < 28%)
- Empresa com prejuizo (nao aproveita)
- Exportadores (creditos tributarios perdidos)

### Armadilhas comuns:

| Armadilha | Problema |
|-----------|----------|
| Anexo V | Aliquota inicial de 15.5% (muito alta) |
| Sublimite estadual | Pode perder ISS unificado |
| Atividades vedadas | Nem tudo pode ser Simples |

---

## Lucro Presumido

### O que e:
Regime onde a Receita Federal PRESUME seu lucro com base em percentuais fixos.

### Caracteristicas:

| Aspecto | Detalhe |
|---------|---------|
| **Limite** | R$ 78M/ano |
| **Presuncao (servicos)** | 32% do faturamento |
| **Presuncao (comercio)** | 8% do faturamento |
| **Tributos** | IRPJ 15%, CSLL 9%, PIS 0.65%, COFINS 3% |

### Calculo simplificado (servicos):

```
Faturamento: R$ 100.000/mes

Lucro presumido = R$ 100.000 x 32% = R$ 32.000
IRPJ = R$ 32.000 x 15% = R$ 4.800
CSLL = R$ 32.000 x 9% = R$ 2.880
PIS = R$ 100.000 x 0.65% = R$ 650
COFINS = R$ 100.000 x 3% = R$ 3.000

TOTAL = R$ 11.330 (11.33% do faturamento)
+ ISS (2-5%) = ~14-16% total
```

### Quando faz sentido:

- Margem de lucro ALTA (> presuncao)
- Servicos com baixa folha
- Empresa que ultrapassou Simples
- Quer simplicidade mas cresceu

### Quando NAO faz sentido:

- Margem de lucro BAIXA (< presuncao)
- Empresa com prejuizo
- Industria com muitos creditos

---

## Lucro Real

### O que e:
Regime onde voce paga imposto sobre o lucro REAL (receitas - despesas).

### Caracteristicas:

| Aspecto | Detalhe |
|---------|---------|
| **Limite** | Nenhum |
| **Base de calculo** | Lucro contabil real |
| **Tributos** | IRPJ 15% (+10% acima 240K/ano), CSLL 9% |
| **PIS/COFINS** | Nao-cumulativo (1.65% + 7.6%) com creditos |

### Quando faz sentido:

- Empresa com PREJUIZO (nao paga IRPJ/CSLL)
- Margem muito baixa (< 8-12%)
- Industria com muitos creditos de PIS/COFINS
- Empresa com altas despesas dedutiveis

### Quando NAO faz sentido:

- Empresa lucrativa com margem alta
- Estrutura contabil simples
- Quem nao quer complexidade

### Custos adicionais:

| Item | Custo Tipico |
|------|--------------|
| Contabilidade | 2-3x mais caro |
| Auditoria | Pode ser obrigatoria |
| Controles | Mais rigorosos |
| Risco | Fiscalizacao mais intensa |

## Comparativo Pratico

### Cenario: Empresa de Consultoria

| Dado | Valor |
|------|-------|
| Faturamento | R$ 150.000/mes |
| Despesas operacionais | R$ 50.000/mes |
| Folha de pagamento | R$ 30.000/mes |
| Lucro real | R$ 70.000/mes |

### Simulacao:

| Regime | Carga Estimada | % Faturamento |
|--------|----------------|---------------|
| **Simples (Anexo V)** | ~R$ 23.000 | 15.3% |
| **Simples (Anexo III)** | ~R$ 16.500 | 11% |
| **Presumido** | ~R$ 21.500 | 14.3% |
| **Real** | ~R$ 22.000 | 14.7% |

**Conclusao:** Simples Anexo III seria melhor, MAS so se Fator R >= 28%
- Fator R atual = 30.000 / 150.000 = 20% (Anexo V!)
- Se aumentar folha para 42.000, entra no Anexo III

---

## Tabela de Decisao Rapida

| Seu Cenario | Regime Provavel |
|-------------|-----------------|
| Faturamento < 4.8M + Alta folha | Simples (Anexo III) |
| Faturamento < 4.8M + Baixa folha | Simples (Anexo V) ou Presumido |
| Faturamento 4.8M-78M + Margem alta | Lucro Presumido |
| Faturamento 4.8M-78M + Margem baixa | Lucro Real |
| Empresa com prejuizo | Lucro Real |
| Industria com creditos | Lucro Real |

## Licao Principal

> **Nao existe regime "melhor". Existe o regime CERTO para seu momento.**
>
> E o momento muda. Por isso precisa revisar periodicamente.

## ARTEFATO DA AULA: Guia de Referencia de Regimes

### O Que Voce Vai Criar
Ficha de consulta rapida com caracteristicas, vantagens, desvantagens e indicacoes de cada regime tributario para usar em decisoes futuras.

### Por Que Isso Importa
Ter referencia rapida evita erros de decisao:
- Consultar quando situacao mudar
- Base para conversa com contador
- Clareza sobre opcoes disponiveis
- Evitar "achismo" em decisoes

**Linha do DRE:** Qualidade de Decisao (decisoes baseadas em conhecimento, nao intuicao)

### Quando Usar
- Ao comparar regimes (aula 2.7)
- Em conversa com contador
- Quando faturamento mudar
- Planejamento anual

### Como Criar
1. Revise conteudo da aula
2. Anote pontos-chave de cada regime
3. Identifique qual se aplica ao seu caso
4. Marque para referencia futura

### Template

```
GUIA DE REFERENCIA - REGIMES TRIBUTARIOS
=========================================

SIMPLES NACIONAL:
- Limite: R$ 4.8M/ano
- Ideal para: Pequenas empresas, alta folha
- Cuidado com: Anexo V (baixa folha), sublimite
- Aliquota tipica: 6-15% (servicos)

LUCRO PRESUMIDO:
- Limite: R$ 78M/ano
- Ideal para: Margem alta, baixa folha
- Cuidado com: INSS patronal por fora
- Aliquota tipica: 11-17% (servicos)

LUCRO REAL:
- Limite: Nenhum
- Ideal para: Prejuizo, margem baixa, creditos
- Cuidado com: Complexidade, custo contabil
- Aliquota: Sobre lucro real

MINHA SITUACAO ATUAL:
- Regime: _______________
- Fator R: _____%
- Margem: _____%
- Indicacao: _______________

TABELA DE DECISAO RAPIDA:
| Se... | Entao provavelmente... |
|-------|------------------------|
| Fat < 4.8M + alta folha | Simples III |
| Fat < 4.8M + baixa folha | Simples V ou Presumido |
| Fat > 4.8M + margem alta | Presumido |
| Prejuizo/margem baixa | Real |
```

## IA NA PRATICA

### Prompt Principal: Comparador de Regimes Personalizado

```
Atue como consultor tributario. Compare os 3 regimes para minha empresa:

DADOS DA EMPRESA:
- Faturamento anual: R$ [valor]
- Lucro anual estimado: R$ [valor]
- Margem de lucro: [__]%
- Folha de pagamento anual: R$ [valor]
- Atividade principal: [descrever]
- CNAE: [codigo]

SIMULE CADA REGIME:

1. SIMPLES NACIONAL:
   - Calcule o Fator R e identifique o Anexo
   - Calcule aliquota efetiva
   - Valor total de impostos/ano

2. LUCRO PRESUMIDO:
   - Use presuncao correta para minha atividade
   - Inclua TODOS os tributos (IRPJ, CSLL, PIS, COFINS, ISS/ICMS)
   - Inclua INSS patronal (que nao existe no Simples!)
   - Valor total de impostos/ano

3. LUCRO REAL (se aplicavel):
   - Use meu lucro real
   - Estime creditos de PIS/COFINS
   - Valor total de impostos/ano

COMPARATIVO FINAL:
| Regime | Carga (%) | Valor/ano | Complexidade |
|--------|-----------|-----------|--------------|

Qual regime e melhor para meu caso e por que?
```

### Prompt Alternativo: Explicador de Regime

```
Explique de forma simples como funciona o [Simples Nacional / Lucro Presumido / Lucro Real] para meu tipo de negocio:

MEU PERFIL:
- Atividade: [descrever]
- Faturamento: R$ [valor]/mes
- Funcionarios: [numero]

EXPLIQUE:
1. Como o imposto e calculado neste regime?
2. Quais tributos estao inclusos?
3. Vantagens para meu tipo de negocio
4. Desvantagens e armadilhas
5. Quando faz sentido vs quando NAO faz
6. Exemplo pratico com meus numeros

Linguagem simples, como se explicasse para alguem sem formacao contabil.
```

### Como Usar o Resultado

1. **Compare os numeros** gerados com sua situacao atual
2. **Identifique** se ha economia em outro regime
3. **Anote as armadilhas** mencionadas (ex: INSS fora do Simples)
4. **Leve os calculos** para validar com seu contador
