# Mente Lendaria - Conteudo Programatico Detalhado

## Framework: Metodo 4C

O curso segue o framework **4C** que representa as 4 etapas de transformacao de informacao em conhecimento acionavel:

1. **Consumir** - Filtrar informacao com intencionalidade
2. **Capturar** - Registrar e organizar o que importa
3. **Conectar** - Criar relacoes entre ideias
4. **Criar** - Produzir output a partir do conhecimento
