SSH com Ed25519
	O Que e SSH
		Secure Shell
			Protocolo criptografado
		Par de chaves
			Chave privada (seu PC)
			Chave publica (GitHub)
		Analogia
			GitHub = cadeado
			Seu PC = chave
		Seguranca
			Nao compartilhe chave privada
	Por Que Ed25519
		Comparacao algoritmos
			RSA
				2048-4096 bits
				Lenta
				Boa seguranca
				Antigo
			Ed25519
				256 bits
				Rapida
				Excelente seguranca
				Recomendado 2024
		GitHub recomenda
			Padrao desde 2022
			Mais seguro e rapido
	Passos de Configuracao
		1. Verificar chave existente
			ls -la ~/.ssh/id_ed25519.pub
			Se existe, pular para passo 3
		2. Gerar nova chave
			ssh-keygen -t ed25519 -C "email"
			Aceitar local padrao
			Passphrase (opcional)
		3. Adicionar ao SSH agent
			eval "$(ssh-agent -s)"
			ssh-add ~/.ssh/id_ed25519
		4. Copiar chave publica
			macOS: pbcopy
			Linux: cat e copiar
			Windows: clip
		5. Adicionar no GitHub
			Settings â†’ SSH and GPG keys
			New SSH key
			Colar chave publica
		6. Testar conexao
			ssh -T git@github.com
			Mensagem: "Hi username!"
	SSH vs HTTPS
		SSH
			git@github.com:user/repo.git
			Mais seguro
			Nao pede senha
			Recomendado
		HTTPS
			https://github.com/user/repo.git
			Computador compartilhado
			CI/CD
	Configuracao Automatica
		macOS
			~/.ssh/config
			AddKeysToAgent yes
			UseKeychain yes
		Linux
			~/.bashrc ou ~/.zshrc
			Iniciar ssh-agent
	Troubleshooting
		Permission denied
			Chave nao adicionada no GitHub
			Ou usando HTTPS
		Could not open connection
			eval "$(ssh-agent -s)"
			ssh-add ~/.ssh/id_ed25519
