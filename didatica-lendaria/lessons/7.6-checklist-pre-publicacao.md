## Checklist Pré-Publicação (8 Categorias)

### 1. CONTEÚDO (Didática)

Valida ANTES de publicar:

- [ ] **GPS presente?**
  - Destino claro (onde aluno vai)?
  - Dor validada (conecta emocionalmente)?
  - Rota lógica (sequência faz sentido)?

- [ ] **2-3 conceitos máximo?**
  - Não ultrapassou Regra de Ouro?

- [ ] **Semiótica forte?**
  - Pelo menos 1 analogia/metáfora?
  - Criou "filme na cabeça"?

- [ ] **Logos, Etos, Patos balanceados?**
  - Lógica (Logos) presente?
  - Credibilidade (Etos) estabelecida?
  - Emoção (Patos) conectada?

- [ ] **7 elementos presentes?**
  - Motivação, Conceito, Semiótica, Prática, Link, Revisão, Hook?

- [ ] **Hook no final?**
  - Cria curiosidade pra próxima aula?

**6/6? Conteúdo validado.**

### 2. ÁUDIO (Qualidade Técnica)

- [ ] **Sem ruídos de fundo?**
  - Cachorro, carro, ventilador, eco?

- [ ] **Volume consistente?**
  - Não tem partes muito baixas ou altas?

- [ ] **Dicção clara?**
  - 100% das palavras audíveis (sem legenda)?

- [ ] **Sem cortes bruscos de áudio?**
  - Transições suaves entre cortes?

**4/4? Áudio OK.**

### 3. VÍDEO (Qualidade Visual)

- [ ] **Foco nítido?**
  - Rosto/tela não está desfocado?

- [ ] **Iluminação adequada?**
  - Rosto visível (não escuro)?

- [ ] **Enquadramento correto?**
  - Rosto centralizado, olhos no terço superior?

- [ ] **Sem distrações visuais?**
  - Fundo bagunçado, objetos estranhos?

**4/4? Vídeo OK.**

### 4. EDIÇÃO (Qualidade de Corte)

- [ ] **Sem erros graves?**
  - Cortou todas as falas erradas?

- [ ] **Pausas longas removidas?**
  - Sem silêncios de 5+ seg?

- [ ] **Cortes suaves?**
  - Transições não causam "pulo" estranho?

- [ ] **Texto na tela legível?**
  - Tamanho grande, contraste alto?
  - Máximo 3-5 seg por texto?

**4/4? Edição OK.**

### 5. RECURSOS (Links e Materiais)

- [ ] **Todos os links funcionam?**
  - Testou CADA link clicando?

- [ ] **Arquivos disponíveis?**
  - PDF, template, código, etc. acessível?

- [ ] **Instruções claras?**
  - Aluno sabe ONDE encontrar recursos?

**3/3? Recursos OK.**

### 6. METADATA (Título, Descrição, Tags)

- [ ] **Título otimizado?**
  - Usa fórmula testada (Problema+Solução, Número, etc.)?
  - Máximo 60 caracteres?
  - Sem typo?

- [ ] **Thumbnail chamativa?**
  - Rosto grande + expressão clara?
  - Texto legível (máx 3 palavras)?
  - Alto contraste?

- [ ] **Descrição completa?**
  - Resumo da aula (2-3 frases)?
  - Links para recursos?
  - Timestamp dos tópicos principais?

- [ ] **Tags relevantes?**
  - Palavras-chave relacionadas ao conteúdo?

**4/4? Metadata OK.**

### 7. ACESSIBILIDADE (Inclusão)

- [ ] **Legendas disponíveis?**
  - Auto-geradas (YouTube) ou manuais?
  - Precisão: 95%+ (se manual)?

- [ ] **Velocidade ajustável?**
  - Plataforma permite 0.75x, 1.25x, 1.5x?

- [ ] **Descrição de imagens (se relevante)?**
  - Narrou o que aparece na tela?

**3/3? Acessível.**

### 8. TESTE FINAL (Experiência Real)

- [ ] **Assistiu aula COMPLETA?**
  - Do início ao fim, como aluno?

- [ ] **Testou em dispositivo diferente?**
  - Mobile? Tablet? Desktop?

- [ ] **Clicou em TODOS os links?**
  - Funcionam?

- [ ] **Praticou o exercício?**
  - Instruções claras? Executável?

**4/4? Pronto pra publicar.**

## Workflow de Checklist (15-20 min)

**Passo 1: Assiste aula completa (10 min)**

Como se fosse ALUNO.

Anota qualquer problema.

**Passo 2: Testa recursos (3 min)**

Clica TODOS os links.

Baixa TODOS os arquivos.

**Passo 3: Valida metadata (2 min)**

Título, thumbnail, descrição, tags.

**Passo 4: Teste em mobile (3 min)**

Abre no celular. Assiste 2 minutos.

Áudio OK? Vídeo OK? Texto legível?

**Passo 5: Publica.**

**Total: 15-20 minutos.**

**Vale a pena. 1 erro evitado = 10-100 alunos salvos.**

## Erros Mais Comuns (Top 5)

### Erro 1: Link Quebrado (40% dos casos)

**Sintoma:** Aluno: "Link não funciona!"

**Prevenção:** Clica TODOS os links antes de publicar.

### Erro 2: Áudio Cortado (25%)

**Sintoma:** Final da frase cortado abruptamente.

**Prevenção:** Assiste até ÚLTIMO segundo do vídeo.

### Erro 3: Typo no Título (15%)

**Sintoma:** "Diddática" em vez de "Didática"

**Prevenção:** Revisa título 2x. Copia pro corretor ortográfico.

### Erro 4: Thumbnail Errada (10%)

**Sintoma:** Thumbnail da Aula 3 na Aula 5.

**Prevenção:** Confere nome do arquivo antes de upload.

### Erro 5: Recurso Não Anexado (10%)

**Sintoma:** "Baixe o template aqui" MAS link não existe.

**Prevenção:** Testa download como se fosse aluno.
